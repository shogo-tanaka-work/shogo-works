GitHub Copilot 教材の「エージェント運用」章に、Agents window の Stable 化・複数セッション並行・リモートエージェントセッション（SSH/Dev Tunnels、切断後継続）を追記。「補完ツール → エージェント基盤」の最新例として位置づける。

企業導入章には Air-gapped BYOK（ネットワーク分離環境での採用）とコマンドリスク評価（実行前のリスク提示・機密非送信）を、ガバナンス/セキュリティ論点として整理。BYOK の実トークン使用量可視化はコスト管理フックとして扱う。

VS Code 月次まとめ記事（copilot-vscode-april-2026 の系列）として継続。次月以降も同フォーマットで差分追記する想定。
