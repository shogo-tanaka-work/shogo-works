## 教材化ポイント

### 「どの IDE でどの AI を使うか」が選択可能になる時代

JetBrains + Copilot + Claude の組み合わせは、「開発ツールが特定の AI ベンダーに縛られなくなる」トレンドの具体例。VS Code / JetBrains をどちらも使う組織がツール乗り換えコストなしにエージェントプロバイダーを変更できる。「AI ツール選定」の研修で、統合度・コスト・パフォーマンスを比較する軸として使える。

### Cloud Agent GA の意味

「クラウド上でエージェントが動く」ことが標準機能になった。ローカル実行とクラウド実行の違い（コンテキスト保持・並行実行・コスト）を教材として整理する機会。「ローカル vs クラウドエージェント」の選択基準を作るコンテンツに使える。

### 組織展開（カスタムエージェント一括配布）

管理者がカスタムエージェントを全メンバーに展開できる機能は、「AI ツールの組織導入 = 個人の設定ではなく IT ガバナンスの課題」という視点で解説できる。AI 活用推進担当者向けの実践コンテンツとして適している。
