Copilot Cloud Agent教材で、UIからの手動起動だけでなく、REST API経由で社内ポータルや運用スクリプトから起動する構成を追加できる。

Knowledge化するなら、API endpointよりも、対象リポジトリ、起動権限、PR作成有無、レビュー責任、Secrets / Variables分離をセットで説明する。現時点ではBusiness / Enterprise対象なので、個人向け教材へは「今後の方向性」として扱う。
