開発者向けAIエージェント教材に向いている。

演習案:

- semantic searchで既存実装を調査
- `githubTextSearch` で関連repoを横断検索
- agentに修正方針を作らせる
- inline diffでレビュー
- browser tab sharingでUI確認
- open terminalのログを読ませてテスト修正
- `/chronicle` で過去作業を振り返る
- remote CLI sessionで長時間作業を継続

既存Knowledgeへ追記するなら、harness-engineeringのlong-running agentsやagent code reviewに「Copilot実装例」として入れる。
