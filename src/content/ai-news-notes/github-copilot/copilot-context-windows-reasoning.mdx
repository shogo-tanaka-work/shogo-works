Copilot 教材の「モデル / コンテキスト設定」章に、コンテキストウィンドウ拡大と推論レベル設定を追加。推論レベルを「軽い補完は低・難所は高」と使い分けることで、トークン消費とレイテンシをコントロールする例を示す（コスト/精度トレードオフ）。

「エージェント / 自動化」章には Agent tasks REST API を追記し、Pro / Pro+ / Max でも利用可能になった点と、CI 連携で「失敗 Action → Fix with Copilot で自動修正」を回す例を載せる。プラン別の機能差（Agent tasks API・Fix with Copilot が Pro/Pro+/Max へ拡大）を表で整理する。VS Code 5 月リリースまとめ（前日教材化済み）とは別更新である点を明記して重複を避ける。
