## 教材化のポイント

### AI クレジットという概念の説明素材

「AI クレジット」とはどういう課金単位かを説明する具体例として使える。チャット・コード補完・Agent 処理などで消費される単位を、電力の「kWh」に例えると初心者にも伝わりやすい。AI ツールの「使い放題」と「クレジット従量」の違いを整理する際の現実例。

### Copilot メトリクス API の全体像を教える文脈

今回の `ai_credits_used` は既存の Usage Metrics API への追加フィールド。「Copilot をどう測定するか」という組織管理の観点で、API を使ったコスト可視化ダッシュボード構築の教材につながる。

### 企業 AI 導入後の「継続管理フェーズ」の事例

「導入して終わり」ではなく「使われているか・費用対効果はどうか」を追跡するフェーズの重要性を伝える事例。AI ツール導入教材の「導入後に何をすべきか」セクションに加えると実践的になる。
