コスト管理教材向け。

- base credits と flex allotment の違い
- agent runが増えると消費が増える構造
- paid planでもcompletion / next edit suggestionsはcredits対象外
- Max planは高頻度agent利用者向け

演習では、1週間の開発タスクを想定して、agent依頼の粒度ごとにcredits消費の見積もり表を作る。
