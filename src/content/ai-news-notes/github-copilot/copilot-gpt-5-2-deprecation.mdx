Copilot のモデル選択を扱う章のモデル一覧を更新し、GPT-5.2 / GPT-5.2-Codex を「非推奨」とし、後継 GPT-5.5 / GPT-5.3-Codex に置き換える。GPT-5.2 が Copilot code review の一部でのみ継続利用可能である点を例外として明記し、「機能ごとに利用可能モデルが異なる」ことを表で示す。

Enterprise 運用章には、管理者がモデルポリシーで後継モデルへのアクセスを明示有効化する必要があること、モデル ID を固定指定したワークフロー・統合の棚卸し手順を追記する。モデル非推奨は繰り返し起きるため、「使用中モデル一覧 × ポリシー設定」を定点レビューする運用テンプレートとして一般化しておく。
