## 教材化のポイント

### Noise プロトコルとは

教材の技術深度に応じて説明の粒度を変える:

- 入門レベル: 「Signal や WireGuard で使われている安全な暗号化プロトコル」
- 中級レベル: 「Noise Framework は鍵交換から対称暗号化まで一貫したセキュリティプロパティを持つ」

Codex のリモート実行環境という文脈では「リモート操作の通信が暗号化・認証されるようになった」という結論から話し始めると導入しやすい。

### リモート実行のセキュリティを問われる場面

AI エージェントがリモートマシンを操作する構成は「何が送受信されているか」が問題になりやすい。Noise リレーの採用は「企業のセキュリティ審査でクリアしやすくなった」という実務上の意味を持つ。

### MCP との接続

エグゼキュータープラグインが stdio MCP サーバーをスレッドごとに起動できるようになった点は、MCP を活用した複雑なエージェント構成への布石として説明できる。
