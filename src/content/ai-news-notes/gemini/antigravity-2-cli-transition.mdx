「CLI エージェント比較」教材があれば、Claude Code / Codex CLI / Gemini CLI / Aider の 4 ツール比較に Antigravity CLI を追加。Gemini CLI 単体ではなく「Antigravity プラットフォーム + 軽量 CLI」というレイヤー構造になった点を明示。`gemini` コマンドを使う既存教材は、移行先注記を入れる必要あり。

「AI エージェントオーケストレーション」教材では、Anthropic Managed Agents / OpenAI Agents SDK / Gemini Managed Agents の 3 社揃ったため、エージェント API の比較表（起動レイテンシ、隔離環境スペック、ツール定義方法、料金、タイムアウト）を組める。

「AI Studio とは」教材があれば、モバイル化と Workspace API 接続、Android ネイティブ生成を反映。

Knowledge へは、Gemini CLI → Antigravity CLI 移行情報のみ最小反映（既存の Gemini CLI 教材があれば「移行先が決まり次第更新」を 1 行追記）。詳細仕様は API ドキュメント側で随時更新されるため、本ニュースから直接 Knowledge には深入りしない。
