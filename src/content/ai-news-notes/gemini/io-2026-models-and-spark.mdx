Gemini モデル一覧 / 料金プラン教材があれば、Gemini 3.5 Flash GA・Omni Flash・3.5 Pro（6月）を反映。Pro と Ultra（$100/月）の差分を明確化し、Ultra の主用途として Spark / Antigravity の利用上限増を併記する。

「AI エージェント比較」教材では、ChatGPT Agent / Claude Skills / Gemini Spark の 3 つを横並びで扱えるようになった。比較軸は「常駐性（24/7 / 都度呼び出し）」「連携先（OS / アプリ / API）」「権限モデル」「料金」「対応地域」。

Workspace 教材では、Spark が Gmail / Docs / Slides の上で能動的に動くため、「ガバナンス論点（個人契約の暴走、ログ・監査、PII 取扱）」を 1 章追加できる。Knowledge へは Gemini モデル一覧の数値（料金 / 性能比）が安定したら最小反映する判断。Spark は米国 Ultra ベータの段階なので、Knowledge 反映は日本展開 / 詳細仕様確定後にする。
