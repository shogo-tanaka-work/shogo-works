models 教材は **Gemini 3.5 Flash**（フロンティア級・半額以下・4 倍速）と **Gemini Omni Flash**（任意入力から動画生成、Plus/Pro/Ultra）を新規モデルとして追加。Omni Flash の **YouTube Shorts / Create app 無償展開** と **開発者 API 数週間以内 rollout** をスケジュール情報として明示。

workspace 教材は **Gemini Spark**（24/7 パーソナルエージェント、Gmail / Docs / Slides でタスク自動化、ロック中実行、Ultra ベータ）を「管理者運用」節として新設。**有効化判断・利用範囲・監査ログ要件** を 3 点セットで書く。Daily Brief（米国 subscribers から rollout）も同節で扱う。

「Workspace × Gemini ガバナンス」教材があれば、同日発表の **Gemini Drive 共有**（2026-05-28 admin / 2026-06-03 end-user）も合わせて扱う。Gemini アセットが Drive 共有ポリシーに従う点を明示。

**SynthID** は別途「AI コンテンツ来歴標準」の独立節を持つ価値あり。1000 億画像超への拡張と **OpenAI / Eleven Labs の採用** を業界収束として記述。

**TPU 8 世代（8t 訓練 + 8i 推論）** は技術ロードマップとして 1 段書く。Vertex AI を使っている組織向けに「将来の性能 / コスト改善示唆」と位置付ける。

**Neural Expressive UI** は全 Gemini ユーザー向けに本日グローバル展開のため、既存スクリーンショットの差し替え案件として独立 TODO リスト化。

**Gemini app 900M MAU、Google 全体 3.2 quadrillion tokens / 月** は規模指標として記録のみ。教材本体には恒久情報として持ち込まない。
