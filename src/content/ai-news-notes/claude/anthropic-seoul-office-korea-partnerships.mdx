## 教材化のポイント

### NAVER の全社展開事例の意味

「全エンジニアに Claude Code を展開」という事例は、以下の観点で教材に使える:

- **スケール**: 数千人規模のエンジニア組織への一斉展開
- **ツール**: Claude Code + Cowork という組み合わせ
- **背景**: AI コーディングツールの「試験導入 → 全社展開」フェーズへの移行

「どうやって全社展開したか」「どんな課題があったか」は今後の発表を待つ必要があるが、先行事例として記録しておく価値がある。

### Anthropic の地域展開パターン

- 米国本社 → 各地域オフィス設置というパターン
- ソウルは Asia-Pacific での拠点のひとつ
- 日本展開については別途確認が必要

「AI ツールのグローバル展開と地域サポートの重要性」という観点で、企業 IT の国際調達や対応言語の話と合わせて語れる。

### 韓国 IT 企業と AI の関係

NAVER・Nexon・Samsung SDS・LG CNS はそれぞれ:
- NAVER: 韓国最大の検索・IT 企業（自社 AI も持つ）
- Nexon: 大手ゲーム会社（AI コード生成の需要が大きい）
- Samsung SDS・LG CNS: 大手グループの IT 部門

自社 AI を持つ NAVER が Claude Code を採用している点は、「競合 AI 企業も外部ツールを使う」という文脈で紹介できる。
