hooks教材に`terminalSequence`を通知・window title・bell用途として追記候補。企業運用教材では、GitHub plugin sourceのHTTPS clone優先、workspace ID指定、`claude agents --cwd <path>`を、チーム利用時の摩擦を下げる設定として扱える。

Knowledgeへ反映する場合は、v2.1.141固有の細かい修正一覧を持ち込まず、hooks、plugin導入、agent session整理の3点だけに絞る。
