## 教材化ポイント

### 1. スキル教材への追記候補

`src/content/knowledge/ai-tools/claude-code/skills.mdx` に、単一スキル呼び出しの説明に加えて「複数スキルをスタックして1コマンドで呼び出す」書き方を追記する余地がある。既存の呼び出し例が `/skill-name` 単体想定になっている場合は更新が必要。

**教材への落とし込み方:**

- 「よく組み合わせるスキルの一覧」を作り、スタック呼び出しのテンプレート化を教材で提案する
- 先頭最大5個という制限を明記し、6個以上必要な場合の分割呼び出しパターンも併記する

### 2. バックグラウンドエージェント信頼性の系譜

v2.1.198（自動コミット/PR作成）に続き、v2.1.199でサブエージェントの部分結果保持・ストリーミングpartial保持が入ったことで、「エージェントに長時間タスクを任せる」運用がv2.1.19x系で段階的に固まってきている。

**教材への落とし込み方:**

- バックグラウンドエージェント運用ガイドで「打ち切られても結果はゼロにならない」という前提を明記できる
