new-commands-2026 教材で `/simplify` の記述があれば `/code-review` に差し替え。effort level の指定例（quick / standard / deep の使い分けの想定例）を 1 セクション追加する候補。

Auto Mode 教材は、`AskUserQuestion` を明示的に使うスキル設計との関係を「Auto でも止まる場面 / 止まらない場面」の例で 1 段補足。MCP 教材ではページネーション対応の必要性を「2 ページ目以降がドロップする実装ミス」の文脈で再強調。
