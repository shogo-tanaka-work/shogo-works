## 教材化のポイント

### アーティファクトの本質的な価値

「セッションをライブで見せる」というコンセプトが新しい。これまでのAIツールの多くは「結果を出力して終わり」だったのが、「進行中のプロセスをリアルタイム共有」という使い方が生まれた。

インシデント対応・プロジェクト進捗・コードレビューなど「今まさに作業中を見てもらいたい」場面での活用例を具体的に示せると理解しやすい。

### Team / Enterprise 限定という注意点

個人ユーザー（Free / Pro）は対象外。教材で紹介する際は対象プランを明確にして誤解を防ぐ。

### URL 不変 + バージョン管理の意味

- URL が変わらない → 「進捗の固定リンク」として Slack やメールに貼れる
- バージョン履歴 → 「あの時点の状態に戻す」が可能

変更前後の比較や時系列での進捗確認に使える点を説明に加えると実務感が増す。
