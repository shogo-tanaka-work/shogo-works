## 教材化ポイント

### 1. worktree隔離の検証手順を教材に追加

`src/content/knowledge/ai-tools/claude-code/plan-mode-worktrees.mdx`に、「subagentのworktree隔離は今回のようなgit操作漏れが過去に起きた」ことと、「隔離が期待どおり効いているかを確認する習慣」を追記する価値がある。

### 2. subagent権限設計の注意点として権限ルール警告を紹介

`src/content/knowledge/ai-tools/claude-code/subagents.mdx`に、`Write(path)`等の実質無効な権限ルールがあること、`Edit(path)`/`Read(path)`が正しい書き方であることを補足できる。
