## 要約

Runway は 2026 年 5 月 21 日、動画編集モデル **Aleph 2.0** と、それを中心に据えた新アプリ **Edit Studio** を発表しました。コンセプトは「すでに持っている動画から、本当に必要な動画を得る」こと。ゼロから動画を生成するのではなく、**手元の動画を作り直す**ためのモデルとアプリです。

最大の特徴は **1フレーム編集の全体伝播**です。動画の中の1フレームを編集して結果をプレビューすると、Aleph 2.0 がその編集を**クリップ全体へ自動で反映**します。これまでの動画編集 AI は、変更したい範囲をショット単位・区間単位で指定する必要がありましたが、Aleph 2.0 は「1フレームをこう変えたい」を起点に、残りを自動で揃えます。

さらに、カットやシーン切り替えのある動画でも、Aleph 2.0 は**関連するショットをまたいで編集を適用**します。複数カットの動画を「ショットごとに同じ編集を繰り返す」必要がなくなり、編集対象は最長 **30 秒・1080p** のクリップに対応します。

これらの機能を中心に設計された新アプリが **Edit Studio** です。マーケティングチームのキャンペーン派生制作、映像制作者の後処理、スモールビジネスオーナーによる既存動画のリフレッシュなどを想定した編集環境で、**最終生成の前に変更を画像としてプレビュー**し、見た目を詰めてから生成に進めます。Edit Studio の Aleph 2.0 は、**全有料プラン**でデスクトップ Web アプリから利用できます。

## 何が変わったか

- 動画編集モデル Aleph 2.0 を発表（Runway のフラッグシップ動画編集モデルの upgrade 版）
- 1フレームを編集 → Aleph 2.0 がその編集をクリップ全体へ自動伝播
- カット・シーン切り替えのある動画でも、関連ショットをまたいで編集を適用（ショット単位の手作業が不要）
- 最長 30 秒・1080p のクリップを編集可能
- 新アプリ Edit Studio を発表（Aleph 2.0 を中心に設計した編集環境）
- Edit Studio では、最終生成前に変更を画像としてプレビューして見た目を確認できる
- Edit Studio の Aleph 2.0 は全有料プランでデスクトップ Web アプリから利用可能

## 業務インパクト（一般企業向け）

この発表が効くのは、**「ゼロから動画を作る」場面より「すでにある動画を作り直す」場面**です。企業の動画運用では、後者の比率が実は高くなります。たとえばマーケティングでは、1本のベース動画から、訴求文言・色味・被写体の小物などを変えた**キャンペーン派生バリエーション**を何本も作ります。これまではバリエーションごとに編集者がショット単位で手を入れていましたが、Aleph 2.0 の1フレーム編集＋全体伝播は、この作業を「1フレームで意図を示す → 全体に展開」へ置き換えます。

複数カットの動画でショットをまたいで編集が適用される点も実務的です。インタビュー動画や商品紹介動画のように切り替えの多い素材を、カットごとに同じ調整を繰り返さずに済むため、編集工数のかかり方が変わります。**動画編集を外注している企業**は、外注の見積もり前提（ショット数×単価のような積み上げ）が現実と合わなくなる可能性があるため、発注内容と費用感を見直す価値があります。

一方で、業務導入では制約も押さえる必要があります。対象は最長 30 秒・1080p のクリップで、長尺コンテンツをそのまま扱えるわけではありません。また Edit Studio はデスクトップ Web アプリでの提供で、利用には Runway の有料プラン契約が必要です。生成 AI で映像を編集する以上、**素材の権利関係・ブランドガイドラインとの整合・最終チェック体制**は従来どおり必要です。「画像プレビューで生成前に確認できる」点は品質管理に有利ですが、公開前の人手レビューを省く理由にはなりません。

## 副業・個人活用視点

動画制作を請け負う個人にとって、Aleph 2.0 は**「修正対応」の生産性を大きく変える**ツールになり得ます。クライアントワークでは「文言だけ変えて」「色味を別パターンで」といった派生・修正の依頼が頻繁に発生し、ここに時間を取られがちです。1フレーム編集＋全体伝播を使えば、修正の1巡あたりの作業時間を圧縮でき、同じ単価でも対応できる案件数を増やせます。

スモールビジネスや個人クリエイターにとっては、**既存動画のリフレッシュ**が現実的な使いどころです。商品やサービスが少し変わるたびに動画を撮り直すのは負担が大きいですが、Edit Studio で手元の動画を編集して使い回せれば、撮影コストをかけずに鮮度を保てます。SNS 向けに、1本のベース動画から複数バリエーションを出す運用とも相性が良いでしょう。

マネタイズの観点では、「Aleph 2.0 / Edit Studio を使った動画修正・派生制作」をサービスメニュー化する余地があります。ただし注意点として、対象が 30 秒・1080p のクリップである以上、長尺案件はそのまま受けられません。ショート動画・広告クリップ・SNS 用素材に絞った「短尺の編集・派生に強い」ポジションで打ち出すのが、現時点では現実的です。
