## 要約

**Google が Gemini 3.5 をベースにした音声翻訳モデル「Gemini 3.5 Live Translate」を発表**しました。**70 以上の言語・2,000 以上の言語ペア**に対応し、**英語を経由しない直接翻訳**を行います。話者のイントネーション・ペース・ピッチを保持した自然な翻訳音声を生成するのが特徴です。

提供面は段階的で、**Google Translate（Android / iOS）でグローバルロールアウト**、**Google Meet はエンタープライズ向けプライベートプレビュー**、**Gemini Live API は開発者向けパブリックプレビュー**です。全音声出力は **SynthID で透かし処理**されます。

## 何が変わったか

- **70 以上の言語・2,000 以上の言語ペア**に対応する音声翻訳モデル。
- **英語を経由しない直接翻訳**。話者のイントネーション・ペース・ピッチを保持した自然な音声を生成。
- 提供面: Google Translate（Android / iOS）でグローバル展開、Google Meet はエンタープライズ向けプライベートプレビュー、**Gemini Live API は開発者向けパブリックプレビュー**。
- 全音声出力に **SynthID 透かし**。

## 業務インパクト（一般企業向け）

多言語でのコミュニケーションを抱える企業にとって、**Google Meet のエンタープライズプレビュー**は会議の言語障壁を下げる実用的な動きです。英語を経由しない直接翻訳と韻律の保持により、機械翻訳特有の不自然さが減り、相手の話し方のニュアンスも伝わりやすくなります。

開発者にとっては、**Gemini Live API のパブリックプレビュー**が大きく、自社アプリ・サービスへリアルタイム音声翻訳を組み込めます。カスタマーサポート、現場作業の多言語支援、インバウンド対応などに応用できます。SynthID 透かしは、合成音声の出所を明示できる点でガバナンス・コンプライアンス上も意味があります。プレビュー段階のため、対象シーンを絞った検証から始めるのが現実的です。

## 副業・個人活用視点

個人では、まず Google Translate アプリでの音声翻訳の自然さ向上を体感できます。海外クライアントとのやり取りや、旅行・現地取材などで使えます。開発者・副業エンジニアにとっては、Gemini Live API のパブリックプレビューが「音声翻訳機能を自作プロダクトに載せる」題材になります。多言語対応を売りにしたアプリや、インバウンド向けツールのプロトタイプを、英語非経由・韻律保持の翻訳を前提に作れるのは新しい切り口です。
