## 要約

Google が 2026年6月の主要AIアップデートをまとめたブログ記事を 2026-07-01 に公開しました。以下の新機能が含まれます。

**Gemini 3.5 Flash Computer Use** については 2026-06-24 にすでに発表・記事化済みのため、本記事では新たに明らかになった機能を中心に取り上げます。

### Gemini 3.5 Live Translate

70言語以上に対応したリアルタイム音声翻訳機能です。単純なテキスト変換ではなく、**話者の抑揚・イントネーションを保持したまま翻訳音声を生成**するのが特徴。多言語の通話・会議での同時通訳として設計されています。

### Gemini Omni Flash の API パブリックプレビュー

動画生成・編集・分析のカスタムワークフローを API 経由で構築できる Gemini Omni Flash がパブリックプレビュー段階に入りました（2026-06-30〜）。Gemini API から呼び出すことで動画を扱う独自アプリケーション・自動化パイプラインが構築できます。

### Gemini Home スピーカー

Gemini を組み込んだ Google Home スマートスピーカーが発表されました。自然な会話と複数リクエストの同時処理をサポート。既存の Google Assistant デバイスからの移行先として位置づけられています。

### Gemini 学習ノートブック

Gemini アプリに学習目標の設定・進捗追跡ができる学習ノートブック機能が追加されました（教育向け）。

## 何が変わったか

- **Gemini 3.5 Live Translate**: 70言語以上のリアルタイム音声翻訳。話者の抑揚を保持
- **Gemini Omni Flash API**: パブリックプレビュー開始。カスタム動画ワークフロー構築向け
- **Gemini Home スピーカー**: 新ハードウェア発表、自然な会話と複数リクエスト同時処理
- **Gemini 学習ノートブック**: 学習目標設定・進捗追跡機能（Gemini アプリ内）

## 業務インパクト（一般企業向け）

**グローバル組織・多言語チーム:**  
Live Translate は海外子会社・パートナーとのオンライン会議、国際カスタマーサポートの通話など、従来は通訳が必要だった場面への適用候補です。話者の抑揚を保持するという設計は、感情や強調のニュアンスが重要なビジネス交渉でも機能しやすいとされています。

**動画を扱う部門・マーケティングチーム:**  
Omni Flash API が使えるようになると、マーケティング動画の自動生成・要約・翻訳を API ワークフローとして組み込めます。動画コンテンツを大量生産・多言語展開する体制を検討している組織は、プレビュー段階から試しておくと先行優位があります。

**スマートオフィス・IT担当:**  
Gemini Home スピーカーの法人利用が展開された場合、会議室への設置・音声コマンドによる会議支援などのユースケースが生まれます。現時点では発表段階であり、具体的な法人向けプランを待ちながら動向をウォッチしてください。

## 副業・個人活用視点

Live Translate は個人でも海外クライアントとのビデオ通話や、英語コンテンツのリアルタイム理解に活用できます。70言語対応で抑揚まで保持するというレベルは、英語ネイティブとの打ち合わせで通訳なしに挑戦してみる際のサポートとして使える水準です。

Omni Flash API はクリエイター向けの動画自動生成サービスや、YouTube コンテンツの多言語展開ツールをAPIで構築する副業アイデアとして検討できます。パブリックプレビュー中に実際にAPI連携を試しておくと、GA時に素早く商用展開できます。
