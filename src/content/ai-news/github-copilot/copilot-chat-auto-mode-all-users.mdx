## 要約

GitHub Copilot Chat の Auto mode が全 Copilot プランのユーザーに開放されました。これまで一部ユーザー向けに提供されていたものが、Free から Max・Business・Enterprise まで全プランで使えるようになっています。

Auto mode は、タスクの複雑さと現在のシステム状態に応じて、使用するモデルを自動で選択します。簡単な質問には軽量モデル、複雑な推論には上位モデルというように、ユーザーが毎回選ばなくても最適な選択が行われます。

有料プランのユーザーは auto mode 使用時に 10% の割引が適用されます。

## 何が変わったか

- Auto mode が github.com と Copilot モバイルアプリで全プランに開放
- タスク複雑さ・システム状態に基づいてモデルを自動選択
- 有料ユーザーは auto mode 使用時に 10% 割引適用
- Free・Pro・Pro+・Max・Business・Enterprise すべてが対象

## 業務インパクト（一般企業向け）

モデル選択のガイドラインをチームに周知する運用コストが下がります。「どのモデルを使えばいいか」という問いに対して、「auto を使えばよい」というシンプルな答えが出るようになりました。

コスト管理の観点では、軽いクエリに重いモデルが使われなくなるため、従量課金環境でのコストが安定しやすくなります。有料ユーザーへの 10% 割引は、volume が大きいチームほど実質的なメリットになります。

## 副業・個人活用視点

「どのモデルを使うべきか」と悩む時間が節約できます。副業案件で素早く回答が欲しいときや、複数タスクを処理しているときに、モデル選択の手間なく最適な結果を得られます。

個人の Pro・Pro+ プランでも割引が効くため、使用頻度が高い人には実際のコスト削減になります。
