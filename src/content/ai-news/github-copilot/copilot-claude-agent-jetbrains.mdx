## アップデート概要

GitHub Copilot の JetBrains IDE 統合に、Claude をエージェントプロバイダーとして選択できる Public Preview が追加された。合わせて Cloud Agent の GA 昇格、組織管理者向けのカスタムエージェント配布機能も発表された。

## Claude as Agent Provider（JetBrains Public Preview）

JetBrains IDE の Copilot Chat でエージェントプロバイダーとして Claude を選択できるようになった。

利用には Claude Code CLI のインストールとパス設定が必要で、Business/Enterprise プランの管理者がプレビュー機能ポリシーを有効化することで組織に展開できる。これにより、JetBrains IDE 内から Claude のエージェント機能（長期タスク実行・コンテキストを超えた自律操作）を Copilot の UI を介して利用できる。

## Cloud Agent GA 昇格

これまでプレビュー機能フラグが必要だった Cloud Agent が正式 GA（一般提供）に昇格した。Copilot がクラウド側でエージェントタスクを実行する機能が標準利用可能になったことを意味する。

## 組織向けカスタムエージェントの一括配布

組織・Enterprise 管理者が定義したカスタムエージェントを、メンバーのワークスペースに自動配布できるようになった。標準化されたエージェント構成を組織全体に展開・管理する運用が可能になる。

## その他の追加機能

- **CLI メッセージキュー**: CLI 実行中に「Add to Queue」「Steer with Message」「Stop and Send」の3オプションでメッセージを送信可能
- **モデル選択の改善**: `/models` スラッシュコマンド追加、大規模コンテキストウィンドウ対応モデルの選択、最近使用モデルの表示
- **AI クレジット可視化**: 各ターンの AI クレジット消費量をリアルタイムで確認できるようになった
