## 要約

2026 年 6 月 1 日の ChatGPT Release Notes に、**求人検索（job search）** と **履歴書（resume）** 機能が追加されました。ChatGPT が Indeed・Upwork・Appcast をはじめ Web 全体から **ライブの求人・フリーランス案件を横断して提示**し、履歴書のアップロード・作成・案件に合わせた整形ダウンロードまで支援します。OpenAI は「求人の発見から履歴書の最適化まで、求職プロセスのより多くの部分を ChatGPT が支援できるようになった」と説明しています。

> 出典の ChatGPT Release Notes（help.openai.com）は当環境では本文の直接取得が制限されるため、日付・内容は公開リリーストラッカー経由で確認し、一次 URL を出典として記録しています。

## 何が変わったか

- **求人検索**: Indeed / Upwork / Appcast などのソースと Web 全体から、ライブの求人・フリーランス案件を横断提示。
- **履歴書サポート**: 履歴書のアップロード、作成、プロフェッショナルに整形したフォーマットでのダウンロード。
- チャット内で「希望条件に合うライブ求人を見つける → 履歴書を案件向けに調整する」という一連の流れを完結できる。
- **提供範囲**: 求人検索は **米国ユーザーの全プラン（Free / Go / Plus / Pro）**。履歴書整形は **web・英語で全プラン（グローバル）**。

## 業務インパクト（一般企業向け）

求人検索・履歴書は一見コンシューマ向けですが、**人事・採用・キャリア支援** の観点で押さえておく価値があります。社員のキャリア面談や履歴書ブラッシュアップの自己支援、再就職支援（アウトプレースメント）、社内公募の準備など、ChatGPT が「キャリア領域の入口ツール」として現場に入り込む余地が広がります。

ただし日本企業にとっては提供条件が制約になります。**求人検索は米国限定、履歴書整形は英語** のため、日本拠点でそのまま業務適用するのは難しく、「履歴書整形は英語 web、求人検索は現状米国」という条件を周知しないと現場で誤解が生まれます。

情シス・人事は、社員が業務 PC の ChatGPT に職務経歴・個人情報を入力するケースを想定しておくべきです。特に在職中の転職活動に関わる情報や、機微な個人データの取り扱いは、利用ガイドラインで明示しておくとトラブルを避けられます。

## 副業・個人活用視点

この機能が最も刺さるのは **副業・フリーランス志向の個人** です。求人検索が Upwork や Appcast などの **フリーランス案件** を横断提示するため、「自分のスキルに合う案件をチャットで探す → その案件向けに履歴書・職務経歴を整える」という流れを ChatGPT 一つで回せます。

履歴書整形は英語 web で全プランに提供されるため、**英語の職務経歴書（レジュメ）を案件ごとに最適化** したい人には今すぐ使えます。海外のフリーランスプラットフォームで受注を狙う層には実用的です。

日本国内の求人をメインに探す場合は、求人検索の米国限定が効いてくるため、当面は「英語レジュメの整形・案件向けカスタマイズ」を主用途と捉えるのが現実的です。職務経歴の棚卸しや案件向けの言い換えは、汎用的な ChatGPT 活用としても応用が利きます。
