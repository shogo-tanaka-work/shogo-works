## 要約

**OpenAI API の Web search ツールが、テキスト検索結果と並行して画像結果も返せる**ようになりました。製品写真・ランドマーク・イベントなど、視覚的な参照が必要なアプリケーション向けの拡張です。テキストと画像を同一レスポンスで扱えます。

## 何が変わったか

- **Web search ツールが画像結果を返却**可能に（テキスト結果と並行）。
- 製品写真・ランドマーク・イベントなど、視覚的参照が必要なユースケースに対応。

## 業務インパクト（一般企業向け）

これまで Web search ツールはテキスト中心でしたが、画像結果が返ることで「見た目」が判断材料になる用途をカバーしやすくなります。たとえば、社内の商品検索アシスタント、出張・観光案内、イベント情報の提示などで、テキストの説明に画像を添えて提示できます。レスポンスにテキストと画像が並ぶ前提で、表示 UI と後段の処理（画像のキャッシュ・出所表示など）を設計しておくとよいでしょう。

## 副業・個人活用視点

個人開発では、商品・場所・イベントを扱うアプリを Web search ツールだけで完結させやすくなります。たとえば、ガジェット比較ボットや旅行プラン作成ツールで、テキストの根拠に加えて画像を提示できると体験が大きく上がります。クライアント案件でも、視覚情報を含む検索体験を一段リッチにする手段として提案できます。
