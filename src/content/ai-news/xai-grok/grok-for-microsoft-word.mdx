## 要約

xAI の Grok が Microsoft 365 の公式アドインとして Microsoft Word に統合されました。Word のリボンからアドインをインストールするだけで、サイドパネルに Grok が起動し、文書の作成・編集・校正・文体改善をアシストします。

最大の特徴は**リアルタイム Web 検索と X (旧 Twitter) データへのアクセス**です。Microsoft Copilot との差別化として明示されており、常に最新情報を参照しながら文書を作成できます。また、メール・SharePoint・Google Drive のコネクタで外部コンテキストを文書内に引き込む機能も搭載しています。

ユーザーは Grok が提案したインライン編集候補を採用・取り消しできる形式で、オーバーライトではなく提案型の UI を採用しています。

## 何が変わったか

- Word サイドパネルから Grok によるドラフト作成・編集・校正・フォーマット統一が可能に
- リアルタイム Web 検索と X データを Word 文書の文脈で参照できる
- メール・SharePoint・Google Drive コネクタで外部ドキュメントのコンテキストを補強できる
- Microsoft 365 ストアからインストール、既存の Word 環境に追加するだけで利用可能
- インライン編集候補として提示され、採用・取り消しをユーザーがコントロール

## 業務インパクト（一般企業向け）

企業での Word 利用シーンで AI アシスタントを使いたいが、Microsoft Copilot の契約コストや機能制限が課題という場合に、代替の選択肢として評価できます。

特に注目すべき点が**リアルタイム検索**です。競合他社の最新情報・市場動向・法改正などを参照しながら文書を作成する業務（市場調査レポート・提案書・プレスリリースなど）では、Copilot よりも有利になる可能性があります。

一方で社内データ連携の観点では、SharePoint コネクタを通じた社内文書へのアクセスが可能ですが、セキュリティポリシーとの整合を確認してから展開する必要があります。情シス部門は、個人が勝手にアドインをインストールするリスクも含めた管理ポリシーの見直しを検討する時期かもしれません。

## 副業・個人活用視点

Microsoft 365 を使っているすべての職種が対象になるため、**ホワイトカラー向け AI 活用サポート**という副業文脈で即応用できる事例です。

提案書・報告書・メール文書の効率化を支援するコンサルや研修を行っている場合、「Word からそのまま使える AI」という導入障壁の低さはセールスポイントになります。試用から始めやすいアドイン形式のため、AI ツール導入経験の少ない層へのファーストステップとして紹介しやすいです。
