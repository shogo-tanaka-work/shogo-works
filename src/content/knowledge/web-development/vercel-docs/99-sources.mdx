## このページの役割

本シリーズ各章で参照した一次ソース（Vercel 公式ドキュメント・Changelog・Knowledge Base・Academy）を章順に一覧化する。記載 URL は 2026-05-08 時点で取得確認済みのものを基本とするが、Vercel のドキュメントは canonical URL の改名が起こり得るため、404 になっていた場合は [Vercel Documentation トップ](https://vercel.com/docs) から再検索する。

robots.txt 上の指示（`ai-train=no` / `ai-input=yes` / `search=yes`）に従い、本シリーズは「教材化（参照・引用・要約）」の用途で出典明示を徹底している。AI モデル学習用途には使わない。
