## Claude Codeとは

Claude Codeは、Anthropicが提供する公式CLIツールです。ターミナルから直接Claudeと対話しながら、コードの読解・編集・実行・Git操作まで一貫して行えます。

従来のAIコーディングツールとの最大の違いは、**エージェント型のワークフロー**を実現している点です。単にコードを生成するだけでなく、ファイルの読み込み、テストの実行、エラーの修正までを自律的に行います。

## 基本的な使い方

### インストール

```bash
npm install -g @anthropic-ai/claude-code
```

### 起動

```bash
claude
```

プロジェクトのルートディレクトリで起動すると、コードベース全体をコンテキストとして認識します。

## 開発現場での活用パターン

### 1. コードリーディング

大規模なコードベースの理解に特に有効です。「この関数の呼び出し元を全て教えて」「このモジュールのアーキテクチャを説明して」といった質問に、実際のコードを根拠にして回答します。

### 2. TDD支援

テスト駆動開発との相性が良いです。

1. まずテストを書く（または書いてもらう）
2. テストを実行して失敗を確認
3. 実装を依頼
4. テストがパスすることを確認

このサイクルをClaude Code内で完結できます。

### 3. リファクタリング

既存コードの改善も得意分野です。型安全性の向上、重複の削除、責務の分離など、具体的な改善指示を出すと効果的です。

## CLAUDE.mdによるプロジェクト設定

プロジェクトルートに `CLAUDE.md` を配置すると、プロジェクト固有のルールやコンテキストを設定できます。

```markdown
# プロジェクト規約
- TypeScript厳密モード
- テストはvitest
- コミットメッセージはConventional Commits形式
```

これにより、プロジェクトの規約に沿った提案が得られます。

## まとめ

Claude Codeは「AIペアプログラマー」という表現が最も適しています。コードの文脈を深く理解した上で、実用的な提案と実行を行ってくれるツールです。

特に以下のような場面で効果を発揮します。

- **新しいコードベースのキャッチアップ**
- **TDDサイクルの高速化**
- **定型的なリファクタリング作業**
- **ドキュメントの自動生成**

まずは小さなタスクから試してみることをおすすめします。
