## 概要

**Codex Marketplace** は、2026年4月15日にリリースされた v0.121.0 で追加されたプラグインエコシステムです。コミュニティや企業が作成したプラグインを `codex marketplace add` コマンドで追加できます。

「Codex for (almost) everything」（4/16）の発表で公開された90以上のプラグインには、Atlassian Rovo・CircleCI・GitLab Issues・Microsoft Suite・Life Sciences ツール等が含まれます。

---

## 基本的な使い方

### プラグインの追加

```bash
# GitHub のマーケットプレイスを追加
/plugin marketplace add openai/codex-plugin-cc

# git URL で追加
/plugin marketplace add https://github.com/your-org/your-plugin.git

# ローカルディレクトリから追加
/plugin marketplace add ./plugins/my-plugin
```

### プラグインのインストール

```bash
# プラグインをインストール
/plugin install codex@openai-codex

# インストール済みプラグインを表示
/plugin list

# リロード（設定変更後）
/reload-plugins
```

---

## marketplace.json の構造

チーム・組織向けのプライベートマーケットプレイスを作成する場合は `marketplace.json` を定義します。

```json
{
  "name": "my-team-marketplace",
  "interface": { "displayName": "My Team Plugins" },
  "plugins": [
    {
      "name": "my-plugin",
      "source": {
        "source": "local",
        "path": "./plugins/my-plugin"
      },
      "policy": {
        "installation": "AVAILABLE",
        "authentication": "ON_INSTALL"
      },
      "category": "Productivity"
    }
  ]
}
```

### 配置場所

| スコープ | パス |
|---|---|
| リポジトリ（チーム共有） | `$REPO_ROOT/.agents/plugins/marketplace.json` |
| 個人（ホームディレクトリ） | `~/.agents/plugins/marketplace.json` |

## codex-plugin-cc：Claude Code × Codex 連携

OpenAI が公式提供する Claude Code 向けプラグインです。Claude Code のセッション内から Codex にタスクを委譲できます。

### インストール

```bash
/plugin marketplace add openai/codex-plugin-cc
/plugin install codex@openai-codex
/reload-plugins
/codex:setup
```

### 主要コマンド

```bash
# 未コミット変更のコードレビュー
/codex:review

# mainブランチとの比較レビュー
/codex:review --base main

# 設計の問題点を対抗的に指摘（バックグラウンド実行）
/codex:adversarial-review --background

# バグ調査をCodexに委譲
/codex:rescue investigate why tests fail

# バックグラウンドタスクの確認・結果取得
/codex:status
/codex:result
/codex:cancel
```

### 使い分けの考え方

Claude Code（Anthropic）と Codex（OpenAI）は競合ツールですが、得意領域が異なります。

| 局面 | 推奨ツール |
|---|---|
| 複雑な設計相談・アーキテクチャ判断 | Claude Code |
| 量産系コードレビュー・CI/CD自動化 | Codex |
| ペアプロ的なインタラクティブ作業 | Claude Code |
| バックグラウンドでの並列処理 | Codex |
