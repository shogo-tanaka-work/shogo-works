# 2026-06-16 Gemini in Chrome 多言語展開

## 公式内容の日本語要約

Chrome ブラウザに統合された Gemini アシスタントがラテンアメリカ・アフリカ・中東を含む新地域へ展開拡大。記事要約・複雑な概念の説明・タブコンテキストに基づく情報検索・メール/SNS 投稿作成・画像生成・Gemini Live による双方向音声会話などが新地域のユーザーで利用可能になる。ロールアウト開始は 2026-06-10 で Rapid/Scheduled Release 両ドメインで 15 日以上かけて段階展開。

## できるようになったこと

- ラテンアメリカ・アフリカ・中東の新地域でブラウザ上部の Gemini アイコンから AI アシスタントを利用可能
- 記事要約、複雑な概念の説明、タブコンテキストに基づく情報検索
- メール・SNS 投稿作成、画像生成
- Gemini Live による双方向音声会話（ブラウザ内）
- 管理者はドメイン・OU・グループ単位で有効/無効を制御可能（ChromeOS / macOS / Windows）

## 影響範囲

- 対象ユーザー: 全 Google Workspace プラン / Workspace Individual / 個人 Google アカウント（新地域）
- 対象プラン: 全プラン
- API / UI / 管理者機能: 管理コンソールで OU・グループ単位の有効/無効制御あり
