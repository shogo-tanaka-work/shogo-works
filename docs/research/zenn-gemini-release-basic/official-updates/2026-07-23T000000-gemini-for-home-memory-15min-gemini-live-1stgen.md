# 2026-07-23 Gemini for Homeが会話メモリ15分に拡大、第1世代デバイスにGemini Live対応

## 公式内容の日本語要約

Gemini for Home全体の会話メモリ（コンテキスト保持）ウィンドウが15分に拡大された。例えば「キッチンの電気をつけて」と言った数分後に「50%に暗くして」と言っても、直前の文脈を理解して対応できるようになった。また、Google Home Premium加入者は、第1世代のGoogle Home MiniおよびGoogle Nest Hubでも「Hey Google, let's chat」でGemini Liveの双方向会話を開始できるようになった（従来は新しいデバイス世代が中心だった機能の対応拡大）。

## できるようになったこと

- 会話メモリウィンドウが15分に拡大し、フォローアップの指示を文脈保持したまま実行可能
- 第1世代Google Home Mini / Google Nest HubでGemini Liveの双方向会話に対応

## 影響範囲

- 対象ユーザー: Google Home / Nest製品ユーザー
- 対象プラン: Gemini Live機能はGoogle Home Premium加入者が対象
- API / UI / 管理者機能: デバイス側音声アシスタント機能（Google Home アプリ経由）

## 教材化メモ

「新しいAI機能は最新ハードウェアだけ」という思い込みに対する反例。第1世代デバイスへの機能拡大は、ソフトウェア側改善（Gemini Live対応拡大）だけで既存ハードウェアの体験を底上げできる例として紹介しやすい。
