# 2026-07-01 Google AI June 2026 まとめ

## 公式内容の日本語要約

Google が 2026年6月の主要AI発表をまとめたブログ記事を公開（2026-07-01）。Gemini 3.5 Flash の Computer Use 対応（2026-06-24 発表済み）に加え、以下の新機能が含まれる。

## できるようになったこと

- **Gemini 3.5 Live Translate**: 70言語以上のリアルタイム音声翻訳。話者の抑揚を保持しながら多言語通話・会議で利用可能
- **Gemini Omni Flash**: API 経由でパブリックプレビュー開始（2026-06-30〜）。カスタム動画ワークフロー構築用途
- **Gemini Home スピーカー**: Gemini 組み込みの Google Home スピーカーを発表。自然な会話と複数リクエスト同時処理をサポート
- **Gemini 学習ノートブック**: Gemini アプリに学習目標設定・進捗追跡機能（教育向け）

## 影響範囲

- Live Translate: Gemini アプリおよび Google Meet 連携
- Omni Flash: Gemini API 利用者（パブリックプレビュー段階）
- Gemini Home: スマートホームユーザー（新ハードウェア）

## 教材化メモ

- リアルタイム音声翻訳（70言語・抑揚保持）は多言語業務の文脈で使える
- Omni Flash はカスタム動画ワークフロー構築の API 利用例として参照可能
