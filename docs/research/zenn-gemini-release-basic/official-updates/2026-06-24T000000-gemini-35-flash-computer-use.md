# 2026-06-24 Gemini 3.5 Flash Computer Use 統合

## 公式内容の日本語要約

Gemini 3.5 Flash に Computer Use（コンピューター操作）機能が統合された。ブラウザ・モバイル・デスクトップの 3 環境を「見て・推論して・操作する」エージェント機能で、Gemini API および Gemini Enterprise Agent Platform から利用できる。プロンプトインジェクション対策の敵対的トレーニング済みで、センシティブな操作への人間承認確認オプションも提供。

## できるようになったこと

- Gemini 3.5 Flash 単一モデルでブラウザ・モバイル・デスクトップの 3 環境を自律操作
- ソフトウェアテスト自動化・業務アプリケーションのワークフロー自動実行
- プロンプトインジェクション対策の敵対的トレーニングによる安全対策
- センシティブな操作への人間承認確認（オプション）
- 間接的プロンプトインジェクション検出（オプション）
- Browserbase によるデモ環境・GitHub でリファレンス実装コードを公開

## 影響範囲

- 対象ユーザー: 開発者・エンタープライズ（Gemini API / Enterprise Agent Platform 経由）
- 対象プラン: Gemini API・Gemini Enterprise Agent Platform
- API / UI: Gemini API のエージェント機能として利用
