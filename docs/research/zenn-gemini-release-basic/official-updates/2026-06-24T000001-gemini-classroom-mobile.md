# 2026-06-24 Gemini in Google Classroom 機能拡張

## 公式内容の日本語要約

Google Classroom の Gemini タブが 2 つの新機能を追加した。①Android・iOS のモバイルアプリから Gemini タブが利用可能に。②「Nano Banana 2」画像生成モデルを使ったインフォグラフィック・コミックストリップ・概念図の生成機能と、Gemini Canvas による 3 種のスタータープロンプト（プレゼンテーション作成・インタラクティブアクティビティ・ファイルのスライド変換）を追加。教師向け機能で 18 歳以上が対象。

## できるようになったこと

- Classroom アプリ（Android・iOS）から Gemini タブで「クイズ生成・ブレスト・導入フック作成」をモバイルから直接実行
- Nano Banana 2 モデルでインフォグラフィック・コミックストリップ・概念図を生成
- Gemini Canvas 上で「プレゼンテーション作成・インタラクティブアクティビティ・ファイルのスライド変換」のスタータープロンプト 3 種を提供

## 影響範囲

- 対象ユーザー: Google Classroom を使う教師（18 歳以上）
- 対象プラン: Education Fundamentals・Standard・Plus
- API / UI: Classroom モバイルアプリ・Gemini Canvas

## 教材化メモ

- 「教育向け AI ツール」の具体例として、生成 AI が教育現場でどう使われているかを示せる
- "Nano Banana 2" という命名は教材で紹介すると記憶に残りやすい（Google の研究チームが使う内部コードネーム文化の例）
- AIの出力を「生徒に共有する前に確認・調整」という Google の公式注意書きは、教育現場における AI 活用のリテラシー教材として使える
