# 2026-05-12 Gemini LTI / Assignments LTI Moodle 対応

## 公式内容の日本語要約

Google Workspace Assignments LTI と Gemini LTI が Moodle に対応した。Canvas / Schoology に加えて Moodle でも、Gemini app や NotebookLM などの Google AI tools を LMS 内から利用できる。

## できるようになったこと

- Moodle から Gemini LTI と Assignments LTI を利用
- 学生・教員が LMS 内で Gemini app / NotebookLM を使った学習支援を受けられる
- Moodle 管理者は LTI Dynamic Registration specification を使った登録が可能

## 影響範囲

- 対象ユーザー: Google Workspace for Education の管理者、教員、学生
- 対象プラン: Education Fundamentals / Standard / Plus
- API / UI / 管理者機能: Moodle LMS 連携、Admin console での Workspace LTI / Gemini service 設定

## 教材化メモ

教育機関向けAI導入の教材で、LMS内にAI支援を埋め込む実例として扱える。NotebookLM の course-specific guidance や audio overview とセットで説明しやすい。
