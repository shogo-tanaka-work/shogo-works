# 2026-05-19 Gemini Spark Ultra ロールアウト（追補）

## 公式内容の日本語要約

Google I/O 2026 で発表された Gemini Spark（24 時間 365 日稼働するパーソナル AI エージェント）が、Trusted Tester 向け展開を経て、**Google AI Ultra 加入者向けベータを 5/22 前後の週から順次開始**している。米国・18 歳以上が対象。Gemini 3.5 Flash で動作し、Gmail・Google Docs などの Workspace アプリとのネイティブ統合を最初から備える。発表記事の公開は 2026-05-19、ロールアウト窓が本日次レポートの対象期間にかかるため `category: rollout` として追補記録する。

## できるようになったこと

- パーソナル AI エージェント Gemini Spark が **AI Ultra 加入者向けベータ**として使えるようになった（米国・18 歳以上、ベータ）。
- Gmail・Google Docs などの Workspace アプリとのネイティブ統合がデフォルト。
- 月額 100 ドルの AI Ultra と 200 ドルの AI Ultra の両プランで利用可能。
- macOS アプリへの統合も予告（後続展開）。

## 影響範囲

- 対象ユーザー: 米国の Google AI Ultra 加入者（18 歳以上）。
- 対象プラン: AI Ultra（$100 / $200）。日本など他国・他プランへの展開時期は本リリース対象外。
- API / UI / 管理者機能: Gemini アプリ / Web 上のパーソナルエージェント機能、Workspace アプリ連携、macOS 統合（予告）。

## 教材化メモ

- 「パーソナル AI エージェント」が ChatGPT / Claude / Grok / Gemini の各プラットフォームで競合カテゴリ化していることを示す事例。教材ではプラットフォーム横断のエージェントカテゴリ整理に組み込むと良い。
- 日本市場での利用可能タイミングは未公表のため、Knowledge には地域・プラン限定の前提を明示して扱う。
