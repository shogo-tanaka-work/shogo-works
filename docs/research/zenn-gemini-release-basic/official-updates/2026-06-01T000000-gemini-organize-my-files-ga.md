# 2026-06-01 Gemini「Organize My Files in Drive」GA

## 公式内容の日本語要約

2026-06-01 の Google Workspace Updates で、**Organize My Files in Drive** が GA（一般提供）になったと告知された。2025-10 のベータから昇格。Gemini が散在したファイルの内容・既存フォルダ構造・整理のベストプラクティスを踏まえ、「どのフォルダに移動すべきか」「新規フォルダを作るべきか」を提案する。ユーザーは My Drive や親フォルダの「Suggest File Moves」から提案を確認し、レビューしてから一括移動できる。

## できるようになったこと

- **整理提案**: Gemini が「既存フォルダへの移動」と「関連ファイル群の新規フォルダ作成」の2セクションで提案。
- **エントリポイント**: My Drive・親フォルダ内の「Suggest File Moves」から提案ビューにアクセス。
- **レビュー後に適用**: 提案は確認してから一括でファイルを移動。
- **プロモ枠**: 2026-07-15 まで、Workspace 顧客は Organize My Files の上限が引き上げられたプロモ枠で試用可能。以降は通常のユーザー単位上限。

## 影響範囲

- 対象ユーザー / プラン: Business Standard / Plus、Enterprise Standard / Plus、Google AI Pro / Ultra（コンシューマ）、Google AI Pro for Education、AI Expanded Access。
- ロールアウト: Rapid Release / Scheduled Release ともに「Available now」、英語でグローバル。
- 前提: 管理者が Drive で Gemini for Workspace を有効化し、エンドユーザー側で Workspace smart features を有効化する必要がある。

- Gemini for Workspace / Drive 活用教材に「Organize My Files（Suggest File Moves）GA」を追記。ベータからの昇格、対象エディション、英語先行、プロモ枠（7/15 まで）を明示。
