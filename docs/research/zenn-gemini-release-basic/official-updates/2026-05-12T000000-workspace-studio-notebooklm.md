# 2026-05-12 Workspace Studio NotebookLM 連携

## 公式内容の日本語要約

Google Workspace Studio に NotebookLM が統合され、既存の notebook を自動化フローの AI 知識ソースとして使えるようになった。新しい Ask NotebookLM step により、notebook 内の要約・調査・洞察に基づく grounded response を生成できる。

## できるようになったこと

- Workspace Studio のフロー内で NotebookLM を参照
- 既存 notebook を業務自動化の知識ソースとして再利用
- Gemini for Google Workspace steps を許可している環境では既定で利用可能

## 影響範囲

- 対象ユーザー: Workspace Studio 利用者、管理者
- 対象プラン: Business Starter / Standard / Plus、Enterprise Standard / Plus、Education、Google AI Pro for Education、Teaching and Learning、AI Expanded Access
- API / UI / 管理者機能: Workspace Studio step、管理者の step / starter アクセス制御

## 教材化メモ

NotebookLM を「資料置き場」ではなく「自動化の根拠データ」にする教材に向く。社内FAQ、授業資料、営業資料の notebook をフローに接続する演習が作れる。
