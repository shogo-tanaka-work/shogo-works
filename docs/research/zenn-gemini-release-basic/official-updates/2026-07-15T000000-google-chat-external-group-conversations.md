# 2026-07-15 Now available: group conversations with external collaborators in Google Chat

## 公式内容の日本語要約

これまでGoogle Chatで社外ユーザーと会話できるのは1対1チャット、または外部アクセスを有効化したスペースに限られていたが、複数の社外ユーザーを含むグループDM（グループ会話）の作成が可能になった。既存の外部スペースと同じ管理者制御が適用され、社外参加者はメール招待を承諾して参加する形式。会話画面には社外メンバーが含まれていることを示すバッジが表示される。Rapid Releaseでの展開完了目標は2026-07-17、Scheduled Releaseは2026-07-24から開始予定。

## できるようになったこと

- 複数の社外ユーザーを含むグループDM（グループ会話）の作成
- 社外参加者はメール招待の承諾で参加
- 会話画面に社外メンバー参加を示すバッジを表示

## 影響範囲

- 対象ユーザー: Google Chatで社外パートナーと協業するユーザー・管理者
- 対象プラン: Workspace各種エディション（外部アクセス管理者設定に準拠）
- API / UI / 管理者機能: Google Chat管理者制御、外部アクセス設定

## 教材化メモ

- Gemini自体の新機能ではないが、Gemini Canvas共有や外部パートナーとの協業ワークフローに影響する基盤アップデートとして関連メモに留める。
