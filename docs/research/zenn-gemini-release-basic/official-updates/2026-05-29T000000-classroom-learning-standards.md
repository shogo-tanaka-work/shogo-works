# 2026-05-29 Google Classroom 学習スタンダード対応

## 公式内容の日本語要約

Google は Google Classroom で **学習スタンダード（learning standards）とスキル** を用いた生徒進捗管理を一般提供開始した。教師は coursework や rubric に学習目標をタグ付けでき、**AI が課題内容を解析して関連する学習目標を提案** する。1EdTech および Common Good Learning Tools との提携を通じて、世界各国の学習標準が Classroom 内から検索・適用可能。

対象は **Workspace for Education Fundamentals / Standard / Plus 全ティア**。利用可能地域は **米国・カナダ・英国・ブラジル・日本・メキシコ・イタリア・オーストラリア**。Learning skills は **英語・スペイン語・イタリア語・日本語・韓国語・マレー語・ブラジルポルトガル語** に対応。ロールアウトは **2026-05-27 から段階開始**、Rapid Release / Scheduled Release ドメイン全体で完了まで最大 15 日。

## できるようになったこと

- **Smart Tagging**: 教師は学習目標を検索するか、AI 提案を受けて課題・教材にタグを付与できる。
- **Performance Analytics**: 個別学習者・クラス全体・特定学習領域単位で達成データを可視化。
- **Global Standards Access**: 1EdTech / Common Good Learning Tools との提携で各国標準へアクセス。
- **Stakeholder Transparency**: 生徒・保護者にも紐付いた学習目標が見える。
- **Admin Visibility**: Education Plus 管理者は "Visit a class" 機能で標準への整合性を監視可能。

## 影響範囲

- 対象ユーザー: 教育機関の教師・管理者・生徒・保護者
- 対象プラン: Workspace for Education Fundamentals / Standard / Plus（管理コンソール設定不要、デフォルト ON）
- API / UI / 管理者機能: Classroom UI（教師・生徒・管理者）

## 教材化メモ

Workspace for Education 向けの社内教材があれば「AI による学習目標タグ提案」を **AI 導入事例** として節追加。日本語にも対応している点で、国内教育機関の運用に直結する。Workspace AI が教育セグメント単独向け機能を出している点は、Workspace ガイダンスの全体観でも触れる価値がある。
