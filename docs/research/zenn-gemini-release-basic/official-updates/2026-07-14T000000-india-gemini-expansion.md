# 2026-07-14 Deepening our commitment to India's AI ambition

## 公式内容の日本語要約

Google I/O Connect India 2026で、インド向けにGemini関連の複数施策が発表された。(1) Atal Innovation Missionと連携し、教員向けAIアシスタント「ATL Saathi」（Gemini搭載のデスクトップWebアプリ）をまず100校でパイロット導入し、将来的に約1万校への展開を目指す。(2) Gemini Liveの対応言語をサンスクリット語・ボージュプリー語・マイティリー語を含む25以上のインドの言語・方言（音声・テキスト）に拡大。(3) インド国内でのデータ処理を保証した形で、Gemini Enterprise Agent PlatformおよびアプリでGemini 3.5 Flashをインド企業向けに提供開始。

## できるようになったこと

- 教員向けAIアシスタント「ATL Saathi」のパイロット導入（100校、将来1万校へ拡大予定）
- Gemini Liveが25以上のインド言語・方言（音声・テキスト）に対応
- インド国内データ処理保証付きでGemini 3.5 Flashをインド企業向けに提供開始

## 影響範囲

- 対象ユーザー: インドの教育機関・教員、インド企業のGemini Enterprise利用者
- 対象プラン: Gemini Enterprise Agent Platform、Geminiアプリ
- API / UI / 管理者機能: Gemini Live言語設定、Gemini Enterprise Agent Platformのモデル選択、データローカライゼーション

## 教材化メモ

- データローカライゼーション（インド国内処理保証）付きでのエンタープライズモデル提供は、日本を含む他地域でのローカライゼーション対応の先行事例として参考になる。
- 教員向けAIアシスタントのパイロット展開は、教育領域でのAI導入プロセス（小規模パイロット→段階的拡大）のケーススタディとして紹介できる。
