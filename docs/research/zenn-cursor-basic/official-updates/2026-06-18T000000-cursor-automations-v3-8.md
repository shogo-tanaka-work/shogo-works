# 2026-06-18 Cursor Automations v3.8

## 公式内容の日本語要約

Cursor Automations の大幅強化。エージェントセッション内で自然言語からオートメーションを生成する `/automate` スキルの追加、Slack 絵文字トリガー、GitHub の新トリガー5種、クラウドエージェントへのコンピュータ使用権限付与など。

## できるようになったこと

- `/automate` コマンドでエージェントセッションから直接オートメーション作成（自然言語でトリガー・指示・ツールを自動構成）
- Slack の絵文字リアクションをオートメーションのトリガーとして設定
- GitHub 新トリガー5種追加: Issue コメント / PR レビューコメント / PR レビュー提出 / レビュースレッド更新 / GitHub Actions ワークフロー完了
- クラウドエージェントが独自のコンピュータを使用してデモや成果物を自動生成（デフォルト有効）
- 未完成状態でのオートメーション保存、PR の自動オープン、メモリファイルの削除

## 影響範囲

- 対象ユーザー: Cursor を使う開発者・エンジニア
- 対象プラン: Automations 機能を利用中のプラン
- API / UI / 管理者機能: Cursor IDE / Cloud

## 教材化メモ

- 「言語でオートメーションを作る」ことのハードルが下がった事例
- Slack + GitHub の組み合わせで開発ワークフロー全体を自動化できる点が実務に近い
