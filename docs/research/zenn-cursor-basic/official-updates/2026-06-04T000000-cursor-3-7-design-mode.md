# 2026-06-04 Cursor 3.7（Design Mode / Context Usage Report）

## 公式内容の日本語要約

Cursor 3.7 がリリースされた。Canvas に「Design Mode」が加わり、UI 要素を Canvas 上で直接選択・注釈して Cursor の編集を誘導できる。あわせて、システムプロンプト・ツールなどコンポーネント別のトークン配分を示すインタラクティブなコンテキスト使用状況レポート、全画面 Canvas 共有、プロンプトを実行する埋め込みボタンが追加された。

## できるようになったこと

- **Design Mode（Canvas）**: UI 要素を Canvas 上で選択・注釈し、自然言語ではなく対象を指し示して編集を指示できる。
- **コンテキスト使用状況レポート**: トークンがシステムプロンプト / ツール / その他にどう配分されているかをインタラクティブに可視化。
- **全画面 Canvas 共有**、**特定プロンプトを実行する埋め込みボタン**。

## 影響範囲

- 対象ユーザー: Cursor 利用者全般。フロントエンド / UI 作業者に特に有効。
- 対象プラン: 記載範囲では全般。
- API / UI / 管理者機能: エディタ UI（Canvas / Design Mode / コンテキストレポート）。

## 教材化メモ

Cursor 教材の「Canvas」章に Design Mode を追記し、UI 要素を指し示す編集（自然言語指示との使い分け）を示す。コンテキスト使用状況レポートは「トークン / コンテキスト管理」観点で、長文セッションのコスト把握・削減に活用する例を加える。
