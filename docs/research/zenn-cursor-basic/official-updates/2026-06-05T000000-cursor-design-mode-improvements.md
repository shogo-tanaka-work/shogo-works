# 2026-06-05 Cursor Design Mode の改善

## 公式内容の日本語要約

Cursor の changelog に 2026-06-05 付で Design Mode の改善が掲載された。ブラウザ上で2つ以上の要素をまとめて選択し、エージェントに複数コンポーネントを同時調整させられる。さらに、エージェント実行中に Design Mode オーバーレイから音声で変更をナレーションでき、待たずに後続の変更をキューイングできる。

## できるようになったこと

- **複数要素の同時選択**: ブラウザで2つ以上の要素をまとめて選び、グループとして同時調整。
- **音声ナレーション指示**: エージェント実行中に音声で変更を伝え、後続変更をキュー投入。
