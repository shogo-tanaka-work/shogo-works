# 2026-06-17 Cursor v3.7

## 公式内容の日本語要約

Cursor v3.7 がリリースされた。主要機能は Cloud Environment Setup（10分以内でクラウド開発環境を構築、再利用可能スナップショット）と Cloud Subagents（`/in-cloud` コマンドで専用 VM にサブエージェント起動）。複数クラウドエージェントの並列実行とデバイス間ハンドオフに対応。

## できるようになったこと

- クラウド上に10分以内で開発環境を構築し、スナップショットで高速再起動
- `/in-cloud` コマンドで専用 VM にサブエージェントを起動
- 複数のクラウドエージェントを並列実行しながらローカル環境をクリーンに保つ
- 作業中にデバイスを切り替えてもセッションを引き継げる（デバイス間ハンドオフ）

## 影響範囲

- 対象ユーザー: 全 Cursor ユーザー（プラン詳細は公式確認要）
- 対象プラン: Business / Pro など（詳細は公式確認要）
- API / UI / 管理者機能: `/in-cloud` CLI コマンド、クラウド環境管理 UI
