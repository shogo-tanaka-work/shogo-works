# 2026-06-04 n8n@2.23.3 / n8n@1.123.52

## 公式内容の日本語要約

n8n stable 2.23.3 がリリースされた。内容はバグ修正 1 件のみで、評価実行（evaluation executions）が status `new` のまま停滞する問題を修正した。あわせて 1.x 系の backport である 1.123.52 も同日公開された。プレリリースの 2.25.3（beta）も同日付で出ている。

## できるようになったこと

- **2.23.3（stable・バグ修正のみ）**: 評価実行が status `new` で停滞する問題を修正（#31702）。
- **1.123.52（1.x backport）**: 同系統のバグ修正。
- 2.25.3 は pre-release（beta）。

## 影響範囲

- 対象ユーザー: n8n セルフホスト / クラウド利用者。
- 対象プラン: 制限なし。
- API / UI / 管理者機能: 評価実行（ワークフロー評価）の安定性。

## 教材化メモ

バグ修正のみのパッチであり、教材化の必要は薄い。評価実行を使っている場合のみ「status new 停滞の修正」を運用メモに残す程度。
