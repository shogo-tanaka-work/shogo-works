# 2026-05-13 n8n 2.20.7

## 公式内容の日本語要約

n8n 2.20.7は、OpenAI vendor defaultsを適用する際にAxiosHeaders instanceを保持するcore bug fixを含む安定版リリース。

## できるようになったこと

- OpenAI連携周辺で、HTTP headersの扱いが崩れる可能性を抑えられる。

## 影響範囲

- 対象ユーザー: n8n 2.20系利用者、OpenAI node / AI連携を使うワークフロー運用者
- 対象プラン: self-hosted / cloud
- API / UI / 管理者機能: core / OpenAI vendor defaults

## 教材化メモ

OpenAI連携を含むn8n運用では、小さなpatchでもヘッダーや認証まわりの安定性に影響する。AIワークフローの障害調査教材で、vendor defaultsとHTTP client設定の確認ポイントとして扱える。
