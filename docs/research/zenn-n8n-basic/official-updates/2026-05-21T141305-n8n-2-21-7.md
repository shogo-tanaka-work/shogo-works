# 2026-05-21 n8n@2.21.7（stable）

## 公式内容の日本語要約

n8n は 2026-05-21 に stable リリース **2.21.7** を公開した（同日先行で 2.21.6 もリリース。2.21.7 は 2.21.6 の修正を包含）。内容はコア層のバグ修正のみで、新機能の追加はない。

## できるようになったこと（修正内容）

- スケジュール実行（scheduled poll）で expression 用の isolate を確実に取得するよう修正。
- webhook 実行データ経路で manual user id を正しく設定するよう修正。
- スケジュール poll の isolate 取得失敗を `__emitError` 経由で報告するよう修正。
- AI ベンダーノードで、API 呼び出し前に空でないプロンプトを検証するよう修正。

## 影響範囲

- 対象ユーザー: n8n セルフホスト / クラウド利用者全般。特にスケジュールトリガー・webhook・AI ノードを使うワークフロー。
- 対象プラン: stable チャネル。
- API / UI / 管理者機能: コア実行エンジンのバグ修正。

## 教材化メモ

バグ修正のみのパッチリリース。AIニュース化は見送り、教材反映も不要。窓内 stable の記録としてのみ保持。
