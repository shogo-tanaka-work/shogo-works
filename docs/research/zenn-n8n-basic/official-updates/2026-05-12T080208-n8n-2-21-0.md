# 2026-05-12 n8n 2.21.0

## 公式内容の日本語要約

n8n 2.21.0 が pre-release として公開された。AI Builder、agent chat、MCP、workflow SDK、評価基盤、credential handling、複数trigger/nodeの webhook verification など、AI workflow / automation の実運用に関わる変更が多い。

## できるようになったこと

- AI Builder で archived workflow 復元、planning context 保持、terminal state の user-visible output 保証を改善
- core 側で Instance AI の liveness timeout、workflow structure validation、MCP execution hang 対応、credential resolution scope などを追加・修正
- Acuity / Asana / Calendly / GitLab / Trello / Twilio など複数 trigger node に webhook request verification を追加
- Figma Trigger Node の OAuth2、Jira Node の OAuth2 3LO、Outlook calendar event fields など連携機能を拡張

## 影響範囲

- 対象ユーザー: n8n のAI機能、Agent、MCP、workflow automation 利用者
- 対象プラン: n8n self-hosted / cloud
- API / UI / 管理者機能: AI Builder、core workflow engine、MCP tools、trigger nodes、credential handling

## 教材化メモ

n8n を AI agent workflow platform として扱う教材で、AI Builder、MCP tool、credential / webhook verification をまとめて紹介できる。pre-release のため、本番導入では stable 反映を確認する扱いにする。
