# 2026-07-02 n8n 2.29.3（pre-release）— Instance AI課金の専用クレジットプール化

## 公式内容の日本語要約

n8n 2.29.3（pre-release）で、Instance AIの課金を専用クレジットプールにルーティングする新機能が追加された。あわせてエディタのパラメータ入力サイズ調整の修正が含まれる。

## できるようになったこと

- Instance AIの課金を専用クレジットプールにルーティングできるように
- エディタのパラメータ入力欄のサイズ調整に関する修正

## 影響範囲

- 対象ユーザー: n8n cloud / インスタンス運用者のAI利用コスト管理担当者
- 対象プラン: n8n 2.x系のpre-release利用者
- API / UI / 管理者機能: Instance AI課金設定、エディタUI

- Instance単位でAI課金を専用プール化する発想は、GitHub CopilotのCost center上限設定（同日発表）と並べて「AIコスト管理の部門別統制」というテーマで比較教材にできる。
