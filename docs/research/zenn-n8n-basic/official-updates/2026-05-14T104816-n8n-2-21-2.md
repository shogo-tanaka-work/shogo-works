# 2026-05-14 n8n 2.21.2 pre-release

## 公式内容の日本語要約

n8n 2.21.2はpre-releaseとして公開された。VM expression engine outputでnested arraysを保持する修正と、VM evaluatorでempty expressionをempty stringに解決する修正が含まれる。

## できるようになったこと

- VM expression engineのnested arrays出力が保持される。
- empty expressionの評価結果が安定する。

## 影響範囲

- 対象ユーザー: n8n 2.21系pre-release利用者
- 対象プラン: self-hosted / pre-release運用
- API / UI / 管理者機能: expression engine / workflow runtime
