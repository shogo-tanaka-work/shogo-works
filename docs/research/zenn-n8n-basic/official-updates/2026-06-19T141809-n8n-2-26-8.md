# 2026-06-19 n8n 2.26.8 — Form Trigger ホットフィックス

## 公式内容の日本語要約

Form Trigger Node で旧バージョンのワークフローがクラッシュする不具合を修正したホットフィックスリリース。認証パラメータにデフォルト値を追加することで対処。新機能の追加はなく、バグ修正のみ。

## できるようになったこと

- Form Trigger Node：旧ワークフローで認証パラメータが欠けていてもクラッシュしなくなった（PR #32629）

## 影響範囲

- 対象ユーザー: n8n 2.26.x を使用しており Form Trigger を含む旧ワークフローを持つユーザー
- 対象プラン: セルフホスト・クラウド共通
- API / UI / 管理者機能: Form Trigger Node のバグ修正のみ

## 教材化メモ

特になし。パッチリリースのみ。
