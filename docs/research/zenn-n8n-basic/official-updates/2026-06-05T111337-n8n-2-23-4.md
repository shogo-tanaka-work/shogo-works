# 2026-06-05 n8n@2.23.4 / n8n@1.123.53

## 公式内容の日本語要約

n8n stable 2.23.4 がリリースされた。内容はバグ修正1件のみで、Data Loader での PDF パース時に DOMMatrix を polyfill する修正（#31700）。あわせて 1.x backport の 1.123.53、pre-release の 2.25.4 / 2.25.5（beta）も同日付近で公開されている。

## できるようになったこと

- **2.23.4（stable・バグ修正のみ）**: Data Loader の PDF パースで DOMMatrix を polyfill（#31700）。
- **1.123.53（1.x backport）**: 同系統の修正。
- 2.25.4 / 2.25.5 は pre-release（beta）。

## 影響範囲

- 対象ユーザー: n8n セルフホスト / クラウド利用者。
- 対象プラン: 制限なし。
- API / UI / 管理者機能: Data Loader（PDF 解析）の安定性。

## 教材化メモ

バグ修正のみのパッチであり教材化の必要は薄い。PDF を Data Loader で扱うワークフローがある場合のみ「DOMMatrix polyfill 修正」を運用メモに残す。
