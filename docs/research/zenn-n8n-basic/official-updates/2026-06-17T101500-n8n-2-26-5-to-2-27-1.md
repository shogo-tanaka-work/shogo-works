# 2026-06-17 n8n 複数リリース

## 公式内容の日本語要約

2026-06-17 に n8n の複数バージョンがリリースされた。stable 系は v2.26.5（サーバー CLI テレメトリ追加）・v2.26.6（DB 接続復旧中のクエリ取得バグ修正）・v1.123.57（LTS 系セキュリティパッチ5件）、pre-release は v2.27.1（MCP・OAuth の IP レートリミットを環境変数で設定可能）。

## できるようになったこと

**v2.27.1（pre-release）**
- `N8N_MCP_IP_RATE_LIMIT` / `N8N_OAUTH_IP_RATE_LIMIT` 環境変数で MCP・OAuth サーバーの IP レートリミットを設定可能
- `N8N_COOP_HEADER` デフォルト値が変更（COOP ヘッダー制御）

**v2.26.6（stable）**
- DB 接続復旧中に不完全なクエリが返される問題を修正

**v2.26.5（stable）**
- サーバー CLI の import/export コマンドにテレメトリ追加

**v1.123.57（LTS stable）**
- @grpc/grpc-js・hono・vue-i18n 等のセキュリティ脆弱性 5件を修正

## 影響範囲

- 対象ユーザー: n8n セルフホストユーザー（主）、クラウドユーザーはベンダー側で管理
- 対象プラン: セルフホスト全プラン
- API / UI / 管理者機能: 環境変数設定（v2.27.1）、LTS セキュリティ対応（v1.123.57）
