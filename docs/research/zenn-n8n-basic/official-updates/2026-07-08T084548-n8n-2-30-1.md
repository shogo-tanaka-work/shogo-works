# 2026-07-08 n8n 2.30.1（pre-release）— トークン交換プロビジョニングのカスタムグローバルロール対応

## 公式内容の日本語要約

n8n 2.30.1（pre-release）がリリースされた。新機能としてトークン交換プロビジョニングでカスタムグローバルロールをサポート。バグ修正では、別Webhookサーバー上で動作している場合のChat機能不具合、実行データ再編集時のnullランデータスロットガードを追加した。

## できるようになったこと

- トークン交換プロビジョニングでのカスタムグローバルロール対応
- 別Webhookサーバー構成でのChat機能修正
- 実行データredaction処理でのnullランデータスロットに対するガード追加

## 影響範囲

- 対象ユーザー: SSO/プロビジョニング連携を行う組織、Webhookサーバー分離構成の運用者
- 対象プラン: 2.30.x系（pre-release）
- API / UI / 管理者機能: トークン交換プロビジョニング、Chat機能、実行データ処理
