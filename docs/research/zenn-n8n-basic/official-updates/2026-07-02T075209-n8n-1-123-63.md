# 2026-07-02 n8n 1.123.63（stable）— 式アイソレートでのDate値保持バグ修正

## 公式内容の日本語要約

n8n 1.123.63（stable）がリリースされた。coreの修正として、式（expression）アイソレートに渡されたDate値が正しく保持されるよう修正された。1.x系（LTS的位置づけ）の保守リリース。

## できるようになったこと

- 式アイソレートに渡したDate値が正しく保持されるよう修正（従来は値が崩れるケースがあった）

## 影響範囲

- 対象ユーザー: 式内でDate型を扱うワークフロー利用者
- 対象プラン: n8n 1.x系（LTS的位置づけ）利用者全般
- API / UI / 管理者機能: 式（expression）エンジンのcore修正

## 教材化メモ

- 地味だが「式の中でDateを扱うと壊れることがある」というn8n特有の落とし穴の一例。式エンジンの型の扱いを説明する教材の注意点として使える。
