# 2026-07-02 n8n 2.28.5（stable）— Salesforceノード・エディタ・式の修正

## 公式内容の日本語要約

n8n 2.28.5（stable）がリリースされた。式アイソレートのDate値保持修正に加え、ノード作成パネルでTool系ノードに「無料クレジット」バッジを表示するエディタ修正、SalesforceノードのJWT認証パスで空の204レスポンスを正規化する修正が含まれる。

## できるようになったこと

- 式アイソレートに渡したDate値が正しく保持されるよう修正
- ノード作成パネルでTool系ノードに「無料クレジット」バッジを表示
- SalesforceノードのJWT認証パスで、空の204レスポンスを正しく正規化するよう修正

## 影響範囲

- 対象ユーザー: Salesforce連携ユーザー、Tool型ノードを探しているユーザー
- 対象プラン: n8n 2.x系利用者全般
- API / UI / 管理者機能: Salesforceノード、ノード作成パネルUI、式エンジン

## 教材化メモ

- SalesforceのJWT認証で204（No Content）が返るケースの扱いは、外部API連携ノードを自作する際の参考になる実例。
