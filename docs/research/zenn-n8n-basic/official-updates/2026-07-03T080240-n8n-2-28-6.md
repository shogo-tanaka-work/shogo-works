# 2026-07-03 n8n 2.28.6（stable）— npmインストール破損バグとAI Gatewayヒント修正

## 公式内容の日本語要約

n8n 2.28.6（stable）がリリースされた。npmインストールを壊す重複zodインスタンス問題を修正するコアの安定性修正に加え、エディタのパラメータ入力サイズ統一、AI Gateway未対応アクション通知の重複防止、Tool利用ノードでのAI Gatewayヒント表示対応が追加された。

## できるようになったこと

- npmインストールを破損させていた重複zodインスタンス問題を修正
- エディタのパラメータ入力欄のサイズを統一
- AI Gateway未対応アクションの通知が重複表示されないよう修正
- Tool利用ノードでAI Gatewayに関するヒントを表示するよう対応

## 影響範囲

- 対象ユーザー: セルフホストn8nの導入者、AI Gateway機能利用者
- 対象プラン: n8n 2.x系利用者全般（セルフホスト含む）
- API / UI / 管理者機能: npmパッケージ依存関係、エディタUI、AI Gateway通知

## 教材化メモ

- npmインストール自体を壊すバグは影響範囲が大きい典型例。セルフホスト運用者向け教材で「依存関係の重複がインストールを壊す」ケースとして紹介できる。
