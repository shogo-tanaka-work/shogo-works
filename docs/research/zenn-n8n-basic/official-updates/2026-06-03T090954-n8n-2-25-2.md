# 2026-06-03 n8n@2.25.2 / n8n@1.123.51

## 公式内容の日本語要約

n8n の stable パッチ 2.25.2 と 1.x backport の 1.123.51 がリリース。いずれもバグ修正のみで新機能はない。

## できるようになったこと

- **n8n@2.25.2（stable, 2026-06-03T09:09:54Z）**:
  - External agent channels がエピソード記憶でユーザー ID を正しく利用するよう修正（#31589）。
  - `n8n import:workflow --activeState=fromJson` がサブワークフロー依存で失敗しうる問題を修正（#31569）。
- **n8n@1.123.51（1.x backport, 2026-06-03T10:21:34Z）**:
  - VM 式エンジンの正常時ログを抑制（#31566）。
  - Prometheus レジストリリセット後に式メトリクスを再登録（#31567）。

## 影響範囲
