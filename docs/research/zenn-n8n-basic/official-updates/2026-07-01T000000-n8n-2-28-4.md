# 2026-07-01 n8n@2.28.4（stable）

## 公式内容の日本語要約

n8n stable 2.28.4 がリリース。主な修正はAI Agentノードでの並列ツールコール構造のチャットメモリ保持問題、プロジェクト変数のグローバル変数オーバーライド不具合、Digest認証のセキュリティ改善。

## 主な変更点

- AI Agent ノードで並列ツールコール構造のチャットメモリが保持されない問題を修正
- プロジェクト変数がグローバル変数を正しくオーバーライドしない不具合を修正
- Digest 認証でサーバーチャレンジ受信後のみ認証情報を送信するよう修正（セキュリティ改善）
- Python タスクランナー向けに `N8N_RUNNERS_ALLOW_TRANSITIVE_IMPORTS` 設定を追加
- Human review ノードがノード検索から発見できるよう改善
