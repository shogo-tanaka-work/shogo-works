# 2026-07-20 n8n 2.30.8 / 2.31.4

## 公式内容の日本語要約

n8nのパッチリリース2件。2.30.8はHITL（Human-in-the-loop）toolsを経由して接続したツールをAI Agentノードが認識できない不具合と、エディタでワークフロー保存時に無効なグループを自動解除しない不具合を修正。2.31.4は2.30.8のうちグループ自動解除の修正のみを含む。`stable`/`beta` の共通タグは2.30.8・2.31.4の内容とそれぞれ一致（同一コミットのリリースタグ付け替え）。

## できるようになったこと

- AI AgentノードがHITL tools経由で接続されたツールを正しく認識する
- ワークフロー保存時に無効なノードグループが自動的に解除される

## 影響範囲

- 対象ユーザー: n8n AI Agentワークフロー利用者、エディタでノードグループを使う利用者
- 対象プラン: self-hosted / Cloud 共通（stableチャンネル）
- API / UI / 管理者機能: AI Agentノード、エディタUI（グループ機能）

## 教材化メモ

HITL（Human-in-the-loop）toolsとAI Agentノードの連携不具合修正は、承認フロー付きAIエージェントワークフローを教材で扱う際の既知の落とし穴・修正済みバージョンとして言及できる。
