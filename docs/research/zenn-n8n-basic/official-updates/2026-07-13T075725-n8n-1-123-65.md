# 2026-07-13 n8n 1.123.65（1.x backport）

## 公式内容の日本語要約

Webhook実行で早期エラー発生時にPromiseを確実にsettleさせる修正（#33998）。同日リリースの`n8n@2.30.4`に含まれる修正のうち1件を1.x系列にバックポートしたもの。

## できるようになったこと

- 1.x系列でもWebhook実行が早期エラー時にハングせず確実に終了するようになった

## 影響範囲

- 対象ユーザー: n8n 1.x系列でWebhookノードを利用するユーザー
- 対象プラン: 1.x backport系列
- API / UI / 管理者機能: Webhookノード
