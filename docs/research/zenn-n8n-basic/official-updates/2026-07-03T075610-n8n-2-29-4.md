# 2026-07-03 n8n 2.29.4（pre-release）— コミュニティノード型定義のAIアシスタント対応

## 公式内容の日本語要約

n8n 2.29.4（pre-release）で、AI Assistantがコミュニティノードの型定義をサポートする新機能が追加された。あわせてBasic LLM Chainノードの「エージェントステップ解析失敗」エラー修正、重複zodインスタンス修正、AI Gateway関連の重複通知・ヒント表示修正が含まれる。

## できるようになったこと

- AI Assistantがコミュニティノードの型定義を認識できるように
- Basic LLM Chainノードで発生していた「エージェントステップ解析失敗」エラーを修正
- 重複zodインスタンス問題を修正
- AI Gateway関連の重複通知・ヒント表示の不具合を修正

## 影響範囲

- 対象ユーザー: コミュニティノード開発者、AI Assistant利用者
- 対象プラン: n8n 2.x系のpre-release利用者
- API / UI / 管理者機能: AI Assistant、Basic LLM Chainノード、AI Gateway通知

## 教材化メモ

- コミュニティノードをAI Assistantが型認識できるようになると、自作ノードの補完精度が上がる。ノード開発者向け教材で「AI支援開発の対象範囲拡大」の例として使える。
