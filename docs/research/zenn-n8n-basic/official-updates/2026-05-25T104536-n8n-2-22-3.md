# 2026-05-25 n8n@2.22.3（pre-release）

## 公式内容の日本語要約

n8n の pre-release バージョン `2.22.3` が 2026-05-25 10:45 UTC に公開された。`core` カテゴリの 1 件のバグ修正のみで、SSO ログイン時に取得したクレーム情報とロールマッピングの診断結果を表面化（surface）し、ロール割り当てが期待どおりにならない原因を切り分けやすくする (#30839)。新機能や挙動変更は含まれない。

## できるようになったこと

- SSO 経由ログイン時に、IdP から受け取った claims とロールマッピングのトレースが診断面で確認できるようになり、「特定ユーザーに想定のロールが付かない」障害の切り分けが速くなる。

## 影響範囲

- 対象ユーザー: SSO（OIDC / SAML）を利用しているエンタープライズ運用者。
- 対象プラン: チャネルは `pre-release`。stable 利用ユーザーは次の stable に取り込まれた段階で適用。
- API / UI / 管理者機能: SSO 診断系の管理者向け面のみ。
