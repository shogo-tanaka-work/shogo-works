# 2026-05-22 n8n@2.22.2（pre-release）

## 公式内容の日本語要約

n8n の pre-release バージョン `2.22.2` が 2026-05-22 10:01 UTC に公開された。スケジュールドポーリング処理での expression isolate 取得失敗時のエラー伝播を改善し、`__emitError` 経由で可視化するよう変更。PostHog 初期化後の group identify 呼び出し位置の誤りも修正。バグ修正のみのリリース。

## できるようになったこと

- スケジュールポーリングが expression isolate を正しく取得できるようになった。
- isolate 取得失敗時のエラーが `__emitError` 経由で適切に報告されるようになった。
- PostHog analytics の group identify が正しいタイミングで呼ばれるようになった。
