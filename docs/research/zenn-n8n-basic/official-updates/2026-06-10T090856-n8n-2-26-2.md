# 2026-06-10 n8n 2.26.2

## 公式内容の日本語要約

n8n 2.26.2（beta）がリリースされた。バグ修正1件と、インスタンス設定の環境テレメトリ追加（内部観測性向上）。PURL ピン固定の不安定なオーバーライドをバージョン非依存の byName へ移行するバグ修正が含まれる。

## できるようになったこと

- インスタンス設定の環境情報をテレメトリとして収集可能に（内部観測性向上）。ユーザー向けの操作変化はなし。

## 影響範囲

- 対象ユーザー: n8n セルフホスト利用者（beta チャンネル）
- ユーザー操作への直接影響なし

## 教材化メモ

教材化対象外（内部テレメトリのみ）。
