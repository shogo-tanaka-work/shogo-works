# 2026-07-22 n8n 2.31.5（stable）

## 公式内容の日本語要約

n8nのstableチャンネルで2.31.5がリリースされた。主な修正は3件。（1）AWSリクエストのヘッダー値を署名前に文字列へ強制変換するように修正（core）、（2）AI AgentノードがHuman-in-the-loop（HITL）ツール経由で接続されたツールを認識できていなかった不具合を修正（core）、（3）エディタのドロップダウンツールチップがメニューの上に表示されるよう修正。同時刻帯に1.xバックポート（n8n@1.123.67）とbeta/pre-release（n8n@2.32.1〜2.32.3）も出ているが、いずれも新機能はなくバグ修正のみ。

## できるようになったこと

- AI Agentノードが、HITLツール経由で接続したツールを正しく認識・利用できるようになった
- AWS関連ノードでリクエストヘッダー値の型不一致による署名エラーが解消

## 影響範囲

- 対象ユーザー: n8n stableチャンネル利用者（AI Agent + HITLツール構成、AWS連携ノード利用者）
- 対象プラン: 制限なし
- API / UI / 管理者機能: AI Agentノード、AWS系ノード、エディタUI

## 教材化メモ

AI Agent + HITLツール接続の不具合修正は、n8nでHuman-in-the-loopを組んでいるワークフロー教材にとって実務上の注意点として言及価値がある（「動かない場合は2.31.5以降にアップデート」という形で）。
