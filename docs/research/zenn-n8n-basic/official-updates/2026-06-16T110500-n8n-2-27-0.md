# 2026-06-16 n8n 2.27.0 (pre-release)

## 公式内容の日本語要約

n8n 2.27.0 が pre-release として公開。実行データの S3 ストレージ保存対応、OpenTelemetry の UI 設定・テストエンドポイント追加、GitHub / Kafka / JWT / Microsoft OneDrive などのノード強化、アイコンピッカー全面改写、API キー UI 再設計、グループ折り畳み時のノード位置変更機能が含まれる。データベース移行（execution_entity テーブルへのインデックス追加）があり、大規模インスタンスでは数分の処理時間が見込まれる。

## できるようになったこと

- **S3 ストレージモード**: 実行データを S3 互換ストレージに保存可能（大量ワークフローの外部ストレージ化）
- **OpenTelemetry**: 管理 UI から設定・テストエンドポイント追加（可観測性の設定が UI で完結）
- **ノード強化**:
  - GitHub: メンバー取得機能追加
  - Kafka: スキーマレジストリ認証対応
  - JWT: カスタムヘッダークレーム対応
  - Microsoft OneDrive: 汎用 OAuth2 対応
- **エディタ改善**:
  - アイコンピッカーを全面改写
  - API キー UI を再設計
  - グループ折り畳み時のノード位置変更機能追加
- **バグ修正**: ワークフロー公開時の認証情報処理、ポーラーの古いバージョン実行防止、OAuth 関連複数修正
- **セキュリティ**: nodemailer・grpc・shell-quote などの依存関係更新

## 影響範囲

- 対象ユーザー: 全 n8n ユーザー（セルフホスト / クラウド）
- 対象プラン: 全プラン
- API / UI / 管理者機能: DB マイグレーションあり（本番導入前に確認推奨）

## 教材化メモ

n8n のエンタープライズ機能強化の事例。S3 ストレージ・OpenTelemetry を組み合わせた本番運用アーキテクチャの教材化候補。ただし pre-release のため stable 確定後に反映を検討する。
