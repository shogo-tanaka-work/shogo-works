# 2026-07-13 n8n 2.30.4（pre-release）

## 公式内容の日本語要約

3件のバグ修正を含むパッチリリース。(1) 実行結果の可視性をシェアライセンス機能から切り離す修正（#33986）、(2) Webhook実行で早期エラー発生時にPromiseを確実にsettleさせる修正（#33996）、(3) Notionノードの古いバージョンでASCII簡略化されたプロパティキーを復元する修正（#34003）。

## できるようになったこと

- 実行結果の可視性設定が、共有ライセンス機能の有無に左右されなくなった
- Webhook実行が早期エラー時にハングせず確実に終了するようになった
- Notionノード旧バージョンでのプロパティキー表示が元に戻った

## 影響範囲

- 対象ユーザー: n8nの実行可視性設定・Webhookノード・Notionノードを利用する全ユーザー
- 対象プラン: pre-release系列（2.30.x）
- API / UI / 管理者機能: 実行管理UI、Webhookノード、Notionノード

バグ修正のみで教材読者への学習価値は薄い。research詳細のみに留める。
