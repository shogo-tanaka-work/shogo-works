# 2026-07-07 n8n 2.29.7 — CodeノードのAIツール併用時クラッシュ・Salesforce JWT認証修正

## 公式内容の日本語要約

n8n 2.29.7がリリースされた（GitHub API上のprereleaseフラグはFalse）。AIツールを含むワークフローでCodeノードが失敗する問題、複数バージョンにまたがるノードのアクション生成方式の不整合、SalesforceノードのJWT認証時のドキュメントアップロード失敗・エラー非表示を修正した。

## できるようになったこと

- AIツールを含むワークフローでのCodeノード失敗を修正
- ノードのアクション生成をバージョン横断でフラットなAction fieldに統一
- Salesforce JWT認証でのドキュメントアップロード失敗・エラー表示を修正

## 影響範囲

- 対象ユーザー: Code node + AIツール併用ユーザー、Salesforce連携ユーザー
- 対象プラン: 2.29.x系
- API / UI / 管理者機能: Codeノード、エディタのアクション生成、Salesforceノード

## 教材化メモ

- Salesforce JWT認証周りの不具合修正が複数バージョンにまたがって続いている（2.28.6でも関連修正あり）。エンタープライズ連携ノードの安定性トピックとして継続ウォッチ対象。
