# 2026-05-13 n8n 1.123.43

## 公式内容の日本語要約

n8n 1.123.43は、1.x系向けのbackportリリース。fast-xml-builder、fast-uri、langsmithなど複数依存関係に関するsecurity issue修正が中心。

## できるようになったこと

- 1.x系運用環境で、依存関係由来のsecurity issue修正を取り込める。

## 影響範囲

- 対象ユーザー: n8n 1.x系を継続運用するself-hosted管理者
- 対象プラン: self-hosted / 1.x系
- API / UI / 管理者機能: runtime dependency / security
