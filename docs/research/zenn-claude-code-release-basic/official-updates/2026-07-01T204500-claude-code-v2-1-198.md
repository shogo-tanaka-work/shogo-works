# 2026-07-01 Claude Code v2.1.198

## 公式内容の日本語要約

Claude Code v2.1.198 が 2026-07-01T20:45Z にリリース。主要な変更は Claude in Chrome の GA（一般提供開始）と、バックグラウンドエージェントの自動化機能（コード作業完了時に自動コミット・プッシュ・ドラフトPR作成）。エージェント通知Hook、`/dataviz` スキル追加、ネットワーク安定性向上、AWS STS トークン自動リフレッシュ等も含む。

## できるようになったこと

- **Claude in Chrome GA**: ブラウザ自動化ツールが正式GA。Claude がブラウザを直接操作し、Webタスクを自動化できる
- **バックグラウンドエージェント自動化**: `claude agents` で起動したセッションがコード作業完了時に自動コミット・プッシュ・ドラフトPR作成
- **エージェント通知Hook**: `agent_needs_input` / `agent_completed` の2種類の Hook イベントを追加
- **`/dataviz` スキル**: チャート・ダッシュボード設計ガイダンスとカラーパレット検証機能
- **ネットワーク安定性**: ECONNRESET などの一時的エラーをバックオフ付き自動リトライに変更
- **AWS STS トークン自動リフレッシュ**: セッション期限切れによる dead-end を防止
- **Explore エージェント改善**: セッションのモデル設定を継承（Opus 上限、従来は Haiku デフォルト）
- **Extended thinking**: subagent と context compaction にカスケード

## 影響範囲

- 対象ユーザー: Claude Code 全ユーザー
- 対象プラン: 全プラン
- Claude in Chrome: Chrome ブラウザ拡張機能と連携した自動化が正式利用可能に

## 教材化メモ

- Claude in Chrome GA は「CLIからブラウザ操作まで一元化」という Claude Code の拡張戦略の典型例
- バックグラウンドエージェントの自動コミット/PR作成は、開発ワークフロー変化の事例として使える
- 通知Hook の設計（`agent_needs_input` / `agent_completed`）はイベント駆動型エージェント設計の参考例
