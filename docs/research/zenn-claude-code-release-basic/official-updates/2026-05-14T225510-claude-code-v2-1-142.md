# 2026-05-14 Claude Code v2.1.142

## 公式内容の日本語要約

Claude Code v2.1.142は、`claude agents`でbackground sessionをdispatchする際の設定flagsを大幅に追加し、Fast modeのdefaultをOpus 4.7へ更新した。plugin root-level `SKILL.md`のsurfacing、plugin detailsでのLSP server表示、MCP timeout修正、background session / daemon / worktreeまわりの安定化も含む。

## できるようになったこと

- `claude agents`で`--add-dir`、`--settings`、`--mcp-config`、`--plugin-dir`、`--permission-mode`、`--model`、`--effort`などを指定できる。
- Fast modeがOpus 4.7をdefaultで使う。
- root-level `SKILL.md`を持つpluginもskillとして認識される。
- plugin detailsにLSP server表示が追加された。
- background sessions、worktrees、MCP timeout、daemon upgrade後の安定性が改善。

## 影響範囲

- 対象ユーザー: Claude Code利用者、background agents / plugins / MCPを使うチーム
- 対象プラン: Claude Code
- API / UI / 管理者機能: CLI / agents / plugins / MCP / model

## 教材化メモ

background agents教材に、dispatch時のflagsでmodel、permission、MCP config、plugin dirを制御する章を追加候補。plugin教材ではroot-level `SKILL.md`とLSP server表示を補足できる。
