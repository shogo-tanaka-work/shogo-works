# 2026-07-22 Claude Code v2.1.218

## 公式内容の日本語要約

Claude Code v2.1.218（stable）がリリースされた。目立つ変更は `/code-review` がバックグラウンドのサブエージェントとして動作するようになったことで、レビュー処理が会話を専有せず、スタックしたスラッシュコマンドをレビュー対象として保持できるようになった。そのほか、スクリーンリーダー向けの単語/行削除アナウンス追加、Windowsパスの `\u` プレフィックス文字化けの修正、左矢印キーによる会話破棄の確認ダイアログ化、MCPサーバー接続失敗時のHTTPステータス表示、Bedrock向けゲートウェイ課金計算の修正など、安定性・アクセシビリティ関連の修正が多数含まれる。

## できるようになったこと

- `/code-review` がバックグラウンドサブエージェントとして実行され、会話を占有しない
- `--ax-screen-reader` モードで単語/行削除時の読み上げに対応
- `/ultrareview` が「review my auth changes」のような説明的な引数でも動作するように修正
- `/code-review ultra` が非対話セッションで正しくクラウドレビューを起動するよう修正
- MCPサーバー接続エラー時にHTTPステータス・エラーテキストを `claude mcp list` / `/mcp` に表示
- スキルの `context: fork` がデフォルトでバックグラウンド実行に変更（`background: false` でオプトアウト可能）

## 影響範囲

- 対象ユーザー: Claude Code 全ユーザー（stable channel）
- 対象プラン: 制限なし
- API / UI / 管理者機能: CLI/TUI、スクリーンリーダーアクセシビリティ、MCP接続診断、Bedrockゲートウェイ課金

## 教材化メモ

`/code-review` のバックグラウンド化は日常運用に直結する変更。教材では「レビューが会話を専有しなくなった」点と、スタックされたスラッシュコマンドをレビュー対象にできる挙動を軽く触れる程度で十分（パッチリリースの範囲、記事化は見送り想定）。
