# 2026-07-20 Claude Code v2.1.216

## 公式内容の日本語要約

Claude Code CLIのパッチリリース。ネットワーク遮断は維持したままファイルシステム分離だけをスキップできる新設定 `sandbox.filesystem.disabled` を追加。長時間セッションでメッセージ正規化コストが二次関数的に増加し数秒単位の遅延・resume遅延が発生していた問題を修正。OAuthトークン失効・ローテーション時のauto mode誤拒否、AskUserQuestionの自由記述回答の文言不整合、Web版Claude Codeのアイドル後の質問重複、worktree分離セッションのgit誤操作・迷子worktree、バックグラウンドセッションの削除不能・エージェント設定リセットなど、開発体験に関わる多数のバグを修正。

## できるようになったこと

- ネットワークegress制御を維持しつつファイルシステム分離だけを無効化できる `sandbox.filesystem.disabled` 設定
- 長時間セッション・resumeの高速化（メッセージ正規化コストの二次関数的増加を修正）
- OAuthトークン期限切れ/ローテーション中もauto modeがコマンドを誤って拒否しない
- worktree分離されたsubagentが共有チェックアウトへgitコマンドを誤って向けない
- バックグラウンドセッション・resume済みsubagentのエージェント設定/ツール制限の復元

## 影響範囲

- 対象ユーザー: Claude Code CLI全ユーザー（特にsandbox設定利用者、長時間セッション運用者、worktree/subagent運用者）
- 対象プラン: 全プラン
- API / UI / 管理者機能: CLI設定（sandbox設定）、Web版Claude Codeセッション管理、subagent/worktree実行系

## 教材化メモ

`sandbox.filesystem.disabled` はファイルシステム分離のみを外し、ネットワーク制御は維持する部分的なsandbox設定緩和。全面的なsandbox無効化とは異なる点を教材で明示する価値がある。worktree分離subagentのgit誤操作修正は、複数worktreeを使う運用フローの信頼性向上として言及できる。
