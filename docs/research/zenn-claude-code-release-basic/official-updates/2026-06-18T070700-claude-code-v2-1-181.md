# 2026-06-18 Claude Code v2.1.181

## 公式内容の日本語要約

Claude Code v2.1.181 がリリースされた。主な変更は `/config key=value` 構文によるプロンプトからの設定変更、macOS サンドボックスの Apple Events 許可オプション追加、Bun 1.4 へのアップグレード、モバイル通知抑制の環境変数追加など。

## できるようになったこと

- `/config key=value` でプロンプトから任意の設定を直接変更可能（例: `/config thinking=false`、`/config model=claude-opus-4-8`）
- macOS サンドボックスで `--dangerously-allow-apple-events` オプションが追加され、GUI アプリ連携コマンドの実行が可能に
- `CLAUDE_DISABLE_MOBILE_NOTIFICATIONS` 環境変数でモバイル通知を抑制可能
- Bun 1.4 へアップグレード（ランタイム安定性・パフォーマンス向上）
- ストリーミングとサブエージェントの内部改善

## 影響範囲

- 対象ユーザー: 全 Claude Code ユーザー
- 対象プラン: Claude Code を利用可能なすべてのプラン
- API / UI / 管理者機能: CLI 設定変更、macOS 環境のみ Apple Events オプション

## 教材化メモ

`/config` 構文はセッション中に設定を試す際のショートカットとして使いやすい。特にモデル切り替えや thinking モードのオンオフを素早くテストしたい場面で有用。教材では「設定を固定する方法（config ファイル）」と「セッション内で試す方法（/config コマンド）」を対比させると理解しやすい。
