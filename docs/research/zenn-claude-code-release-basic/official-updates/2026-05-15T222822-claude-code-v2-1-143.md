# 2026-05-15 Claude Code v2.1.143

## 公式内容の日本語要約

Claude Code v2.1.143 は、background sessions、`claude agents`、plugins、Windows / PowerShell まわりを広く改善した更新。特に plugin dependency enforcement、plugin marketplace での projected context cost 表示、background session が model / effort / MCP 設定を保持する改善、Windows での PowerShell tool 既定有効化が目立つ。

## できるようになったこと

- plugin 有効化時に依存関係を自動処理し、無効化時は依存中 plugin を警告
- `/plugin` 画面で per-turn / per-invocation の推定コンテキストコストを確認
- background session が model、effort、MCP、settings、fallback model などを保持
- `claude agents` から background sessions の既定値をより細かく指定
- Windows / PowerShell 利用時の既定挙動と操作安定性を改善

## 影響範囲

- 対象ユーザー: Claude Code 利用者、特に plugin / background session / Windows 利用者
- 対象プラン: Claude Code 全般
- API / UI / 管理者機能: CLI / plugins / agents / background sessions

## 教材化メモ

Claude Code の進化が「単体CLI」から「複数agent・plugin・長時間実行を前提にした作業環境」へ進んでいることを示す更新。plugin dependency、context cost、background session 設計をまとめて扱う教材に向く。
