# 2026-05-22 Claude Code v2.1.149

## 公式内容の日本語要約

Anthropic は 2026-05-22 22:09 UTC（2026-05-23 07:09 JST）に Claude Code **v2.1.149** を公開した。前日 v2.1.148 のホットフィックスから一段進めて、コスト可視化・Enterprise 向け MCP 設定・主要 OS 系の安全性修正をまとめた更新。

## できるようになったこと

- `/usage` がスキル / サブエージェント / プラグイン / MCP サーバー単位で**コストの内訳を表示**するようになった。どの構成要素がコストを食っているかを直接見ながらチューニングできる。
- Enterprise 向けに `allowAllClaudeAiMcps` マネージド設定が追加された。`managed-mcp.json` と組み合わせて **claude.ai クラウド MCP コネクタを一括許可**できる。
- `/diff` 詳細ビューがキーボード操作で**スクロール可能**になった（矢印キー・`j`/`k`・PgUp/PgDn・Space・Home/End）。
- Markdown 出力が **GFM タスクリストチェックボックス**（`- [ ]` / `- [x]`）を正しく描画するようになった。

## 主なバグ修正・セキュリティ修正

- PowerShell の `cd` 系組み込み関数を経由して、許可されたワークスペース外へ抜け出せた**パーミッションバイパスを修正**。
- macOS で `find` コマンドがシステムファイル / vnode テーブルを枯渇させ、ホストごとクラッシュさせる問題を修正。
- 上記以外にも複数の安定性・UX 修正（CHANGELOG.md の v2.1.149 セクション参照）。

## 影響範囲

- 対象ユーザー: Claude Code 利用者全般。特にコスト管理を運用に組み込んでいるチーム、Enterprise（claude.ai クラウド MCP の集中管理）、PowerShell ユーザー、macOS ユーザー。
- 対象プラン: プラン制限の記載なし（CLI 更新）。Enterprise 設定 `allowAllClaudeAiMcps` は Enterprise 管理者向け。
- API / UI / 管理者機能: CLI 機能（`/usage`、`/diff`）、Enterprise マネージド設定（`allowAllClaudeAiMcps` + `managed-mcp.json`）、PowerShell/macOS 環境のセキュリティ。

## 教材化メモ

- new-commands-2026 教材の `/usage` 説明を、「実行コスト総量を見る」から「**スキル / サブエージェント / プラグイン / MCP サーバー単位の内訳を見る**」へ更新候補。
- Enterprise 設定教材があれば `allowAllClaudeAiMcps` + `managed-mcp.json` の組み合わせを 1 段追記（claude.ai クラウド MCP コネクタの集中許可手段）。
- PowerShell の `cd` 系経由でワークスペース外に抜けられた件は、過去の権限モデル説明に「v2.1.149 で修正済み」と短く追記する余地あり。
