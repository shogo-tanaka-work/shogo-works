# 2026-07-09 Claude Code v2.1.206

## 公式内容の日本語要約

`/cd`に`/add-dir`同様のディレクトリパス補完を追加。`/doctor`がチェックイン済みCLAUDE.mdの削減提案を行うように。`/commit-push-pr`が設定済みpushリモートへの`git push`を自動許可。Gateway機能がAnthropic運営の公開ゲートウェイエンドポイントへの`/login`に対応。`EnterWorktree`がプロジェクトディレクトリ外のworktreeに入る際に確認を要求。バックグラウンドエージェントがClaude Code更新後にバックグラウンドでアップグレードされるように。他、期限切れログイン時の誤ったモデルエラー表示、`claude --resume`/`--continue`のキーボード入力無反応、MCPサーバーの`request_timeout_ms`無視、`CLAUDE_CODE_EXTRA_BODY`のバックグラウンドワーカー無視、OAuth MCPサーバーのトークンリフレッシュ失敗時再認証要求など多数のバグ修正。

注記: 本ファイル作成時点でGitHub Releasesページ（https://github.com/anthropics/claude-code/releases）にv2.1.206のリリースエントリが未作成（v2.1.205が最新表示）。CHANGELOG.mdとnpm registryのみ先行更新されている状態のため、次回チェックで正式GitHub Release公開後に内容差分がないか再確認する。

## できるようになったこと

- Gatewayの公開エンドポイントへのログイン
- /doctorによるCLAUDE.md削減提案
- /cdのディレクトリパス補完
- commit-push-prの自動push許可
- worktree外への進入時の確認

## 影響範囲

- 対象ユーザー: Claude Code CLI全ユーザー。特にGateway運用者・CLAUDE.md運用者・worktree運用者・バックグラウンドエージェント利用者
- 対象プラン: 該当なし（CLIツール）
- API / UI / 管理者機能: Gateway設定、CLI設定
