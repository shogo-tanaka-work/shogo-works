# 2026-05-12 Claude Code v2.1.140

## 公式内容の日本語要約

Claude Code v2.1.140 が公開された。Agent tool の `subagent_type` 指定が大文字小文字や区切り文字の違いに強くなり、agent color palette も更新された。加えて `/goal`、settings hot-reload、background service、managed settings、Windows の event-loop stall、Read tool validation、plugin warnings などの不具合が修正された。

## できるようになったこと

- `Code Reviewer` のような表記でも `code-reviewer` に解決しやすくなった
- `/goal` が hooks 制限時に黙って止まる代わりに明確なメッセージを出す
- plugin の component folder が `plugin.json` 設定で無視される場合に警告される

## 影響範囲

- 対象ユーザー: Claude Code 利用者、plugin / hooks / subagent を使う開発者
- 対象プラン: Claude Code
- API / UI / 管理者機能: CLI、agent tool、plugin、managed settings

## 教材化メモ

Claude Code の agent / plugin 運用教材で、subagent 名の指定ミス、hooks 制限、plugin manifest と folder 構成の衝突をチェックリスト化できる。
