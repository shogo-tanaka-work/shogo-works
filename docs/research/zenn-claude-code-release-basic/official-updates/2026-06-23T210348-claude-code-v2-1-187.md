# 2026-06-23 Claude Code v2.1.187

## 公式内容の日本語要約

Claude Code v2.1.187 がリリースされた（v2.1.186 の翌日）。セキュリティ設定の強化、組織レベルのモデル制限、マウス操作の改善、そして日本語・韓国語・中国語テキストのペーストが文字化けしていたバグ修正が含まれる。

## できるようになったこと

**新機能:**
- `sandbox.credentials` 設定でサンドボックス内コマンドが認証情報ファイル・秘密環境変数を読み取れないようにブロック
- 組織設定によるモデル制限を model picker・`--model`・`/model`・`ANTHROPIC_MODEL` に適用。制限されたモデル選択時に "restricted by your organization's settings" メッセージを表示
- フルスクリーンモードのメニュー（パーミッションプロンプト・`/model`・`/config`等）でマウスクリックによる選択が可能に

**バグ修正（主要）:**
- 韓国語・中国語・日本語 (CJK) テキストをペーストすると、バイト単位で拡張キーイベントを送るターミナルで文字化けしていた問題を修正
- リモート MCP ツール呼び出しが 5分間応答なしでハングするバグを修正（`CLAUDE_CODE_MCP_TOOL_IDLE_TIMEOUT` で上書き可能）
- `--resume` が `-p` で生成なしの場合に "No conversation found" で失敗する問題を修正
- `--json-schema` / `agent({schema})` の構造化出力で成功後も `StructuredOutput` を無限再呼び出しするバグを修正
- Remote Control セッションの起動が CA トラスト処理追加により約 2.7s 遅延していた問題を修正
- `/update` over Remote Control でスタートアップダイアログが挟まってハングする問題を修正
- agents view 上のバックグラウンドジョブが構造化出力なしで終了した場合に "working" 状態に固まる問題を修正
- チャンネル接続が agents view ナビゲーション後・`/bg`・`/tui`・`/update` 後にドロップする問題を修正
- サブエージェント深度トラッキングの修正（再開したサブエージェントが元の spawn 深度を復元）
- エージェント worktree の登録リーク修正（kill されたエージェントの `.git/worktrees/` エントリを自動クリーンアップ）
- macOS Ghostty のフルスクリーンモードで Cmd+click が URL を開かない問題を修正
- `/share` のアップロード中に Esc・Ctrl-C・Ctrl-D が効かない問題を修正

**改善:**
- `/install-github-app`: GitHub Actions ワークフロー設定がオプション化（App のみインストールしてワークフロー/シークレット設定をスキップ可能）
- `/btw`: ←/→ 矢印で過去の回答をステップ移動できるよう改善
- `/plugin`: 最近使っていないプラグインを表示してクリーンアップを促すよう改善
- [VSCode] 大きなセッション再開時に拡張機能が無応答になるバグを修正

## 影響範囲

- 対象ユーザー: Claude Code 全ユーザー（CJK 修正はアジア言語ユーザーに特に重要）
- 対象プラン: 全プラン
- API / UI: CLI・VSCode・フルスクリーン・Remote Control 全般

## 教材化メモ

- `sandbox.credentials` は組織展開時のセキュリティポリシー設計の具体例として使える
- 組織モデル制限は「AI ガバナンスのツール制御」という文脈で教材化できる
- CJK ペースト修正は日本語ユーザーへの影響が大きく、アップデート推奨の文脈で言及できる
- MCP ツールタイムアウト修正は「MCP サーバーが応答しない場合のハンドリング」という実装教材に使える
