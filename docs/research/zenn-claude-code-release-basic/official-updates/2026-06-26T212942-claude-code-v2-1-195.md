# Claude Code v2.1.195

## 主要変更点

### CLAUDE_CODE_DISABLE_MOUSE_CLICKS
- フルスクリーンモードでマウスクリック・ドラッグ・ホバーを無効化する環境変数を追加
- ホイールスクロールは引き続き有効

### hook matcher バグ修正（重要）
- ハイフン付き識別子（例: `code-reviewer`, `mcp__brave-search`）が部分一致してしまうバグを修正
- 現在は完全一致のみ。ハイフン付き MCP サーバーの全ツールにマッチするには `mcp__brave-search__.*` のようにワイルドカードを使う

### 音声入力修正（macOS）
- デフォルト入力デバイスが変わった後の長時間セッションで無音をキャプチャしてしまう問題を修正

### 音声入力修正（日本語・中国語・タイ語）
- スペースなしで書く言語（日本語・中国語・タイ語）で voice dictation の auto-submit が発火しなかったバグを修正

### プラグイン関連修正
- プロジェクト `.claude/settings.json` のみで有効化された外部プラグインがインストール同意を毎回求めなくなっていたバグを修正
- プラグインの `plugin.json` の `name` とマーケットプレイスエントリ名が異なる場合に `/plugin` Enable/Disable が機能しなかった問題を修正

### バックグラウンドエージェント修正
- 新しい Claude Code バージョンが書いたジョブが `claude agents` から消えたりデータが失われる問題を修正
- クラッシュしたバックグラウンドタスクを再度開いたとき最大5秒間ブランクスクリーンが表示される問題を修正
- control socket 起動失敗時にバックグラウンドエージェントデーモンが到達不能で動き続けてリスタートをブロックする問題を修正

### その他の改善
- Linux での音声モード: SoX はあるがオーディオキャプチャデバイスなしの場合に「マイクなし」と「SoX 未インストール」を区別するようになった
- `claude agents` の完了リストが縦スペースを有効活用するよう改善。短いターミナルではヘッダーが圧縮されライブセッションが見えたまま
- Remote セッション起動時にコンテナ起動中のプロビジョニングチェックリストを表示
