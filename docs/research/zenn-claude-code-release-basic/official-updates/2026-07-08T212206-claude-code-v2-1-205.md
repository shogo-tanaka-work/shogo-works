# 2026-07-08 Claude Code v2.1.205

## 公式内容の日本語要約

Claude Code v2.1.205 が 2026-07-08T21:22Z にリリース。`/doctor`（エイリアス`/checkup`）が単なるチェックから、診断・修復まで行う総合セットアップチェックに強化された。エージェントビューではPRリンクや状態表示（分類済みヘッドライン・詳細ピーク）が改善。将来のClaude Desktopペイン名変更に備えて「Claude Browser」というMCPサーバー名が予約され、ユーザー設定のMCPサーバーがこの名前を使えなくなった。auto modeはセッション文書の改ざんをブロックし、`rm -rf`実行時に変数解決不能な場合は確認を挟むよう改善された。オートアップデートのメモリ使用量が約400MB削減された。

## できるようになったこと

- `/doctor`（エイリアス`/checkup`）が診断・修復まで行う総合セットアップチェックに強化
- エージェントビューでPRリンク表示・状態表示（分類済みヘッドライン・詳細ピーク）が改善
- 「Claude Browser」というMCPサーバー名を予約（Claude Desktopのペイン名変更に備えた措置）
- auto modeがセッション文書の改ざんをブロックするよう改善
- auto modeで`rm -rf`実行時に変数が解決不能な場合、確認を挟むよう改善
- オートアップデートのメモリ使用量を約400MB削減

## 影響範囲

- 対象ユーザー: 全ユーザー。MCPサーバー名に「Claude Browser」「Claude Preview」を使っている場合は要確認。エージェントビュー・auto mode利用者
- 対象プラン: 全プラン
- API / UI / 管理者機能: `/doctor`コマンド、エージェントビューUI、MCPサーバー名前予約、auto modeの安全機構

## 教材化メモ

- `/doctor`の総合診断化はトラブルシューティング教材の入口コマンドとして案内価値が高い
- MCPサーバー名「Claude Browser」予約は、独自MCPサーバーを命名する際の注意点として教材に反映すべき
