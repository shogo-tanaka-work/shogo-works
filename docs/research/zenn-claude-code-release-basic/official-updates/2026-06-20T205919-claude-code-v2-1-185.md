# 2026-06-20 Claude Code v2.1.185 — ストリーム停止インジケーター改善

## 公式内容の日本語要約

ストリームが止まった際のヒントメッセージ文言とタイミングを変更したマイナーリリース。

## できるようになったこと

- 旧メッセージ: `No response from API · Retrying in …`（10秒後表示）
- 新メッセージ: `Waiting for API response · will retry in …`（20秒後表示）
- 表示タイミングを10秒→20秒に延長することで誤検知を減らし、文言もより正確な意味に変更

## 影響範囲

- 対象ユーザー: Claude Code 全ユーザー
- 対象プラン: 全プラン
- API / UI / 管理者機能: TUI インジケーター表示変更のみ

## 教材化メモ

特になし。UX 文言修正のみ。
