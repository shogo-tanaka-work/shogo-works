# 2026-06-18 Claude Code artifacts support

## 公式内容の日本語要約

Claude Codeセッション中の作業進捗を、インタラクティブなウェブページ（アーティファクト）としてリアルタイムで可視化・共有できる機能が追加された。インシデント調査、ダッシュボード、リリースチェックリストなど様々な用途に対応。URLは更新時も変わらず、バージョン履歴・復元も可能。Team / Enterprise プランが対象。

## できるようになったこと

- セッションの進捗をライブのウェブページとして生成・共有
- コードベース・コネクター・会話内容のコンテキストをもとに自動生成
- チームメンバーへのリアルタイム共有（組織内認証ユーザー限定）
- バージョン履歴管理と以前バージョンへの復元
- 組織レベルのアクセス制御・保持期間設定

## 影響範囲

- 対象ユーザー: Claude Code Team / Enterprise プランのユーザー
- 対象プラン: Team, Enterprise
- API / UI / 管理者機能: UI

## 教材化メモ

- 「Claude Codeの出力をチームに見せる方法」として紹介できる
- アーティファクトのURLは不変なので、進捗レポートとして固定リンク共有が可能
- 管理者は保持期間とアクセス制御を設定できる点がエンタープライズ運用で重要
