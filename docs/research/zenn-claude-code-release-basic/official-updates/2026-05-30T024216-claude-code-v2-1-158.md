# 2026-05-30 Claude Code v2.1.158

## 公式内容の日本語要約

Anthropic は 2026-05-30 02:42 UTC（JST 11:42）に Claude Code **v2.1.158** を公開。**Auto mode が Bedrock / Vertex / Foundry でも Opus 4.7 / 4.8 で利用可能** になった。利用には環境変数 **`CLAUDE_CODE_ENABLE_AUTO_MODE=1`** のオプトインが必要。

これまで Auto mode は Anthropic 直接 API での利用が中心で、クラウドプロバイダー経由（Bedrock / Vertex / Foundry）では制限されていた。v2.1.158 で **Enterprise が AWS / GCP / Azure 経由で Claude を運用しているケース** でも Auto mode を解禁できるようになった。

## できるようになったこと

- AWS Bedrock 経由 Claude（Opus 4.7 / 4.8）で Auto mode を利用可。
- Google Cloud Vertex AI 経由でも同様。
- Microsoft Foundry 経由でも同様。
- ただし `CLAUDE_CODE_ENABLE_AUTO_MODE=1` を明示的に設定する必要があるため、組織ポリシーで段階導入できる。

## 影響範囲

- 対象ユーザー: Bedrock / Vertex / Foundry 経由で Claude Code を運用する Enterprise / 開発チーム
- 対象プラン: Bedrock / Vertex / Foundry 経由の Opus 4.7 / 4.8 利用枠
- API / UI / 管理者機能: 環境変数 `CLAUDE_CODE_ENABLE_AUTO_MODE=1` を組織配布する形（managed settings 等）

## 教材化メモ

managed-settings 教材で **`CLAUDE_CODE_ENABLE_AUTO_MODE`** を「Enterprise 配布の環境変数」として明記。これまで「Bedrock / Vertex / Foundry では Auto mode は使えない」と書いていた箇所があれば訂正（オプトインで利用可）。
