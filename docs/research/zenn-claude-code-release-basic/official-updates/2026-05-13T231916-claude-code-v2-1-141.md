# 2026-05-13 Claude Code v2.1.141

## 公式内容の日本語要約

Claude Code v2.1.141は、hooks、plugin、workload identity federation、agent dashboard、feedback、permission promptなど、日常運用の細部を広く改善するリリース。新機能としては、hook JSON outputの`terminalSequence`、HTTPSでのGitHub plugin cloneを優先する環境変数、workspace ID指定、`claude agents --cwd <path>`などが追加された。

また、background agentのpermission mode保持、Rewind menuの要約、plugin menu navigation、MCP connectorやRemote Control関連の安定化など、長時間・複数セッション運用で効く修正が多い。

## できるようになったこと

- hooksからdesktop notification、window title、bellなどのterminal sequenceを出しやすくなった。
- GitHub SSH keyがない環境では、plugin sourceをHTTPSでcloneする設定を使える。
- workload identity federationでworkspaceを明示できる。
- `claude agents --cwd <path>`で特定ディレクトリのsession listへ絞り込める。
- `/feedback`で直近24時間または7日間のsessionを含められる。

## 影響範囲

- 対象ユーザー: Claude Code利用者、plugin / hook / background agentを使う開発チーム
- 対象プラン: Claude Code利用環境全般
- API / UI / 管理者機能: CLI / hooks / plugins / agents / MCP / permission

## 教材化メモ

hooks教材では、`terminalSequence`を「通知・タイトル・bellを返す用途」として補足できる。チーム利用教材では、HTTPS clone優先、workspace ID、`--cwd`でのsession絞り込みを、企業環境でのClaude Code運用パターンとして扱える。
