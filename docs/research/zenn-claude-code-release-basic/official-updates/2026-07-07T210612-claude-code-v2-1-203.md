# 2026-07-07 Claude Code v2.1.203

## 公式内容の日本語要約

Claude Code v2.1.203 が 2026-07-07T21:06Z にリリース。ログイン期限切れ前に警告を表示するようになり、手動権限モード時にはフッターにバッジが追加される。MCPの `roots/list` に追加作業ディレクトリが反映されるようになり、変更時には通知が送信される。バックグラウンドエージェントの応答不能、PATH継承の不具合、`ANTHROPIC_BASE_URL`が消失する不具合、daemon自動アップグレード時に全セッションが停止してしまう不具合など、多数の安定性バグが修正された。VSCode拡張には「Enable Remote Control for all sessions」設定が追加された。

## できるようになったこと

- ログイン期限切れ前の警告表示を追加
- 手動権限モード時にフッターへバッジを表示
- MCPの`roots/list`に追加作業ディレクトリを反映し、変更時に通知を送信するように変更
- バックグラウンドエージェントが応答不能になる不具合を修正
- バックグラウンドエージェントのPATH継承不具合を修正
- バックグラウンドエージェントで`ANTHROPIC_BASE_URL`が消失する不具合を修正
- daemon自動アップグレード時に全セッションが停止してしまう不具合を修正
- VSCode拡張に「Enable Remote Control for all sessions」設定を追加

## 影響範囲

- 対象ユーザー: バックグラウンド/エージェントビュー利用者、Remote Control・VSCode拡張利用者、MCPサーバー運用者
- 対象プラン: 全プラン
- API / UI / 管理者機能: VSCode拡張設定、MCPの`roots/list`、バックグラウンドエージェント基盤

## 教材化メモ

- バックグラウンドエージェントの安定性バグ修正が続いているため、教材では「まだ枯れていない機能」という位置付けで注意喚起すると良い
- MCPの`roots/list`変更通知対応は、MCPサーバー開発教材で作業ディレクトリ同期の実例として使える
