# 2026-07-08 Runway Dev

## 公式内容の日本語要約

RunwayがNews（changelogではなくNews）で「Runway Dev」を発表した。画像・動画・音声・リアルタイムキャラクターモデルを単一APIで統合利用できる開発者・エンタープライズ向けプラットフォームで、自社モデル（Gen-4.5、Aleph 2.0、Act-Two）に加えSeedance、GPT Image 2、ElevenLabs等のサードパーティモデルにも第一級アクセスを提供する。広告ローカライズ等の定型タスクを実行する「Recipes」、カスタムパイプラインを組める「Workflows」、リアルタイムアバターの「Characters」の4要素で構成される。SOC 2 Type II準拠・99.9%稼働保証・データ非学習契約などエンタープライズ機能を備え、Adobe・Figma・Shutterstock等が既に採用と記載されている。

## できるようになったこと

- 自社モデル（Gen-4.5 / Aleph 2.0 / Act-Two）とサードパーティモデル（Seedance / GPT Image 2 / ElevenLabs等）への統一API アクセス
- 広告ローカライズ等の定型タスクを実行する「Recipes」
- カスタムワークフロー構築（Workflows）
- リアルタイムキャラクター・アバター機能（Characters）
- SOC 2 Type II準拠、データ非学習契約、99.9%稼働保証などエンタープライズ向け保証

## 影響範囲

- 対象ユーザー: 開発者・エンタープライズ顧客（既存のクリエイター向けRunwayアプリとは別軸のB2B展開）
- 対象プラン: Runway Dev専用プラットフォーム（詳細な価格体系は本文に記載なし）
- API / UI / 管理者機能: 新規API基盤の提供

## 教材化メモ

Runwayの大型発表はchangelogではなくNewsに出るという既知パターンを今回も踏襲（changelog側は07-02のAgent Skillsが最新のまま、Runway Devの反映なし）。サードパーティモデル（Seedance等競合含む）を統合するマルチモデルAPI戦略は、他社のAPIアグリゲーション動向と比較する教材ネタになりうる。
