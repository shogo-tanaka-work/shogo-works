# 2026-07-02 Runway Agent Skills

## 公式内容の日本語要約

Runway Agent 2.0（2026-06-25公開）に「Agent Skills」機能が追加された。チャット欄で `/` を入力してSkillを選択し、コマンド1つで広告キャンペーン作成・コマーシャル制作・広告のローカライズ（コピー翻訳・ビジュアル差し替え）などを自動実行できる。changelog上は日付のみの記載で、時刻（JST基準で対象期間内か境界か）は特定できていない。

## できるようになったこと

- `/`コマンドでSkillを呼び出し、広告キャンペーンの一括生成を自動化
- コマーシャル制作をSkill経由でワンステップ化
- 広告のローカライズ（コピー翻訳・ビジュアル差し替え）を自動実行

## 影響範囲

- 対象ユーザー: マーケター・広告制作者
- 対象プラン: All Plans
- API / UI / 管理者機能: Runway Agent（https://app.runwayml.com/agent）のUI上でのSkill選択機能

## 教材化メモ

Runway Agent 2.0（6/25）からの派生強化。既存のエージェント型UIにSkillという定型ワークフロー呼び出し概念を追加した形。他社（Genspark AI Workspace等）のテンプレート/Skill機構と比較する教材ネタになりうる。
