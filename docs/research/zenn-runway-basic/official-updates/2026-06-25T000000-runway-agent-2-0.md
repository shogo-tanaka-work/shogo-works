# Runway Agent 2.0

## 概要

Runway の AI マーケティングエージェントの第2世代。キャンペーン全体の生成からデータ分析・多プラットフォーム展開までをカバーするクリエイティブパートナーツール。

## 主要ポイント

- **対象**: 全プラン
- **機能3点**:
  1. キャンペーン全体をゼロから作成（Complete campaign creation）
  2. クリエイティブ改善のためのデータ分析（Data analysis for creative optimization）
  3. 複数プラットフォーム・フォーマット・市場へのスケール展開
- **価格インセンティブ**: コード `AGENT` 使用で全プラン30%割引

## 公式引用

> "Make marketing that drives revenue. Runway Agent creates entire campaigns, analyzes data to improve your creative and scales your marketing across platforms, formats and markets."
