# 2026-06-18 Runway Studio

## 公式内容の日本語要約

Runway 上で動画のトリム・結合・並び替え・エクスポートまでを一つの統合インターフェースで完結できる「Studio」機能がリリースされた。全プランで利用可能。

## できるようになったこと

- 動画クリップのトリム・ステッチ・並び替えを単一画面で実施
- 最終動画のエクスポートまでを Runway 外ツール不要で完了
- ポストプロダクションワークフローの一元化

## 影響範囲

- 対象ユーザー: 動画クリエイター、映像制作者（全プラン）
- 対象プラン: 全プラン
- API / UI / 管理者機能: UI

## 教材化メモ

- 「AI生成動画の後処理も同じ画面でできる」というワークフロー完結の価値を説明できる
- 従来は外部編集ツールが必要だった後処理が不要になるためクリエイターの敷居が下がる
