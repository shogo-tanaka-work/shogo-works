# 2026-05-21 Runway Aleph 2.0 と Edit Studio

## 公式内容の日本語要約

Runway は 2026-05-21、動画編集モデル **Aleph 2.0** と、それを中心に据えた新アプリ **Edit Studio** を発表した。「すでに持っている動画から、本当に必要な動画を得る」ことをコンセプトに、1フレームの編集を動画全体へ伝播させる編集ワークフローを提供する。

## できるようになったこと

- **1フレーム編集の全体伝播**: 動画内の1フレームを編集して結果をプレビューすると、Aleph 2.0 がその編集をクリップ全体へ自動で反映する。
- カット・シーン切り替えのある動画でも、関連するショットをまたいで編集を適用。ショットごとの個別編集が不要になった。
- 最長 **30秒・1080p** のクリップを編集可能。
- **Edit Studio**: Aleph 2.0 を中心に設計された新しいアプリ体験。マーケティングチームのキャンペーン派生制作、映像制作の後処理、スモールビジネスの既存動画のリフレッシュなどを想定。最終生成前に、変更を画像としてプレビューして見た目を詰められる。

## 影響範囲

- 対象ユーザー: 動画編集を行う Runway 利用者。マーケティング担当、映像制作者、スモールビジネスオーナー。
- 対象プラン: Edit Studio の Aleph 2.0 は **全有料プラン**でデスクトップ Web アプリから利用可能。
- API / UI / 管理者機能: Web アプリ（Edit Studio）。
