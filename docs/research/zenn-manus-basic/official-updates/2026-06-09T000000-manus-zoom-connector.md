# 2026-06-09 Manus Zoom Connector

## 公式内容の日本語要約

Manus に Zoom 連携機能「Zoom Connector」が追加された。Zoom のミーティング録画・文字起こし・Zoom Docs へのアクセスと操作が可能になり、ミーティング後のフォローアップ生成、繰り返しパターンの分析、週次レポートの自動化などのワークフローを Manus 上から直接実行できる。会議内容を次の業務フローへシームレスにつなぐことを主眼とする。

## できるようになったこと

- **Zoom Connector** で Zoom のミーティング録画・文字起こし・Zoom Docs にアクセス・操作。
- ミーティング後の**フォローアップ生成**、繰り返しパターンの分析、**週次レポート自動化**などを Manus 上から実行。

## 影響範囲

- 対象ユーザー: Manus 利用者（会議の多い業務、議事・フォローアップを自動化したい個人・チーム）
- 対象プラン: 記載なし（Manus の Zoom コネクタ利用者）
- API / UI / 管理者機能: Manus の Zoom コネクタ

## 教材化メモ

Manus の「コネクタ」章に Zoom Connector を追記。会議録画・文字起こし・Zoom Docs を入力に、フォローアップメール生成・週次レポート自動化・繰り返し議題の分析を行うワークフロー例を示す。前日の複数 Google アカウント対応に続く「会議→業務フロー」連携の強化として位置づける。
