# 2026-07-14 Introducing PowerPoint Slides on Manus

## 公式内容の日本語要約

Manusが、Web形式で作成したスライドをPowerPointに変換する従来方式ではなく、最初からネイティブの.pptxファイルを直接生成する新機能を発表した。生成されるスライドには埋め込みデータテーブル付きの編集可能なチャートが含まれ、PowerPoint上でそのまま編集できる。入力にはスプレッドシート、研究トピック、PDF、URL、会議メモなど複数形式に対応し、Manus上でチャートデータを直接編集すると変更がリアルタイムに反映される。1.6モデルおよびMaxモデルのユーザー向けにベータ提供を開始した。

## できるようになったこと

- ネイティブ.pptx形式でのスライド直接生成（Web形式からの変換を経由しない）
- 埋め込みデータテーブル付きの編集可能なチャートをPowerPoint上でそのまま編集可能
- スプレッドシート・研究トピック・PDF・URL・会議メモなど複数入力形式からのスライド生成
- Manus上でのチャートデータ編集がリアルタイムにスライドへ反映

## 影響範囲

- 対象ユーザー: 1.6モデル・Maxモデル利用者（ベータ）
- 対象プラン: 1.6/Maxモデル利用プラン
- API / UI / 管理者機能: スライド生成機能、チャート編集UI

## 教材化メモ

- 資料作成AIとしての実務適合度（編集可能性・データ連動）を高める機能。Genspark AI Slides等、他社スライド生成機能との比較教材のネタになる。
- 「Web形式変換」から「ネイティブファイル直接生成」への移行は、他のドキュメント生成系AI機能（Excel/Word生成等）でも今後同様の流れが予想され、トレンドとして紹介できる。
