# 2026-06-08 Manus 複数 Google アカウント接続

## 公式内容の日本語要約

Manus で複数の Gmail / Google カレンダーアカウントを同一タスクから操作できるようになった。従来は1アカウントに限定されていたが、仕事用・個人用・クライアント用など複数アカウントを同時に接続し、タスク実行時に使うアカウントを選択できる。カレンダー間の二重予約防止や、クライアント業務の切り分けが容易になる。

## できるようになったこと

- 複数の **Gmail / Google カレンダーアカウント** を同時接続。
- タスク実行時に**使用アカウントを選択**可能（仕事用 / 個人用 / クライアント用など）。
- カレンダー横断の二重予約防止、クライアント業務の分離が容易に。

## 影響範囲

- 対象ユーザー: Manus 利用者（複数 Google アカウントを使い分ける個人・フリーランス・代行業務）
- 対象プラン: 記載なし（Manus の Google コネクタ利用者）
- API / UI / 管理者機能: Manus の Google コネクタ（アカウント接続・選択 UI）

## 教材化メモ

Manus の「コネクタ」章に、Gmail / Google カレンダーの複数アカウント接続と実行時選択を追記。複数アカウントを使い分けるユースケース（クライアント業務の分離、私用・社用の切り分け、カレンダー横断の予約調整）を例示する。既存コネクタ機能の派生強化（単一→複数アカウント）として位置づける。
