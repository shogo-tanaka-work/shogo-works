# 2026-06-17 Grok on Amazon Bedrock

## 公式内容の日本語要約

xAI の Grok モデルが Amazon Bedrock のマネージドサービス経由でアクセス可能になった。AWS エコシステムとの統合が実現し、Bedrock 経由で Grok を呼び出せる。

## できるようになったこと

- Amazon Bedrock の API・SDK から Grok モデルを呼び出せる
- AWS のセキュリティ・コンプライアンス基盤上で Grok を利用できる
- 既存の Bedrock ワークフローに Grok を組み込める

## 影響範囲

- 対象ユーザー: Amazon Bedrock ユーザー、AWS を利用する企業
