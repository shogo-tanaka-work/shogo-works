# 2026-06-17 Grok Imagine Video 1.5 広範リリース

## 公式内容の日本語要約

xAI の image-to-video モデル「Grok Imagine Video 1.5」が広範リリースされた。Aurora-2 エンジンベースで 720p 動画を約25秒（従来40秒以上）で生成可能。物理シミュレーションの精度も向上している。

## できるようになったこと

- 720p 動画を約25秒で生成（従来比 40% 以上高速化）
- Aurora-2 エンジンによる物理シミュレーション精度向上
- 広範リリースにより多くの Grok ユーザーがアクセス可能に

## 影響範囲

- 対象ユーザー: Grok ユーザー（プラン詳細は公式確認要）
- 対象プラン: 詳細は公式確認要
- API / UI / 管理者機能: Grok UI から image-to-video 機能として利用
