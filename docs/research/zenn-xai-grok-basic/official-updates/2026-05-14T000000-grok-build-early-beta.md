# 2026-05-14 Grok Build Early Beta

## 公式内容の日本語要約

xAIは、professional software engineeringとcomplex coding work向けのcoding agent / CLIであるGrok Build Early Betaを発表した。SuperGrok Heavy subscribers向けに先行提供され、terminalからinstallして使える。plan mode、diff review、plugins、hooks、skills、MCP servers、parallel subagents、worktree integrations、headless mode、ACP supportなどを備える。

## できるようになったこと

- terminal上でGrok Buildを使い、coding agentとして作業できる。
- plan modeで実行前にplanを承認・修正できる。
- AGENTS.md、plugins、hooks、skills、MCP serversを読み込める。
- parallel subagentsとworktree integrationで大きな調査・実装を分割できる。
- headless modeとACP supportでscript / automation / agent orchestrationに組み込める。

## 影響範囲

- 対象ユーザー: SuperGrok Heavy subscribers、coding agent利用者、開発チーム
- 対象プラン: SuperGrok Heavy early beta
- API / UI / 管理者機能: CLI / agent / plugins / MCP / ACP
