# 2026-06-18 Grok for Microsoft Word — Microsoft 365 アドインとして提供開始

## 公式内容の日本語要約

Grok が Microsoft 365 アドインとして Microsoft 365 ストアで提供開始した。Word のサイドパネルからドラフト作成・編集・校正・文体改善・フォーマット統一が可能になる。リアルタイムの Web 検索と X (旧 Twitter) データへのアクセスを文書内で利用でき、メール・SharePoint・Google Drive との連携コネクタも搭載する。

## できるようになったこと

- Word のサイドパネルから Grok を使ってドラフト作成・編集・校正・フォーマット統一
- リアルタイム Web 検索と X データを Word 文書の中で参照可能
- メール・SharePoint・Google Drive コネクタで外部コンテキストを補強
- ユーザーがインライン編集候補を採用・取り消し可能な UI
- Microsoft Copilot との差別化：リアルタイムインターネットアクセスとビジュアルダイアグラム生成

## 影響範囲

- 対象ユーザー: Microsoft 365 サブスクライバー
- 対象プラン: Microsoft 365（Word リボンからアドインをインストール）
- API / UI / 管理者機能: Word アドイン（UI 提供のみ、独立 API は未公表）
