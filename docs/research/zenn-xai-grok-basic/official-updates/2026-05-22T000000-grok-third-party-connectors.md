# 2026-05-22 Grok サードパーティコネクター拡充

## 公式内容の日本語要約

xAI が Grok にサードパーティコネクター 4 種を追加した。Grok を離れずに Vercel でのサイトビルド・デプロイ、Canva でのデザイン作成、Gamma でのプレゼンテーション生成、S&P Global からのリアルタイム市場データ取得が可能になる。Grok を「オールインワン型のプロダクティビティハブ」に進化させる方針の一環。

> 公式 URL（`grok.com/release-notes` / `docs.x.ai/developers/release-notes` / `x.ai/news`）は直アクセスで HTTP 403 を返す環境があるため、本記録は二次ソース（basenor.com の Grok 月次更新まとめ）経由で内容確認。`date_precision: date-only`、`status: 保留（公式直アクセス不可・二次ソース経由）` として扱う。

## できるようになったこと

- Grok 内から Vercel への**サイトビルド・デプロイ**が可能になった。
- Canva での**デザイン作成**を Grok から直接行えるようになった。
- Gamma を通じた**プレゼンテーション生成**が Grok 経由で可能になった。
- S&P Global の**リアルタイム市場データ**取得を Grok 内で実行できるようになった。

## 影響範囲

- 対象ユーザー: Grok 利用者全般。フロントエンド開発者（Vercel）、デザイナー / マーケ（Canva・Gamma）、金融・調査担当（S&P Global）。
- 対象プラン: 公式 release notes 直確認不可のため、プラン別の提供範囲はサマリーでは未確定（要、公式アクセス可能時に追補）。
- API / UI / 管理者機能: Grok アプリ / Web の機能拡張。
