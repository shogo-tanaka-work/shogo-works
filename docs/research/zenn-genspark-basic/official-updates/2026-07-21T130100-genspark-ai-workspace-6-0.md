# 2026-07-21 Genspark AI Workspace 6.0

## 公式内容の日本語要約

Gensparkが「AIワークスペース 6.0」を発表。目玉機能は複数プラットフォームをまたいで作業情報を自動集約・整理するクラウド記憶機能「SecondBrain」と、それと自動連携する専用AIボイスレコーダー「SecondBrain Note」（最大35時間録音、自動文字起こし・要約）。人とAIエージェントがチャットグループで協働できる「GenTeam」、文体学習型メール自動化クライアント「GenMail」、複数データソースを統合するダッシュボード作成ツール「AgentBase」、サイト/アプリ/画像/ポスター制作向けの「Design」も同時公開。ChatGPT・Gemini・Claudeなど70以上のAIモデルを統合し、蓄積した文脈に基づく出力を目指す。

Genspark公式Blog（`genspark.ai/blog`）では「2026-07-20」付、日本語プレスリリース（PR TIMES）では「2026-07-21 13:01 JST」付で確認（タイムゾーン差によるもの。JST基準で対象窓内）。

## できるようになったこと

- SecondBrain: 作業データを横断的に自動集約・整理するクラウド記憶
- SecondBrain Note: 会議・会話を録音→自動文字起こし・要約するAIボイスレコーダー端末（Gensparkの初ハードウェア製品）
- GenTeam: AIエージェント（Codex/Claudeクラスを含む）と人がチャットグループで役割分担・協働できる空間
- GenMail: モバイル/デスクトップ対応のメールクライアント（文体学習・自動化）
- AgentBase: 複数データソース統合のダッシュボード作成
- Design: サイト・アプリ・画像・ポスター制作ツール

## 影響範囲

- 対象ユーザー: Genspark全ユーザー（個人〜チーム/エンタープライズ）
- 対象プラン: 詳細プラン別提供範囲は公式ページ参照
- API / UI / 管理者機能: Webワークスペース、専用ハードウェア（SecondBrain Note）、モバイル/デスクトップアプリ

## 教材化メモ

「AIに記憶を持たせる」実装アプローチ（クラウド記憶＋専用録音デバイス連携）として、パーソナルAIアシスタント文脈の教材事例に使える。GenTeamのような「人とAIエージェント混成チャット」は、マルチエージェント協働UIの具体例として紹介できる。
