# 2026-07-09 Muse Spark 1.1

## 公式内容の日本語要約

Meta Superintelligence Labsが、エージェント型タスク向けのマルチモーダル推論モデル「Muse Spark 1.1」を発表した。1Mトークンのコンテキストウィンドウを能動的に管理し、複数アプリにまたがるcomputer-useワークフローや、大規模で複雑なコードベースのバグ修正・機能実装・コード移行に強みを持つ。マルチエージェントによる並列タスク実行、引用付きの組み込み検索、構造化出力にも対応する。新設の「Meta Model API」を通じて公開プレビュー段階でデベロッパー提供が始まり、入力$1.25/百万トークン・出力$4.25/百万トークンの課金体系が設定された。Metaが自社モデルで法人向け有償課金を行う初めての事例。Meta AIアプリ・meta.ai上では「Thinking」モードとして先行提供されている。

## できるようになったこと

- 1Mトークンのコンテキストウィンドウを持つエージェント型推論モデルの利用開始
- 複数アプリにまたがるcomputer-useワークフロー、大規模コードベースでのコーディング支援
- マルチエージェントによる並列タスク実行、引用付き検索、構造化出力
- Meta Model API（公開プレビュー）経由での有償API提供（入力$1.25/出力$4.25、百万トークンあたり）

## 影響範囲

- 対象ユーザー: 開発者（Meta Model API経由）、Meta AIアプリ/meta.aiの一般ユーザー（Thinkingモード経由）
- 対象プラン: Meta Model API公開プレビュー（新設の有償課金体系）
- API / UI / 管理者機能: 新規API基盤（Meta Model API）の提供開始

## 教材化メモ

Metaが自社モデルAPIで初めて法人向け有償課金に踏み切った点が注目点。07-07発表の「Muse Image/Muse Video」（画像・動画生成）とは別系統で、こちらはエージェント型コーディング・推論モデル。OpenAI/Anthropic/xAI等との「エージェント型コーディングモデル」競争に本格参入したという文脈で扱える。`src/content.config.ts`のaiNews.tool enumに`meta-ai`が未対応のため、記事化する場合はenum追加の要否をユーザーに確認する必要がある。
