# 2026-07-07 Meta AI Muse Image / Muse Video

## 公式内容の日本語要約

Meta Superintelligence Labsが新画像/動画生成モデル「Muse Image」「Muse Video」を発表した。Muse Imageは単純なtext-to-imageではなく、Web検索・コード実行（QRコード生成・チャート作成等）を伴うエージェント型の推論・自己修正（self-refine）を特徴とし、複数の参照画像を合成する精密編集にも対応する。Arenaのtext-to-image/画像編集部門で人間評価Elo2位を獲得。Muse Videoは高い視覚的忠実度とネイティブ音声対応を持つプレビュー版として同時発表され、時系列一貫性を重視した設計。あわせてAI生成画像を検証できる不可視透かし「Content Seal」も導入された（検出ツール: meta.ai/identification）。

## できるようになったこと

- Muse Image: Web検索・コード実行を組み合わせたエージェント型の高精度画像生成・編集（本日より提供開始）
- 複数参照画像からの合成、指示への忠実な追従
- Muse Video: 音声対応・高視覚忠実度の動画生成（プレビュー、開発中）
- Content Seal: AI生成画像であることを検証できる不可視透かし・検出ツール

## 影響範囲

- 対象ユーザー: Meta AIアプリ / meta.aiサイトの一般ユーザー、Instagram（米国のみ）・WhatsApp（一部国）ユーザー
- 対象プラン: 無料提供（Muse Image）。Muse Videoはクリエイター・Meta AIユーザー向けに近日提供予定でVideo自体は開発中扱い
- API / UI / 管理者機能: Meta AIアプリ / meta.ai / Instagram Stories / WhatsAppでの機能提供。Facebookは対応予定
