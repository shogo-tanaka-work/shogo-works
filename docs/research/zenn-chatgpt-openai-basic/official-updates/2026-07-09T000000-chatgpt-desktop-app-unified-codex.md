# 2026-07-09 ChatGPTデスクトップアプリがCodexアプリと統合 — Chat/Work/Codexを1アプリに

## 公式内容の日本語要約

既存のCodexデスクトップアプリを更新すると、Chat・Work・Codexを1つに統合した新しいChatGPTデスクトップアプリになる。既存Codexユーザーは更新後もCodexビューがデフォルトで開き、タスク・プロジェクトはそのまま維持される。旧ChatGPTデスクトップアプリは「ChatGPT Classic」に改名された。この統合は、同日発表のGPT-5.6 GA・ChatGPT Work発表と合わせた製品ライン再編の一環で、後述のAtlasブラウザ廃止発表ともつながっている。

## できるようになったこと

- macOS/Windowsのデスクトップアプリ1つでChat・Work・Codexの3機能を利用可能に
- 既存Codexユーザーは設定・タスク・プロジェクトを維持したまま移行
- 旧ChatGPTデスクトップアプリは「ChatGPT Classic」として区別

## 影響範囲

- 対象ユーザー: デスクトップ（macOS/Windows）利用者全般、無料プラン含む全プラン
- 対象プラン: 全プラン（機能面の利用可否は各プランの既存制限に従う）
- API / UI / 管理者機能: デスクトップアプリUIの統合。開発者はCodexアイコン表示を維持可能

## 教材化メモ

- 「複数アプリの統合によるプロダクトライン再編」の事例として、Atlasブラウザ廃止と合わせて説明すると流れが分かりやすい
