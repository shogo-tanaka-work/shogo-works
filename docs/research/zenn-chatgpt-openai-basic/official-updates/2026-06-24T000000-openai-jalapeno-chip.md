# 2026-06-24 OpenAI Jalapeño カスタム AI 推論チップ発表

## 公式内容の日本語要約

OpenAI が Broadcom と共同で開発した初の独自カスタム AI チップ「Jalapeño」を 2026-06-24 に発表した。LLM 推論（Inference）専用 ASIC で、テープアウトまでの開発期間はわずか 9 ヶ月。ボード・ラック・システム統合は Celestica が担当。Microsoft が初期生産の約 40% を購入予定。2026年末の展開開始を目標とし、「その後数年でギガワットスケールへ拡大」と明言している。

**注記**: openai.com/index/openai-broadcom-jalapeno-inference-chip/ が 403 のため、二次ソース（CNBC・TechCrunch・Tom's Hardware・The Decoder）で内容を補完。

## できるようになったこと

- OpenAI が自社設計の推論チップを量産展開できるようになる（現在は NVIDIA GPU に依存）
- OpenAI 自身の AI モデルをチップ設計プロセスに活用（AI による AI チップ設計）
- ギガワットスケールのデータセンター向けに最適化された性能/ワット比を実現（数値未公開）
- Microsoft との連携で初期展開の確実性を確保

## 影響範囲

- 対象ユーザー: エンドユーザーへの直接影響は 2026年末以降
- 対象プラン: インフラレベルの変更のため既存プランへの影響は段階的
- API / UI: 長期的には推論コスト低減・キャパシティ拡大への影響

## 教材化メモ

- 「AI インフラの垂直統合」戦略の具体例。Apple Silicon への転換と同じ「ハードウェア自社化で競争優位を作る」設計
- 9ヶ月という設計スピードは「AI を使った AI チップ設計」の実証例として教材化できる
- Microsoft が 40% を購入するという資本関係の深さはビジネス事例として使える
