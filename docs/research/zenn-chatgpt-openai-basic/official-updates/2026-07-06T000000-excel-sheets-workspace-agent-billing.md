# 2026-07-06 ChatGPT for Excel/Sheets・Workspace AgentがEnterprise/Edu向けトークン課金クレジットへ移行

## 公式内容の日本語要約

これまで無料提供されていたChatGPT for Excel/Sheetsタスク、Workspace Agentの実行が、7月6日より入力トークン・キャッシュ入力トークン・出力トークンに基づくトークン課金クレジット方式へ移行した。目安としてExcel/Sheetsタスクは1件あたり5〜20クレジット、Workspace Agent実行は5〜25クレジット（GPT-5.5利用時）。同時期にChatGPT for PowerPointはBusiness/Enterprise向けに8月6日まで無料期間を延長中と案内されている。

## できるようになったこと

- （課金方式の変更）ChatGPT for Excel/Sheets、Workspace Agentがトークン課金クレジット消費の対象に
- ChatGPT for PowerPointは引き続き8/6まで無料期間中

## 影響範囲

- 対象ユーザー: ChatGPT Enterprise/Edu（一部Business）の管理者・利用部門
- 対象プラン: Business/Enterprise/Edu
- API / UI / 管理者機能: 管理者機能（クレジット消費管理・予算計画）

## 教材化メモ

- クレジット消費目安（Excel/Sheets 5〜20、Workspace Agent 5〜25）は予算計画の説明に使える具体数値
- 無料機能の有償化タイミングの実例として、他ツールの料金改定説明の参考にできる
