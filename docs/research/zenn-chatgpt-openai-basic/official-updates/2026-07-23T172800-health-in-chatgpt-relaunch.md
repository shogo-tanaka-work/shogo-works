# 2026-07-23 ChatGPT Healthが米国ユーザー向けに拡大再展開

## 公式内容の日本語要約

OpenAIが「Health in ChatGPT」を米国の18歳以上のログイン済みユーザー向けにWeb・iOSで、Free/Go/Plus/Proの全プランに展開した。2026年1月の初期パイロットからのフィードバックを踏まえ、Apple Healthや対応する医療記録（患者ポータル等）を安全に接続し、直近の検査結果と過去の検査結果の比較、前回診察からの変化の要約、睡眠・活動・運動と体調の関連の把握などができるようになった。健康関連の会話・接続アプリ・アップロードファイルは、ChatGPT本体とは分離されたサンドボックス環境で保持され、専用の暗号化・分離による保護が追加されている。診断・治療の代替ではなく、日常的な疑問の整理を支援する位置づけ。

## できるようになったこと

- Apple Healthおよび対応医療記録（患者ポータル等）をChatGPTに安全接続
- 検査結果の経時比較、前回受診からの変化の要約
- 睡眠・活動・運動データと体調の関連についての会話
- サイドバーの「Health」から利用開始

## 影響範囲

- 対象ユーザー: 米国の18歳以上、ログイン済みユーザー（欧州経済領域・スイス・英国は対象外）
- 対象プラン: Free、Go、Plus、Pro
- API / UI / 管理者機能: Web・iOSのみ。Codexでは未提供
