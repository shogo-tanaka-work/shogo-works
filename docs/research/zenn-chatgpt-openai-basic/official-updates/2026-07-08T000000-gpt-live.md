# 2026-07-08 GPT-Live発表 — ChatGPT Voiceが全二重会話に対応

## 公式内容の日本語要約

OpenAIが新世代音声モデル「GPT-Live」（GPT-Live-1、GPT-Live-1 mini）を発表した。フルデュプレックス（全二重）方式を採用し、AIが「聞きながら話す」ことができるため、相槌・割り込み・間の取り方など会話の自然さが向上する。有料プランはGPT-Live-1、Freeプランを含む既存のAdvanced Voice ModeはGPT-Live-1 miniに置き換わる形で、世界のChatGPTユーザーへロールアウトを開始した。裏側では複雑な質問をGPT-5.5系フロンティアモデルへ委譲する構成になっている。API提供は今後の予定でサインアップ受付のみ。専用のGPT-Live System Cardも同日公開され、児童保護・自傷対応・音声なりすまし防止などの安全対策が明記された。

## できるようになったこと

- ChatGPT Voiceが全二重会話に対応し、相槌・割り込みなど自然な会話体験が可能に
- 有料プランはGPT-Live-1、Freeプランを含む既存Advanced Voice ModeはGPT-Live-1 miniへ置き換え
- 複雑な質問はGPT-5.5系フロンティアモデルへ裏側で委譲

## 影響範囲

- 対象ユーザー: 全プラン（Free含む）のChatGPT音声利用者
- 対象プラン: 全プラン
- API / UI / 管理者機能: UI（音声モード）が対象。API展開は今後（サインアップ受付のみ）
