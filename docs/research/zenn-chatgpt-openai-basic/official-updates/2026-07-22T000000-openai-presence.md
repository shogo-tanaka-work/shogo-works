# 2026-07-22 OpenAI Presence

## 公式内容の日本語要約

OpenAIが企業向けの管理型AIエージェントプラットフォーム「OpenAI Presence」を発表した。音声・チャット（将来的にメールも）でエージェントを構築・展開・運用・継続改善するための仕組みで、企業のポリシー/SOP、アクセス権限の制御、エスカレーションルール、事前デプロイシミュレーション（想定質問・エッジケース・高リスクシナリオのテスト）をバンドルする。リアルタイム音声にはGPT-Liveを利用し、Codexエージェントが運用ログを分析して改善案を提案するが、実際の適用は人間の承認を要する。カスタマーサポート、アウトバウンド営業、調達、IT、HRなどのユースケースを想定し、限定的General Availability（limited GA）として提供開始。セルフサーブではなく、OpenAIのForward Deployed Engineersや厳選されたシステムインテグレーターが導入を主導する。自社の英語音声サポート窓口では、人手を介さず75%の入電を解決している。

## できるようになったこと

- 音声/チャットのAIエージェントを、ポリシー・権限・エスカレーションルール込みで管理型プラットフォームとして展開
- 本番投入前のシミュレーション（想定質問・エッジケース・高リスクシナリオ）
- Codexエージェントによる運用ログ分析と改善提案（人間承認必須）
