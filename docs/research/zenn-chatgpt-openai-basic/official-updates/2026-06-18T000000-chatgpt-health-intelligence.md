# 2026-06-18 ChatGPT 健康インテリジェンス強化

## 公式内容の日本語要約

GPT-5.5 Instant が健康・医療分野の回答品質を大幅に向上。260名以上の医師・60カ国との協力のもと開発され、医師パネルによる評価で医師が書いた回答より高スコアを記録した。事実誤りが71%減少し、緊急ケアの検出精度も改善。毎週2億3000万人が健康目的でChatGPTを利用している中、無料ユーザーにも展開された。

## できるようになったこと

- 無料ユーザーが GPT-5.5 Instant の高精度な健康回答を利用可能
- 緊急受診が必要なケースの検出精度向上
- 不確実性を誇張せず適切に説明する回答スタイル
- HealthBench Professional で最新フロンティアモデル相当の性能を達成

## 影響範囲

- 対象ユーザー: 全ユーザー（無料含む）
- 対象プラン: 全プラン
- API / UI / 管理者機能: UI
