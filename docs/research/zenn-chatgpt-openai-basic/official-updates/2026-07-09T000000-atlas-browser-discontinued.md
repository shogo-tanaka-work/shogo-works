# 2026-07-09 ChatGPT Atlasブラウザの廃止を発表（8/9停止予定）

## 公式内容の日本語要約

OpenAIのJames Sun氏がXポストで、スタンドアロン型デスクトップブラウザ「ChatGPT Atlas」（2025-10ローンチ）の廃止を発表。2026-08-09に提供終了予定。Atlasで培ったエージェント型ブラウジング機能は、新しい統合ChatGPTデスクトップアプリとGoogle Chrome拡張機能に再配置される。同日発表のデスクトップアプリ統合（Chat/Work/Codex統合）に伴うプロダクトライン再編の一環。詳細案内はアプリ内通知とメールで提供予定。

## できるようになったこと

- （廃止発表のため新機能はなし。Atlasのエージェント型ブラウジング機能は新デスクトップアプリとChrome拡張へ移管）

## 影響範囲

- 対象ユーザー: ChatGPT Atlas利用者（Free/Plus/Pro/Go、Business/Enterpriseベータ利用者含む）
- 対象プラン: 全プランのAtlas利用者
- API / UI / 管理者機能: スタンドアロンブラウザアプリの終了。管理者はEnterprise向けAtlas設定の見直しが必要
