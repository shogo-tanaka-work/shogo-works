# 2026-07-09 GPT-5.6がMicrosoft 365 Copilotの優先モデルに採用

## 公式内容の日本語要約

GPT-5.6ファミリーが、Word・Excel・PowerPoint・Copilot Chat・Copilot Coworkの新しい優先（preferred）モデルとして導入された。エージェント的なマルチステップ作業でのパフォーマンス・コスト効率・成果物の完成度が向上したとされる。OpenAI公式とMicrosoft公式（Tech Community）の両方で同日発表されている。

## できるようになったこと

- Microsoft 365 CopilotのWord/Excel/PowerPoint/Chat/CoworkでGPT-5.6を優先モデルとして利用
- 複雑なマルチステップ・エージェント作業の処理効率が向上

## 影響範囲

- 対象ユーザー: Microsoft 365 Copilot利用企業・ユーザー全般
- 対象プラン: Microsoft 365 Copilotライセンス保有者
- API / UI / 管理者機能: Copilotのモデルピッカー・バックエンド切替。管理者によるモデル選択制御が関わる可能性
