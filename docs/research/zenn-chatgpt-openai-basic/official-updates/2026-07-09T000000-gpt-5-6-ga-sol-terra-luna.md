# 2026-07-09 GPT-5.6（Sol/Terra/Luna）がChatGPT・API・Codexで一般提供開始

## 公式内容の日本語要約

2026-06-26開始の限定プレビュー（政府審査済み約20パートナー限定）を経て、2026-07-09にGPT-5.6ファミリー（Sol=フラッグシップ、Terra=バランス型、Luna=最速・低コスト）がChatGPT・API・Codexで一般提供（GA）開始。ロールアウトは07-09にグローバルで始まり、以後24時間かけて段階的に完了する方式で、07-10時点でも地域・プラン・管理者設定によって展開途上のケースがある。API価格は入出力1MトークンあたりSol $5/$30、Terra $2.50/$15、Luna $1/$6。「Ultra」モードでサブエージェントによる並列作業も追加された。

## できるようになったこと

- ChatGPT / API / Codexの3面同時にGPT-5.6ファミリーがGA
- Free/GoプランはTerraのみ利用可能
- Plus/Pro/Business/Enterprise/Eduは Sol・Terra・Luna を選択可能、effortレベル設定にも対応
- Pro/Enterpriseは最難タスク向けの「Sol Pro」も選択可能

## 影響範囲

- 対象ユーザー: ChatGPT全プラン利用者、API利用者
- 対象プラン: Free/Go（Terraのみ）、Plus以上（Sol/Terra/Luna選択可）
- API / UI / 管理者機能: モデルピッカー、APIモデル一覧、ワークスペース管理者によるモデルアクセス制限が対象
