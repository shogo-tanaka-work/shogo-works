# 2026-07-15 Custom instructions character limit raised to 5,000

## 公式内容の日本語要約

Plus・Pro・Enterprise・Business・Education向けに、ChatGPTカスタムインストラクションの保存可能文字数が従来の1,500文字から5,000文字に拡大された。Free・Goユーザーは1,500文字のまま据え置き。より詳細なブリーフィング文書レベルの指示を保存できるようになった。

## できるようになったこと

- カスタムインストラクションの保存文字数が1,500文字から5,000文字に拡大（Plus/Pro/Enterprise/Business/Education）

## 影響範囲

- 対象ユーザー: Plus・Pro・Enterprise・Business・Educationプランユーザー
- 対象プラン: Plus / Pro / Enterprise / Business / Education（Free・Goは対象外）
- API / UI / 管理者機能: カスタムインストラクション設定

## 教材化メモ

- 業務利用でのパーソナライズ精度向上に直結する地味だが実務的な改善。プロンプトエンジニアリング教材で「カスタムインストラクションに何を書くべきか」の分量目安をアップデートする際の参考情報になる。
