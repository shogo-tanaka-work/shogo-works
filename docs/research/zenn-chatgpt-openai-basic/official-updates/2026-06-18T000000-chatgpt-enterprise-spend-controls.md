# 2026-06-18 ChatGPT Enterprise 支出管理・利用分析強化

## 公式内容の日本語要約

ChatGPT Enterprise のグローバル管理コンソールに、詳細な利用分析とクレジット支出管理機能が追加された。ユーザー別・製品別・モデル別の消費量を可視化し、管理者がワークスペース全体・チーム単位・個人単位で利用上限を設定できる。エンドユーザー自身も残クレジットを確認して追加申請が可能。

## できるようになったこと

- 管理者がワークスペース全体のデフォルト上限を設定
- チームまたは個人ごとの個別上限・例外設定
- ユーザー・製品・モデル別の消費トレンドをグローバルコンソールで一覧表示
- エンドユーザーが自分のクレジット残量を確認し追加申請

## 影響範囲

- 対象ユーザー: ChatGPT Enterprise 管理者・利用者
- 対象プラン: ChatGPT Enterprise
- API / UI / 管理者機能: 管理者コンソール
