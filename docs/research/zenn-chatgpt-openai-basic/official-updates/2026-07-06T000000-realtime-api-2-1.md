# 2026-07-06 Realtime API新モデル gpt-realtime-2.1 / gpt-realtime-2.1-mini を提供開始

## 公式内容の日本語要約

OpenAIが低遅延音声・マルチモーダル向けの新Realtimeモデル「gpt-realtime-2.1」「gpt-realtime-2.1-mini」の2種を提供開始した。改善キャッシュにより全Realtime系音声モデルでp95レイテンシを25%以上削減。上位のgpt-realtime-2.1は英数字認識・無音/ノイズ処理・割り込み挙動を改善し、mini版は低コストでのリアルタイム推論・ツール利用に対応する。料金はgpt-realtime-2.1-miniが音声入力$10/出力$20、gpt-realtime-2.1が入力$32/出力$64（各100万トークン当たり）。

## できるようになったこと

- gpt-realtime-2.1: 英数字認識・無音/ノイズ処理・割り込み挙動の改善
- gpt-realtime-2.1-mini: 低コストでのリアルタイム推論・ツール利用
- 全Realtime系音声モデルでp95レイテンシを25%以上削減

## 影響範囲

- 対象ユーザー: Realtime API利用の開発者・エンタープライズ
- 対象プラン: API利用者全般
- API / UI / 管理者機能: API（Realtime API）

## 教材化メモ

- レイテンシ改善幅（p95で25%以上）は音声アプリのベンチマーク比較に使える数値
- 上位/mini構成の料金差は用途別モデル選定の教材例になる
