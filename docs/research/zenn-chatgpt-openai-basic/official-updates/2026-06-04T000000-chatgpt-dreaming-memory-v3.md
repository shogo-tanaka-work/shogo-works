# 2026-06-04 ChatGPT メモリ「Dreaming」（Dreaming V3）

## 公式内容の日本語要約

OpenAI が ChatGPT のメモリを強化する「Dreaming（Dreaming V3）」を発表した。バックグラウンドで多数の会話からメモリ状態を合成し、常に新鮮で関連性の高い文脈を会話へ供給する。2026-06-04 に米国の Plus / Pro ユーザーから提供開始し、数週間で他国および Free / Go ユーザーへ拡大予定。計算コストを約 5 分の 1 に削減したことで、無料アカウントにもパーソナライズされたメモリを初めて提供する。

## できるようになったこと

- **会話からの自動学習**: 「覚えておいて」と明示しなくても、会話の中で自然に現れた文脈をメモリに取り込む。
- **時間経過でのメモリ自動更新**: 例として「7 月にシンガポールへ行く予定」を、旅行後に「2026 年 7 月にシンガポールへ行った」へ自動で改訂する。
- **無料層への初提供**: 計算コスト約 5x 削減により、Free アカウントでもパーソナライズドメモリを利用可能に。Plus / Pro はメモリ保存容量が 2 倍に。
- **透明性とコントロール**: 合成されたメモリはメモリサマリーページで確認・追加・修正でき、どの話題をいつ持ち出すかを指示できる。

## 影響範囲

- 対象ユーザー: まず米国の Plus / Pro。数週で他国・Free / Go へ拡大。
- 対象プラン: Free / Go / Plus / Pro（段階的）。
- API / UI / 管理者機能: ChatGPT アプリの UI（メモリサマリー）が中心。
