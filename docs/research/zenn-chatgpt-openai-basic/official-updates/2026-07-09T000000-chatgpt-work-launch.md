# 2026-07-09 ChatGPT Workを新規ローンチ — 長時間自律実行のエージェント機能

## 公式内容の日本語要約

OpenAIが「ChatGPT Work」を発表。目標を渡すだけでスプレッドシート・スライド・ドキュメント・Webサイトなどの完成物を作るエージェント機能で、GPT-5.6を搭載。数時間かけて複数ステップの複雑プロジェクトを自律実行し、Slack/Teams/Google Drive/SharePoint等と連携。Scheduled Tasksによるバックグラウンド実行にも対応する。Web/モバイルはPro/Enterprise/Eduから提供開始し、数日以内にPlus/Businessへ拡大予定。デスクトップは全プラン（Free含む）で提供される。

## できるようになったこと

- 目標（ゴール）を渡すだけで、複数アプリ・ファイルをまたいだ長時間タスクを自律実行
- 完成したスプレッドシート・スライド・ドキュメント・Webサイトなどの成果物を生成
- Slack / Teams / Google Drive / SharePoint等の外部ツール連携
- Scheduled Tasksでのバックグラウンド実行

## 影響範囲

- 対象ユーザー: 業務での長時間自律タスクを求めるユーザー
- 対象プラン: Web/モバイルはPro/Enterprise/Eduから開始、数日内にPlus/Businessへ拡大。デスクトップは全プラン（Free含む）
- API / UI / 管理者機能: UI新機能。外部ツール連携のため管理者による接続許可設定が関わる可能性

## 教材化メモ

- 「ゴール入力型」の長時間自律エージェントの実例として、Claude CoworkやGensparkのAI Workspaceとの比較教材になる
- 同日発表のGPT-5.6 GA・デスクトップアプリ統合とセットで「OpenAIのエージェント戦略転換」を説明する題材に使える
