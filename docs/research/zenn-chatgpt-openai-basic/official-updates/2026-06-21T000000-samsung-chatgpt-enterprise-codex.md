# 2026-06-21 Samsung Electronics が ChatGPT Enterprise・Codex を全従業員展開

## 公式内容の日本語要約

Samsung Electronics が ChatGPT Enterprise と Codex を全従業員に展開する計画を OpenAI と共同発表した。韓国国内の全従業員と全世界の DX（Device eXperience）部門の全従業員への展開を即時開始し、2026年末までに全世界展開完了を目指す。「OpenAI 史上最大規模のエンタープライズ展開の一つ」と位置付けられている。

2023年3月、Samsung 社内エンジニアが機密ソースコードと会議議事録を ChatGPT 経由で流出させる事故が発生し、生成 AI の全社使用を禁止していた。約3年後に Samsung SDS が OpenAI のリセラーパートナーシップを締結し、セキュリティ対策を整備した上で方針を転換した。

## できるようになったこと

- ChatGPT Enterprise によるガバナンス・アクセス管理・データ保護を担保した AI 業務活用
- Codex によるコード作成・レビュー・デバッグの開発者生産性向上
- 技術系以外の社員もアイデアをソフトウェア・社内ツール・自動化ワークフローに変換可能
- Google Gemini・Anthropic Claude と並列利用のマルチベンダー AI スタックを構築

## 影響範囲

- 対象ユーザー: Samsung Electronics 全従業員（約26万人規模・段階展開）
- 対象プラン: ChatGPT Enterprise、OpenAI Codex
- API / UI / 管理者機能: Enterprise 管理コンソール経由

## 教材化メモ

企業 AI 導入の大規模事例として重要。2023年情報漏洩禁止→2026年全社展開という転換は、AI リスク管理と導入判断のタイミングを教える事例。マルチベンダー AI スタック（OpenAI + Gemini + Claude）の共存という企業実態も教材化できる。Codex の韓国での週次アクティブユーザー 800%増という数値は Codex 普及の実態を示す。
