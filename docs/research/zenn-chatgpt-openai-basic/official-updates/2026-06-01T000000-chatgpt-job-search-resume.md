# 2026-06-01 ChatGPT に求人検索と履歴書作成が追加

## 公式内容の日本語要約

2026-06-01 の ChatGPT Release Notes に、求人検索（job search）と履歴書（resume）機能が追加された。ChatGPT が Indeed・Upwork・Appcast をはじめ Web 全体からライブの求人・フリーランス案件を提示し、履歴書のアップロード・作成・整形ダウンロードまで支援する。「求人の発見から履歴書の最適化まで、求職プロセスのより多くの部分を ChatGPT が支援できるようになった」と説明されている。

> 注: 出典の ChatGPT Release Notes（help.openai.com）は当環境では HTTP 403 で本文直接取得が不可。日付・内容は releasebot 経由で確認し、一次 URL を出典として記録。

## できるようになったこと

- **求人検索**: Indeed / Upwork / Appcast などのソースと Web 全体から、ライブの求人・フリーランス案件を横断して提示。
- **履歴書サポート**: 履歴書のアップロード、作成、プロフェッショナルに整形したフォーマットでのダウンロード。
- ChatGPT が「希望条件に合うライブ求人を見つけ → 履歴書を案件に合わせて調整する」という一連の流れを補助。

## 影響範囲

- 対象ユーザー: 求人検索は **米国ユーザー（全プラン: Free / Go / Plus / Pro）**。履歴書整形は **web・英語で全プラン（グローバル）**。
- 対象プラン: 全サブスクリプション階層。
- API / UI / 管理者機能: ChatGPT クライアント（UI）の新機能。API 提供の言及はなし。
