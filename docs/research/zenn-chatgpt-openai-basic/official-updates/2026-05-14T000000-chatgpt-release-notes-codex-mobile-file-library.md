# 2026-05-14 ChatGPT Release Notes: Codex mobile preview and File Library expansion

## 公式内容の日本語要約

ChatGPT Release Notesの2026-05-14見出しで、Codex remote access from the ChatGPT mobile appとFile Library expands to Free and Go usersが追加された。Codex mobile previewは、Mac host上で動くCodexをChatGPT mobile appから操作する機能。File LibraryはFree / Goにも拡張され、ファイル再利用とstorage managementが使えるようになる。

## できるようになったこと

- ChatGPT mobile appからCodex threadを開始・継続・承認・レビューできる。
- Codexのproject context、approvals、plugins、screenshots、terminal output、diffs、test resultsをmobile側で確認できる。
- Free / GoユーザーもFile Libraryでファイルを探し、再利用できる。
- Storage managementでplan別の保存容量を確認・管理できる。

## 影響範囲

- 対象ユーザー: ChatGPT mobile app利用者、Codex利用者、Free / Go / Plus / Business / Proユーザー
- 対象プラン: Codex mobile previewは全プラン、File LibraryはFree / Goへ拡張
- API / UI / 管理者機能: ChatGPT mobile / File Library / Storage
