# 2026-06-28 OpenAI Codex：使用量上限インシデント

## 公式内容の日本語要約

一部ユーザーで Codex の使用量上限が予想より速く消費されるという報告が 2026-06-28 から継続中。原因は不正利用・フラウド防止システムが特定アカウントを誤ってレート制限していること。OpenAI は「影響は特定アカウントに限定されており、サービス全体の障害は観測されていない」と説明。ステータスは "Monitoring" 継続中（2日間以上）。

## 影響範囲

- 対象ユーザー: 一部の Codex ユーザー（特定アカウント）
- 全体サービス可用性: API 99.99%、ChatGPT 99.80%、Codex 99.96%（3〜6月通算）
