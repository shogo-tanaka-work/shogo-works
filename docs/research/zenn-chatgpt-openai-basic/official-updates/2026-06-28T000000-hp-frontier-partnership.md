# 2026-06-28 OpenAI：HP Inc. が Frontier プラットフォームを採用

## 公式内容の日本語要約

HP Inc. が OpenAI の Frontier プラットフォームを採用する戦略的パートナーシップを発表した。顧客向け体験の改善・社内業務変革・ソフトウェア開発への AI エージェント展開が主な用途。2026年2月から開始したパイロット評価（エージェント機能・セキュリティ・エンタープライズ統合）を経て正式契約に至った。Intuit・Oracle・Uber に続く Frontier 採用企業として HP が参画。

## できるようになったこと

- HP のエンタープライズ環境で OpenAI Frontier のエージェント機能を展開
- 顧客体験改善・社内業務・ソフトウェア開発の3領域に活用

## 影響範囲

- 対象ユーザー: OpenAI Frontier を評価している大企業
