# 2026-06-16 OpenAI Codex v0.140.0

## 公式内容の日本語要約

**新機能:**

- **`/usage` ビューの追加**: 日次・週次・累計のアカウントトークン利用状況を確認できるビューが追加（#27925）。
- **`/goal` の大きなコンテンツ対応**: 大容量テキスト・大量貼り付けブロック・画像添付がリモート app-server セッションを含めて保持されるようになった（#27508, #27509, #27510）。
- **永続的セッション削除 `/delete`**: `codex delete`、`/delete`、app-server `thread/delete` を通じた永続的なセッション削除が可能に。確認ガード・サブエージェントクリーンアップ付き（#25018, #27476）。
- **`/import` の追加（Claude Code からのインポート）**: セットアップ・プロジェクト設定・最近のチャットを選択的に Claude Code からインポートできる機能（#27070, #27071, #27703）。
- **`@` による統合メンションメニュー**: `@` タイプでファイル・プラグイン・スキルを一括検索できるメニューがデフォルトに（#27499）。
- **Amazon Bedrock マネージド API キー認証**: CLI および MCP OAuth 認証情報を暗号化ローカルストレージで管理（#27443, #27689 等）。

**バグ修正:**

- 破損した SQLite 状態データベースをロールアウトデータから自動バックアップ・再構築（#26859, #27719）
- `Esc` 押下時の `/review` クラッシュ修正（#22879）
- MCP 信頼性改善: 起動失敗のリトライ・OAuth 認証情報の無効通知・明示的無効化サーバーの保持（#25147, #26713, #27414）
- リモートプラグインのアンインストールリクエスト修正・インストール時の認証要求サーフェス改善（#27085, #27223）
- 更新無視の永続化・実行中 hook インジケーターの正常クリア（#27619, #27783）
- 非 TTY バックグラウンドコマンドの Ctrl-C 中断サポート（#26734）

**パフォーマンス改善:**

- Git 組み込みファイルシステムモニター保持・重複履歴読み取り回避・アーカイブルックアップ高速化・ターン差分レンダリングキャッシュで大規模リポジトリ・長期セッションの応答性向上（#26880, #27031, #27276, #27489）
- 実験的な `/realtime` 音声コントロールと関連オーディオ依存を TUI から削除（#27801）

## できるようになったこと

- `/usage` でトークン消費量を日次・週次・累計で確認できる
- `/delete` でセッションを完全削除できる（以前は削除手段が限られていた）
- `/import` で Claude Code のプロジェクト設定・チャット履歴を Codex に取り込める
- `@` 一発でファイル・プラグイン・スキルを統合検索できる
- Amazon Bedrock 認証情報を Codex に直接登録・暗号化保存できる

## 影響範囲

- 対象ユーザー: OpenAI Codex 全ユーザー
- 対象プラン: Bedrock 認証は AWS 環境・エンタープライズ利用者に重要
- API / UI / 管理者機能: `/import` は Claude Code → Codex 移行ユーザーに直接関連
