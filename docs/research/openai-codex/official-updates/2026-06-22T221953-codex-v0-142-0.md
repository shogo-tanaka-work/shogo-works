# 2026-06-22 OpenAI Codex v0.142.0 stable

## 公式内容の日本語要約

多数の新機能を含む stable リリース。利用上限リセットクレジットの `/usage` 画面での確認・還元、プラグインのカテゴリ整理、エージェントスレッドへのトークン予算設定、App-server レベルのマルチエージェント委譲制御、インデックス付き Web 検索モード、スケジュール時刻リマインダーが主な追加機能。

## できるようになったこと

- `/usage` コマンドで利用上限リセットクレジットの確認・還元が可能に（確認・リトライ・再取得状態付き）
- `/plugins` がリモートプラグインを「OpenAI Curated / Workspace / Shared with me」の3カテゴリに整理表示。関連ターンに対してプラグインを自動推薦・インストール提案
- エージェントスレッドをまたぐ「ロールアウトトークン予算」が設定可能に。残余予算リマインダー付き、予算超過時に自動中断
- App-server クライアントでマルチエージェント委譲を「無効・明示要求のみ・積極的」の3段階でスレッド・ターンごとに設定可能
- インデックス付き Web 検索モード追加: ライブ検索を許可しつつ、直接ページアクセスはサーバー承認 URL のみに制限
- UTC 時刻の定期リマインダーを受信・現在時刻の直接取得（App-server クロック経由含む）が可能に

## 主要バグ修正

- Linux TUI: Ctrl+Z サスペンド → `fg` 再開後の表示崩壊を修正
- Exec-server プロセスと stdio MCP セッションが一時的な切断（署名URL リフレッシュ等）を乗り越えて継続するよう修正
- リモート環境でネイティブパス・シェル・`AGENTS.md` 検出・サンドボックス動作が OS をまたいで維持されるよう修正
- 親エージェントがサブエージェントの終端エラーを空の成功として受け取っていた問題を修正
- goal-first スレッドが `thread/list` と `thread/search` から消失していた問題を修正

## 影響範囲

- 対象ユーザー: OpenAI Codex ユーザー（全プラン）
- 対象プラン: 全プラン（App-server 機能は API 利用者向け）
- API / UI / 管理者機能: CLI / TUI 改善・App-server API 追加

## 教材化メモ

エージェントスレッドへのトークン予算設定は「AI エージェントのコスト管理」という企業導入で重要なテーマの具体実装例として使える。インデックス付き Web 検索モードは、セキュリティポリシーを保ちながら Web 情報を参照させる設計パターンの事例。
