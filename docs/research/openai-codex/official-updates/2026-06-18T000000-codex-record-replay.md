# 2026-06-18 Codex Record & Replay (macOS)

## 公式内容の日本語要約

macOS 向け Codex アプリに Record & Replay 機能が追加。ユーザーが一度デモとして実行した繰り返しワークフロー（例: 経費申請、YouTube動画アップロード）を Codex が学習し、再利用可能なスキルとして保存・編集できる。Computer Use のセットアップが必要。EU域内では利用不可。

## できるようになったこと

- 反復タスクを一度見せるだけでスキルとして自動生成
- 生成したワークフローファイルを開いて調整・拡張
- 録画の開始・停止をユーザーが制御
- 生成されたスキルを再実行して繰り返し利用

## 影響範囲

- 対象ユーザー: macOS版 Codex ユーザー（EU除外、段階的展開）
- 対象プラン: Codex macOS アプリ利用者
