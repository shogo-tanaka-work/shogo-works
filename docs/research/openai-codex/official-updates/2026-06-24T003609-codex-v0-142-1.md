# 2026-06-24 OpenAI Codex v0.142.1 stable

## 公式内容の日本語要約

Codex CLI の stable パッチリリース。Windows 向けにシステムプロキシサポートがオプトイン追加された。PAC（Proxy Auto-Config）・WPAD・静的プロキシ・バイパスルールに対応。

## できるようになったこと

- Windows システムプロキシ設定（PAC・WPAD・静的プロキシ・バイパスルール）を使って Codex CLI の認証通信をルーティング可能
- プロキシが必要な企業ネットワーク環境での Codex CLI 利用がオプトインで可能に

## 影響範囲

- 対象ユーザー: Windows で Codex CLI を使う企業ユーザー
- 対象プラン: 全プラン
- API / UI: CLI の認証処理
