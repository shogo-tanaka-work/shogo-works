# 2026-06-04 OpenAI Codex rust-v0.137.0

## 公式内容の日本語要約

OpenAI Codex の安定版 rust-v0.137.0 がリリースされた。TUI 操作性、Enterprise/管理者向け機能、Remote Control、プラグイン、ホスト型 Web/画像ツール、マルチエージェント v2 にまたがる更新を含む。直前の窓内には alpha.4 / alpha.5 のプレリリースも出ている（詳細化対象外）。

## できるようになったこと

- **TUI**: F13〜F24 のキーバインド対応、検索可能メニューでのペースト、推論のみのコンパクトなステータス/タイトル表示。
- **Enterprise / 管理者**: 月次クレジット上限の表示、クラウド管理の設定バンドル（config bundle）適用、EDU ワークスペース対応。
- **Remote Control**: app-server v2 RPC 経由でペアリング開始、コントローラー権限の一覧表示・取り消しが可能に。
- **プラグイン**: `codex plugin list --json` で機械可読出力、リモートカタログ提案のキャッシュ。
- **ホスト型ツール**: より多くの code-mode フローで Web/画像ツールを利用可能。単独 Web 検索を並列実行できる。
- **マルチエージェント v2**: スレッドごとにランタイム選択を保持。spawn したエージェントのフォローアップ・メタデータ既定を整理。
- **権限**: 権限リクエスト/承認に環境アイデンティティを付与。managed MITM proxy が子コマンドへ読める CA バンドルをエクスポート。

## 影響範囲

- 対象ユーザー: Codex CLI 利用者全般。Enterprise/EDU 管理者は新機能の影響が大きい。
- 対象プラン: 一般機能は全ユーザー。クレジット上限・クラウド管理設定・EDU は Enterprise/EDU ワークスペース。
- API / UI / 管理者機能: TUI（UI）、app-server v2 RPC（API）、config bundle・クレジット上限（管理者機能）。

## 教材化メモ

Codex 教材の「プラグイン」「Enterprise 運用」章に、`codex plugin list --json` と月次クレジット上限・クラウド管理設定バンドル（EDU 含む）を追記。Remote Control のペアリング/権限取り消しは「リモート実行・ガバナンス」の論点として整理。並列 Web 検索は調査系ワークフローの高速化フックとして触れる。
