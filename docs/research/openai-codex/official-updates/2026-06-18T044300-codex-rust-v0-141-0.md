# 2026-06-18 Codex rust-v0.141.0

## 公式内容の日本語要約

リモート実行環境にて認証付き Noise リレーチャネルを採用し、クロスプラットフォームでネイティブのワーキングディレクトリとシェルを維持できるようになった。エグゼキュータープラグインが stdio MCP サーバーをスレッドごとに起動可能になるなど、リモート実行の信頼性を大幅強化。

## できるようになったこと

- リモート実行環境が認証付き Noise リレーチャネルを採用（セキュリティ向上）
- クロスプラットフォームでネイティブのワーキングディレクトリとシェルを保持
- エグゼキュータープラグインがスレッドごとに stdio MCP サーバーを起動可能
- Windows サンドボックス実行の認証情報修正・PowerShell コマンド処理時間延長
- アイドル状態の exec-server リレーが接続を維持するよう修正

## 影響範囲

- 対象ユーザー: Codex CLI 利用者全般
- 対象プラン: 全プラン
- API / UI / 管理者機能: CLI

**alpha リリース（件数のみ）:** 期間内に 0.141.0-alpha.7、0.142.0-alpha.1〜3 計4件公開。
