# OpenAI Codex v0.142.2 (stable)

## 主要変更点

### MCP ツールのデフォルト動作変更
- MCP ツールがデフォルトでツール検索（tool search）を使用するようになった
- より正確なツール選択が可能に

### macOS プロキシサポート
- macOS の認証クライアントがシステムプロキシ（PAC、WPAD）を適切に処理するように改善
- 企業環境のプロキシ設定を自動で尊重

### ダークモード対応
- プラグインのダークモードロゴをサポート

### 安全性 UI の改善
- safety-buffering（安全バッファリング）UI を改善

### Bedrock 認証情報エラー改善
- AWS Bedrock の認証情報エラーメッセージが改善

### バグ修正
- MCP サーバー関連の複数バグを修正
- PowerShell 安全性関連の修正
