# 2026-07-09 Codex rust-v0.144.0

## 公式内容の日本語要約

宣言済みの読み取り専用アクションは許可しつつ書き込み時のみ確認を求める「writes」アプローチモードを追加。MCPツールが実験的opt-inなしにインタラクティブ認証をリクエスト可能に。App-serverホスト側でランタイム時にCodex認証を提供しログイン成功後にホスト先ページへリダイレクトできるようになった。usage-limitリセットクレジットに種別・有効期限表示と選択機能を追加。グローバルpnpmインストールを診断・更新で正しく検出するよう修正。Ultra推論選択時、マルチエージェント高並列で使用量が急増しうる場合に警告を表示。バグ修正としてIntel macOSでのCode Modeクラッシュ、Windowsサンドボックスでの書き込みルート削除不可、TUIでの制御シーケンス貼り付け破損なども解消。

派生改善: リモートプラグインのnamespace解決をroot単位で1回にまとめ読み込み時間短縮、hookのコマンド実行トレース追加、`/review`のブランチピッカー高速化。

## できるようになったこと

- writes承認モード（読み取り自動許可・書き込みのみ確認）
- MCPツールのインタラクティブ認証（opt-inなし）
- app-serverホスト側のCodex認証・ページリダイレクト
- usage-limitリセットクレジットの種別・有効期限表示

## 影響範囲

- 対象ユーザー: CLI/エージェント運用者全般。MCPコネクタ利用者、CI/エージェント連携運用者
- 対象プラン: 該当なし（CLIツール）
- API / UI / 管理者機能: 承認モード設定、MCP認証
