# 2026-05-22 Anthropic Project Glasswing 中間報告

## 公式内容の日本語要約

Anthropic は 2026-05-22 に Project Glasswing の進捗を公表した。Claude Mythos Preview モデルを用いた防御的サイバーセキュリティ研究で、約 50 のパートナーと連携し、世界の重要ソフトウェアをスキャンしている取り組みの中間報告。オープンソースプロジェクト 1,000 件以上で 6,202 件の高・重大度脆弱性を特定し、研究全体では合計 1 万件超の高・重大度脆弱性を検出したと報告。一方で、パッチ適用ペースが検出ペースに追いつかないという課題も明示している。

## できるようになったこと

- Claude Mythos Preview モデルが、防御的サイバーセキュリティ分野で**大規模な脆弱性検出に実用段階**にあることが公式に確認された。
- Project Glasswing は招待制リサーチプレビューとして外部パートナー（約 50 組織）と運用されており、**重要ソフトウェアの体系的スキャン**が進んでいる。
- オープンソース 1,000 件以上で 6,202 件、合計で 1 万件超の高・重大度脆弱性が検出された具体的数値が公開された。

## 影響範囲

- 対象ユーザー: 防御的セキュリティチーム、OSS メンテナー、サプライチェーンセキュリティ責任者、リサーチパートナー候補。
- 対象プラン: リサーチプレビュー（招待制）。一般プランからの直接利用は今回の発表対象外。
- API / UI / 管理者機能: 該当なし（リサーチ報告）。

## 教材化メモ

- Claude の活用領域として「コードレビュー・開発支援」だけでなく、**防御的セキュリティ（脆弱性検出）も Anthropic 自身が成果を示す段階**に来た、という事実として教材で参照可能。
- 数値（6,202 件 / 1 万件超）は鮮度のある情報のため恒久 Knowledge に直接持ち込まず、ニュース記事リンク経由で扱う。
- 業界全体の「検出と適用のギャップ」という Anthropic の指摘は、セキュリティ運用カテゴリの参考ポイント。
