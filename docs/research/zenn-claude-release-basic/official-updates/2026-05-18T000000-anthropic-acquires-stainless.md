# 2026-05-18 Anthropic が Stainless を買収

## 公式内容の日本語要約

Anthropic は、SDK・MCP サーバーツール生成を専門とする Stainless の買収を発表した。Stainless は2022年創業で、API 提供開始当初から Anthropic 公式 SDK の生成を担ってきた。TypeScript / Python / Go / Java / Kotlin など複数言語の SDK、CLI、MCP サーバーコネクタを生成し、数百社が開発者・AI エージェントの API 連携に利用している。

Anthropic は「エージェントは接続できる対象の有用性以上にはなれない（Agents are only as useful as what they can connect to）」とし、エージェント接続性を Claude の中核能力と位置づけて Stainless の知見を取り込む。買収により Claude Platform 上での開発者体験とエージェント接続性の強化を狙う。Stainless 創業者 Alex Rattray は、SDK 品質への注力が Anthropic の Claude API エコシステム構想と一致すると述べている。

## できるようになったこと

- Anthropic 公式 SDK / MCP サーバーツールの生成チームが Anthropic 社内に統合された。
- Claude Platform の SDK・コネクタ整備とエージェント接続性強化が、組織的に内製化される見通し。
- （現時点では既存 SDK / API の利用方法そのものに即時の破壊的変更はなし。今後の体験改善の布石。）

## 影響範囲

- 対象ユーザー: Claude API / SDK を使う開発者、Claude Platform でエージェントを構築する企業
- 対象プラン: Claude Platform / Claude API 全般
- API / UI / 管理者機能: SDK 生成・MCP コネクタ・エージェント接続レイヤー（中長期の体験改善）

## 教材化メモ

Claude Platform / API 教材では、公式 SDK が Stainless 由来であること、今後 SDK・MCP コネクタ整備が内製強化される見通しを「エコシステムの方向性」として最小限触れられる。速報的な買収の経緯は教材本文に持ち込まず、「公式 SDK の供給体制」「エージェント接続性の重要性」という恒久的論点のみ反映候補。
