# 2026-07-09 Claude「Reflect」提供開始

## 公式内容の日本語要約

Settings > Reflectに、過去1/3/6/12ヶ月の会話傾向・よく使うトピック・作業パターンを振り返れるダッシュボード（ベータ）を追加。あわせてSettings > Time and focusで休憩リマインダーや静音時間（quiet hours）を設定可能に。「Claudeより速くできても自分でやり続けたいことは何か」といった内省的な問いかけも行う。プライバシー面ではincognitoチャット・連携ツールの元データ・健康関連統合の会話は対象外。

## できるようになったこと

- 過去1/3/6/12ヶ月の利用傾向・トピック・作業パターンの振り返り
- 休憩リマインダー・静音時間の設定

## 影響範囲

- 対象ユーザー: Free/Pro/Max全プランのメモリ機能ONユーザー
- 対象プラン: Free/Pro/Max
- API / UI / 管理者機能: UI（web・デスクトップ）、個人向け機能。管理者機能への影響なし。

## 教材化メモ

AIウェルビーイング・内省を促す機能設計の実例として、AI時代の働き方教材に使える。
