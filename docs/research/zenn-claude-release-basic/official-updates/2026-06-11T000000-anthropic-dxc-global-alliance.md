# 2026-06-11 Anthropic × DXC グローバルパートナーシップ

## 公式内容の日本語要約

Anthropic と DXC Technology が**複数年にわたるグローバルパートナーシップ**を発表した（2026-06-11）。DXC は Claude Partner Network の **Global Premier パートナー**に。

**規模と内容:**
- DXC は**数万人の Claude 認定エンジニア**を育成予定。Anthropic Partner Academy で 90 日の選抜トレーニングを経て認定。
- 銀行・航空会社・保険会社・製造業・政府機関が DXC に依存するミッションクリティカルシステムへ Claude モデルを統合。
- 初期フォーカス分野: 保険・サイバーセキュリティ・アプリケーションサービス。

**既存実績:**
- DXC が自社の AI ネイティブオーケストレーションプラットフォーム「DXC OASIS」（2026-04 ローンチ）の開発に Claude を主要ツールとして利用。Claude によりソフトウェア納品を推定 10x 加速、コードの 95%超が Claude 生成後に人間レビュー。
- DXC OASIS は現在 50 社超の顧客に展開済み。

## できるようになったこと

- Claude が DXC 経由で金融・航空・公共・製造といった規制業界のミッションクリティカルシステムに本番導入される体制が整う
- 企業は DXC の Claude 認定エンジニアを通じてアジェンティック AI の設計・展開・ガバナンスを支援を受けられる

## 影響範囲

- 対象ユーザー: 規制業界（金融・航空・保険・製造・政府）の CIO・IT 責任者
- 対象プラン: Enterprise / DXC マネージドサービス経由
- API / UI / 管理者機能: DXC OASIS（AI ネイティブオーケストレーションプラットフォーム）経由での Claude 統合が主経路

## 教材化メモ

- Claude を「個人や開発者向けツール」から「規制業界のミッションクリティカルシステム基盤」へと位置付ける動きの具体例。
- DXC OASIS の「95%超のコードが Claude 生成後に人間レビュー」という数値は、エンタープライズでの AI 活用度合いの事例として有用。
- 「数万人の Claude 認定エンジニアを 90 日で育成」は Anthropic Partner Academy の規模感を示す事例。
