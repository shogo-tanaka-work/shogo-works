# 2026-06-12 Fable 5・Mythos 5 アクセス停止

## 公式内容の日本語要約

Anthropic は 2026-06-12 午後 5:21（ET）、米国政府から **Export Control 指令**を受け、Claude Fable 5 および Claude Mythos 5 への全アクセスを停止した。

**指令の内容（Anthropic 声明より）:**
- 国家安全保障を根拠とした Export Control 指令
- 外国籍を持つ全ユーザー（米国内在住の外国籍 Anthropic 従業員を含む）への Fable 5/Mythos 5 アクセスを停止するよう命令
- 実質的な影響: コンプライアンス確保のため、**全顧客の Fable 5/Mythos 5 アクセスを突然停止**
- 政府はジャイルブレイク（セーフガード回避）の方法が発見されたと判断していると Anthropic は理解している
- 指令書に具体的な国家安全保障上の懸念事項は記載されていなかった

**Anthropic のポジション:**
- 法的指令には従っている
- ただし「狭い範囲のセーフガード回避の発見が数億人のユーザーへの展開中止の理由になるべきではない」と声明で異議を唱えている
- 提示されたジャイルブレイクは「特定のコードベースを読ませて修正させる」という手法で、他社モデルでも同等の能力が確認できると主張
- 透明性と技術的事実に基づいた法定プロセスを求めている

**影響範囲:**
- 停止対象: Fable 5 / Mythos 5（全ユーザー）
- 影響なし: その他すべての Anthropic モデル（Claude Opus 4.8、Sonnet 4.6、Haiku 4.5 等）

## できるようになったこと / できなくなったこと

- **できなくなったこと**: Fable 5 / Mythos 5 の利用
- **影響なし**: Opus 4.8、Sonnet 4.6、Haiku 4.5 等は引き続き利用可能

## 影響範囲

- 対象ユーザー: Fable 5 / Mythos 5 を利用していた全顧客
- 対象プラン: 全プラン（Enterprise / Pro / Team / PAYG すべて）
- API / UI / 管理者機能: API での `claude-fable-5` / `claude-mythos-5` モデル指定も停止

## 教材化メモ

- 2026-06-09 ローンチから3日後の突然の停止という異例の事態。
- AI モデルの利用可否に政府規制が関与するケース。企業が AI サービスに依存する際の「規制リスク」「可用性リスク」の典型例として使える。
- Anthropic が法的指令に従いながら公式声明で異議を唱えるという対応は、AI 企業とガバナンスの関係を示す事例。
