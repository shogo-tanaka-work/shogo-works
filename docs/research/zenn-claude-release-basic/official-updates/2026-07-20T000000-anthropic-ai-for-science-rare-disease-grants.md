# 2026-07-20 Anthropic AI for Science 希少疾患研究グラント

## 公式内容の日本語要約

Anthropicが「AI for Science」プログラムの一環として、希少遺伝性疾患研究者・バイオテック企業向けにClaudeクレジット最大5万ドル（6ヶ月間）を付与するグラント募集を開始。2トラック制で、Basic Science Track（Monarch Initiativeと連携した臨床研究者・患者団体向け、疾患メカニズム解明の加速）とBiotech Track（初期段階バイオテック企業向け、Claudeによる文書作成・規制対応・治療戦略分析支援で創薬タイムライン短縮）を用意。申請締切は2026年8月2日23:59 PST。AIはデータ不足や保険承認のような非技術的障壁は解決できない点も明記。

## できるようになったこと

- 希少疾患研究者・患者団体・バイオテック企業がClaudeクレジット（最大5万ドル/6ヶ月）に応募可能
- Monarch Initiativeとの連携による疾患メカニズム研究支援
- 創薬プロセス（文書作成・規制対応・治療戦略分析）へのClaude活用支援

## 影響範囲

- 対象ユーザー: 希少疾患領域の臨床研究者、患者団体、初期段階バイオテック企業
- 対象プラン: グラント選定者向け（Claude API/Platformクレジット付与）
- API / UI / 管理者機能: 該当なし（研究助成プログラム）

## 教材化メモ

製品機能アップデートではなく研究助成プログラムだが、Claudeのライフサイエンス領域活用事例（疾患メカニズム解析・創薬文書自動化）として業界別ユースケース教材に転用できる。
