# 2026-05-13 Claude for Small Business

## 公式内容の日本語要約

Anthropicは、Claude Cowork上で小規模事業者向けに業務アプリ連携、エージェント型ワークフロー、スキル、AI活用講座をまとめたClaude for Small Businessを発表した。QuickBooks、PayPal、HubSpot、Canva、Docusign、Google Workspace、Microsoft 365など、既存業務ツールをClaudeから扱う構成になっている。

公式ページでは、15本のready-to-run agentic workflowsと15本のskillsを提供すると説明している。金融、オペレーション、営業、マーケティング、人事、顧客対応など、小規模チームで人手が足りやすい業務が主な対象。

## できるようになったこと

- Claude Cowork内でClaude for Small Businessを有効化し、既存ツールを接続して業務フローを実行できる。
- QuickBooks、PayPal、HubSpot、Canva、Docusignなどの業務ツールをClaudeの作業対象にできる。
- 送信、投稿、支払いなどの外部アクションは人間が承認する設計として説明されている。
- PayPalと連携したAI Fluency for Small Business講座や、2026-05-14開始の米国都市ツアーも用意される。

## 影響範囲

- 対象ユーザー: 小規模事業者、個人事業主、少人数の営業・経理・運用チーム
- 対象プラン: Claude Coworkを中心とするClaude利用者
- API / UI / 管理者機能: UI / connectors / skills / workflows

## 教材化メモ

中小企業向けAI導入教材では、「AIツールを単体で入れる」のではなく、既存の会計、CRM、契約、制作、メール基盤に接続して業務単位で設計する例として扱える。特に、人間承認と既存権限の継承を前提にした説明は、AIガバナンス教材にも接続しやすい。
