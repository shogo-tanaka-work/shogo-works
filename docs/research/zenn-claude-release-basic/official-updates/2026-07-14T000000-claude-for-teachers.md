# 2026-07-14 Introducing Claude for Teachers

## 公式内容の日本語要約

米国K-12（幼稚園〜高校3年）の認定教員向けに、Claudeのプレミアム機能・指導用スキル・全50州の学習標準に準拠したカリキュラム連携を無償提供する「Claude for Teachers」を発表。データは学習（モデル訓練）には使用されず、生徒情報はK-12向けデータ保護契約（DPA）でFERPA（家族の教育権とプライバシーに関する法律）準拠を担保する。Claude Code・Cowork等の機能も含まれる。American Federation of Teachers（全米教員連盟）と連携して展開する。

## できるようになったこと

- 米国K-12認定教員がClaudeのプレミアム機能を無償利用可能に
- 指導用スキル・全50州の学習標準準拠カリキュラム連携を提供
- FERPA準拠のK-12向けDPAでデータ保護を担保（学習データとして不使用）

## 影響範囲

- 対象ユーザー: 米国K-12の認定教員
- 対象プラン: Claude for Teachers（無償の教育版アクセスパス）
- API / UI / 管理者機能: Claude Code、Cowork、カリキュラム連携機能

## 教材化メモ

- 教育機関向け無償アクセスパスは、企業向け提供とは別セグメントでのAI採用促進策。教育現場向け展開のケーススタディとして、日本の教育機関でのAI活用検討時の比較材料になる。
- FERPA準拠のデータ保護契約は、教育データを扱うAIサービスのコンプライアンス設計の実例として紹介できる。
