# 2026-07-23 Claude認定資格4種の役割別プログラムをブログで紹介

## 公式内容の日本語要約

Anthropicが、顧客対応でClaudeを活用する人向けに4種類の役割別認定資格をまとめたブログ記事を公開した。Associate（Foundations）、Developer（Foundations）、Architect（Foundations / Professional）の3ロール4資格構成で、Pearson VUE（OnVUE）経由のオンライン/テストセンター受験に対応。スコアは100〜1000のスケールで合格ラインは720。認定資格プログラム自体は2026年7月上旬から段階的に展開されていたが、本記事はその全体像をまとめて紹介する公式ブログ投稿。

## できるようになったこと

- Associate - Foundations（CCAO-F）: 業務でClaudeを使う非開発者（運用・マーケ・PM・教育・コミュニケーション等）向け
- Developer - Foundations（CCDV-F）: API・エージェント・ツール・統合を構築するエンジニア向け
- Architect - Foundations（CCAR-F）: 本番Claudeアプリケーションを設計するアーキテクト向け
- Architect - Professional（CCAR-P）: 本番システムを end-to-end で設計・ガバナンスする上級者向け

## 影響範囲

- 対象ユーザー: Claude Partner Networkに紐づく組織メールアドレスを持つ利用者（個人メール不可）
- 対象プラン: 該当なし（認定資格プログラム、受験料は資格ごとに異なる）
- API / UI / 管理者機能: Pearson VUE経由の受験プラットフォーム

## 教材化メモ

パートナー・実務者向けの体系的な認定制度が4資格構成で確立された。教材カリキュラム設計時に「どのロール（Associate/Developer/Architect）を対象にした教材か」を対応付けると、既存の資格体系との接続がしやすい。
