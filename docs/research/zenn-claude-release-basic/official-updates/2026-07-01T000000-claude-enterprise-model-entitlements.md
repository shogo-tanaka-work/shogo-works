# 2026-07-01 Claude Enterprise モデルエンタイトルメント

## 公式内容の日本語要約

Claude Enterprise プランの管理者が、ユーザーごとのモデルアクセス権と effort level 設定を制御できる「Model Entitlements」機能がローンチ。組織内で特定モデルへのアクセスを制限・許可できるガバナンス機能。

## できるようになったこと

- Enterprise 管理者がユーザーごとにアクセス可能なモデルを設定できる
- effort level（思考の深さ）を組織ポリシーとして制御できる
- 特定モデルへのアクセスを組織全体で制限・許可するガバナンス機能

## 影響範囲

- 対象ユーザー: Claude Enterprise 管理者
- 対象プラン: Enterprise のみ
- 管理コンソールから設定可能

## 教材化メモ

- Enterprise AI ガバナンス教材の「モデルアクセス制御」セクションに使える
- "どのモデルを・誰に・どの effort で使わせるか" を管理する設計の実例
