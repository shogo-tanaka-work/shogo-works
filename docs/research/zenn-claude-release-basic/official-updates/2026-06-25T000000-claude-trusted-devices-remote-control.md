# Claude — Trusted Devices for Remote Control

## 概要

Team/Enterprise プランの管理者が、メンバーのローカル Claude Code セッションへのリモートアクセスを「信頼済みデバイス」に限定できる認証レイヤーを追加。

## 主要ポイント

- **対象プラン**: Team / Enterprise のみ
- **機能**: ローカルで動作している Claude Code セッションをリモートから閲覧・操作する前にデバイス認証を必須化できる
- **目的**: 組織デプロイ環境での不正なリモートセッションアクセスを防止

## 公式引用

> "Admins for Team and Enterprise plans can now require members to verify their device before viewing or steering local Claude Code sessions remotely."
