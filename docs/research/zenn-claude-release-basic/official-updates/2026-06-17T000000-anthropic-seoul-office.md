# 2026-06-17 Anthropic ソウルオフィス開設

## 公式内容の日本語要約

Anthropic が韓国・ソウルにオフィスを開設し、韓国のAIエコシステムとのパートナーシップを発表した。NAVER（全エンジニアに Claude Code を展開）、Nexon、LG CNS、Samsung SDS、Hanwha Solutions との大規模パートナーシップが含まれる。KAIST 等の研究コンソーシアムとも連携。

## できるようになったこと

- 韓国法人ユーザーが現地サポートを受けやすくなる
- NAVER 全エンジニアへの Claude Code / Cowork 大規模展開が進む
- 韓国の大企業（IT・ゲーム・製造・エネルギー）での Claude 導入がより加速する見通し

## 影響範囲

- 対象ユーザー: 韓国法人・エンタープライズユーザー、研究機関
- 対象プラン: Enterprise / API
- API / UI / 管理者機能: 直接的な機能変更なし（ビジネス展開・サポート強化）

## 教材化メモ

大企業が全エンジニアに Claude Code を展開している事例として使える。NAVER の事例は「AI コーディングツールを組織全体に展開する場合の検討事項」の文脈で紹介価値がある。
