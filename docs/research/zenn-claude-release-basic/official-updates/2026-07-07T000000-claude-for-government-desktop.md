# 2026-07-07 Claude for Government DesktopにClaude Code / Cowork追加

## 公式内容の日本語要約

FedRAMP High認証環境で提供されるClaude for Government Desktopに、商用版と同一のClaude Code / Claude Coworkがパブリックベータで追加された。管理者による設定既定値の指定や部門別支出制御、改ざん検知可能なハッシュチェーン監査ログ（ATOプロセス対応）などのガバナンス機能も同時に提供される。エージェントは標準的なMDM経由でのデプロイに対応し、契約・請求は引き続きAnthropicが担当するため、別途クラウドプロバイダとの契約は不要。

## できるようになったこと

- Claude for Government DesktopでClaude Code / Claude Coworkがパブリックベータ利用可能に
- 管理者による設定既定値の指定・部門別支出制御
- 改ざん検知可能なハッシュチェーン監査ログ（ATOプロセス対応）
- 標準MDM経由でのエージェントデプロイに対応
- 契約・請求はAnthropicが引き続き担当（別途クラウドプロバイダ契約不要）

## 影響範囲

- 対象ユーザー: 政府機関顧客・調達担当者
- 対象プラン: Claude for Government
- API / UI / 管理者機能: Desktop・管理コンソール（支出制御・監査ログ）
