# 2026-06-03 Claude Partner Network に Services Track と Partner Hub を追加

## 公式内容の日本語要約

Anthropic が Claude Partner Network に「Services Track」（Select / Preferred / Global Premier の3階層）と「Partner Hub」（進捗確認・パートナー検索ポータル）を追加。Enterprise への Claude 導入を支援するパートナー認定制度の拡充。MCP コネクター経由で Claude 内から直接パートナー情報へアクセスできる。

## できるようになったこと

- **Services Track**: パートナーを Select / Preferred / Global Premier の3階層で認定。
- **Partner Hub**: パートナーの進捗確認や、導入支援パートナーの検索ができるポータル。
- MCP コネクター経由で Claude 内からパートナー情報へ直接アクセス。

## 影響範囲

- 対象ユーザー: SI / 代理店などの導入支援パートナー、および Claude を導入する Enterprise。
- 対象プラン: Enterprise 導入支援の枠組み。
- API / UI / 管理者機能: Partner Hub（ポータル）、MCP コネクター連携。

## 教材化メモ

記事化はしない（SI/代理店向けの認定制度で一般読者への直接インパクトが薄いため）。エンタープライズ導入動向のメモとして保持。Claude を企業導入する際に「導入支援パートナーを探せる経路ができた」点だけ、必要なら Knowledge の導入ガイドに将来反映する候補。
