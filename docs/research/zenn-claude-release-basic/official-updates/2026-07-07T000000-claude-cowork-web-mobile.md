# 2026-07-07 Claude Coworkがweb/モバイルに拡大

## 公式内容の日本語要約

これまでデスクトップ限定だったClaude Coworkが、web（claude.aiホーム画面）とモバイル（iOS/Androidアプリのサイドバー）でも利用可能になった。デバイスをまたいでセッションとファイルが同期し、オフライン中も作業が継続され、スケジュールタスクはデバイスが接続されていなくても実行される。人間の判断が必要な箇所では、デバイスを横断して通知が届く。ローンチ記念としてCowork利用上限が2026-08-05まで倍増される。Maxプランから順次ベータ展開され、他プランへは追って拡大予定。デスクトップは引き続き主要な深い作業用インターフェースとして位置付けられる。公開された1.2万セッションの利用データでは、業務運用33.4%・コンテンツ制作16.4%・ソフトウェア開発8.7%という利用分布が示された。

## できるようになったこと

- Claude Coworkがweb（claude.aiホーム画面）で利用可能に
- Claude Coworkがモバイル（iOS/Androidアプリのサイドバー）で利用可能に
- デバイスをまたいだセッション・ファイルの同期、オフライン中の作業継続
- スケジュールタスクがデバイス非接続でも実行される
- 人間の判断が必要な箇所でデバイス横断の通知
- ローンチ記念でCowork利用上限を2026-08-05まで倍増

## 影響範囲

- 対象ユーザー: Maxプランから順次（他プランへ拡大予定）
- 対象プラン: Max（先行展開）
- API / UI / 管理者機能: web（claude.aiホーム画面）・モバイルアプリ（iOS/Android）

- 利用分布データ（業務運用33.4%等）はCoworkの実務ユースケース紹介に引用できる
