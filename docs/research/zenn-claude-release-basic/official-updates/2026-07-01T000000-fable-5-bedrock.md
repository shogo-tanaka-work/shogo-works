# 2026-07-01 Claude Fable 5 on Amazon Bedrock 提供再開

## 公式内容の日本語要約

Amazon Bedrock および Claude Platform on AWS で Fable 5 の利用が 2026-07-01 より再開。提供リージョンは米国東部（us-east-1）とストックホルム。Mythos レベルの能力を全顧客に提供しつつ、強化されたガードレールを実装。複数日間にわたる長時間非同期タスク処理に対応。Advanced Prompt Optimization との連携をサポート。サイバーセキュリティ・生物学・化学・健康分野の有害プロンプトは Opus 4.8 にフォールバック。

## できるようになったこと

- Amazon Bedrock で Fable 5 が再利用可能に（us-east-1・Stockholm）
- 長時間非同期タスク（複数日間）の処理に対応
- Advanced Prompt Optimization との連携
- 強化されたガードレール（有害分野は Opus 4.8 フォールバック）

## 影響範囲

- 対象ユーザー: AWS Bedrock 利用者・Claude Platform on AWS 利用者
- 対象リージョン: us-east-1、Stockholm
