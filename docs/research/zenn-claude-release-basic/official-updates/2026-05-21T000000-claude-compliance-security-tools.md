# 2026-05-21 Claude がセキュリティ・コンプライアンスツール連携を拡大

## 公式内容の日本語要約

Anthropic は Claude のリリースノート（support.claude.com）に 2026-05-21 付で「Claude now works with more security and compliance tools」というエントリを掲載した。Claude Compliance API を介したセキュリティ・コンプライアンスプラットフォームとの連携が拡大し、IT・セキュリティチームが組織全体の Claude 利用を、他のアプリと同じツールで統制できるようになる、という内容。

## できるようになったこと

- Claude Compliance API（組織・ワークスペース・利用状況のセキュリティ関連データへプログラムからアクセスする API）を介した、外部のセキュリティ・コンプライアンスプラットフォームとの連携が拡大した。
- IT・セキュリティチームが、すでに他の SaaS で使っている統制・監査ツールの中で Claude の利用を可視化・ガバナンスできるようになる。

## 影響範囲

- 対象ユーザー: Claude を組織導入している企業の情シス・セキュリティ・コンプライアンス部門。
- 対象プラン: 企業向け（Compliance API は Enterprise 系のガバナンス機能。詳細条件は公式ドキュメントを参照）。
- API / UI / 管理者機能: 管理者・セキュリティ運用向け。Compliance API を経由した外部ツール連携。

## 教材化メモ

Compliance API 経由の外部セキュリティツール連携が「拡大した」という派生改善（enhancement）。Claude の Knowledge 教材にコンプライアンス / ガバナンス専用記事は現状なく、速報的な連携先リストを教材へ持ち込む必要はない。恒久情報としては「Claude には Compliance API があり、組織の Claude 利用を外部の統制ツールから監査できる」という枠組みのみ。教材本体への追記は見送り、ニュース記事 + 教材化メモに留める。
