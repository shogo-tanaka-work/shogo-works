# 2026-07-15 Self-serve HIPAA configuration for Claude Enterprise and API

## 公式内容の日本語要約

これまで営業担当者経由での手続きが必要だったHIPAA対応（BAA=事業提携契約の締結、導入ガイド）が、管理者による自己完結型フローで完結できるようになった。Claude EnterpriseとClaude Platform（API）の両方が対象。

## できるようになったこと

- 管理者がセルフサーブでHIPAA対応設定（BAA締結含む）を完結可能に
- Claude EnterpriseとClaude Platform（API）の双方で対応

## 影響範囲

- 対象ユーザー: 医療関連組織の管理者
- 対象プラン: Claude Enterprise、Claude Platform（API）
- API / UI / 管理者機能: 管理コンソールのコンプライアンス設定、BAA締結フロー

## 教材化メモ

- 営業サイクルを経ずにコンプライアンス設定が完結できるようになった点は、医療関連組織の導入障壁低下として紹介価値がある。Bedrock/Vertex経由でのHIPAA対応との違いを比較する教材ネタにもなる。
