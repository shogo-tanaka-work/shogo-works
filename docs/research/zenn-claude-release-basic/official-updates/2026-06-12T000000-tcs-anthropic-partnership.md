# 2026-06-12 TCS × Anthropic パートナーシップ

## 公式内容の日本語要約

Tata Consultancy Services（TCS）と Anthropic が**複数年のグローバルパートナーシップ**を発表（2026-06-12）。DXC パートナーシップ（2026-06-11）に続く大型エンタープライズ連携。

**規模:**
- TCS が Claude を **50,000 名の従業員に 56 カ国**で提供
- TCS の企業クライアント向けにも Claude ベースのソリューションを構築

**対象産業:**
金融サービス・ヘルスケア・公共部門・生命科学・航空・通信・医療技術

**具体的な実装:**
- **Diligenta（TCS の英国生命保険事業）**: 2,200 万人以上の保険加入者にサービス提供
- **Claude Code 活用**: 銀行部門のソフトウェアエンジニアリング効率化
- **TCS iON**: 年間 7,500 万件の評価で Claude 研修を提供

**TCS の役割:**
- 内部導入: エンジニアリング・財務・法務・マーケティング・営業チームで Claude を活用
- 保険業向けクレーム処理、銀行向け融資アドバイザリーなど業界固有のオファリングをパッケージ化

## できるようになったこと

- TCS 経由でクライアント企業が Claude ベースの業界特化ソリューションを導入できる経路が整う
- 規制産業（金融・保険・医療・公共）向けに、TCS のドメイン専門性 + Claude の AI 能力を組み合わせたサービスが提供される

## 影響範囲

- 対象ユーザー: TCS の企業クライアント（規制産業を中心）
- 対象プラン: Enterprise / TCS マネージドサービス経由
- API / UI / 管理者機能: TCS が独自のソリューション（Diligenta 等）に Claude を組み込む形態
