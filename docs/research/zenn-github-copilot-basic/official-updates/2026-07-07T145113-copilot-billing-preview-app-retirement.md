# 2026-07-07 Copilot Billing Previewアプリが8月3日に廃止予定

## 公式内容の日本語要約

Copilotの利用ベース課金モデル理解のために提供されていた「Copilot Billing Preview」アプリが、2026年8月3日に廃止される。GitHub公式の課金設定機能がアプリの提供範囲を超えて拡張された（ユーザー単位予算、Cost Center、使用量プールの割当など）ため、アプリ側は今後の開発対象から外れる。

## できるようになったこと

- （廃止予告のため新機能追加はなし）
- 代替として、課金設定内のAI Usageページ（グルーピング・フィルタ・エクスポート対応）、予算管理機能、組織/エンタープライズ向けユーザー単位予算、Usage API/Billing APIによる生データ取得を利用可能

## 影響範囲

- 対象ユーザー: Copilot Billing Previewアプリで支出状況を確認していた管理者
- 対象プラン: 利用ベース課金を利用する全プラン
- API / UI / 管理者機能: 課金設定UI、Usage API、Billing API

## 教材化メモ

- 「アプリからネイティブ機能への統合」の典型例。個別ツールが本体機能に取り込まれ廃止される流れは、SaaS運用管理の教材で汎用的に使える。
