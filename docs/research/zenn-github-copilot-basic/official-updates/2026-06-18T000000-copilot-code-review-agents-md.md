# 2026-06-18 Copilot コードレビュー AGENTS.md 対応

## 公式内容の日本語要約

Copilot のコードレビュー機能がリポジトリルートの AGENTS.md ファイルを読み込むようになり、ファイルに記載した指示に沿ったレビューフィードバックを生成できるようになった。あわせてドラフト PR への「Request」ボタン追加と PR タイムラインの表示整理が行われた。

## できるようになったこと

- AGENTS.md を置くだけで Copilot がプロジェクト独自のレビュールールを自動参照
- ドラフト PR の状態でも Reviewer ピッカーから直接 Copilot をリクエスト可能
- PR タイムライン上の Copilot レビューイベントが整理されて可読性向上

## 影響範囲

- 対象ユーザー: 全プラン・全ユーザー（GA）
- 対象プラン: 全プラン
- API / UI / 管理者機能: UI（GitHub.com PR画面）

## 教材化メモ

- AGENTS.md でコードレビューのルールを言語化する実践として活用できる
- プロジェクト固有のコーディング規約をCopilotに教え込む手段が増えた
