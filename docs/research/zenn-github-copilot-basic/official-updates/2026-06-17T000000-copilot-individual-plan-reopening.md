# 2026-06-17 Copilot 個人プラン サインアップ再開

## 公式内容の日本語要約

GitHub Copilot の Student / Pro / Pro+ / Max 個人プランの新規サインアップが再開された。プラン上限に近づいた際のアップグレードオプションと、従量課金オプションが新たに追加された。

## できるようになったこと

- 個人ユーザーが再度 Copilot に新規登録できるようになった
- Pro → Pro+ → Max へのアップグレードパスが整備された
- 従量課金オプションでリクエスト量に応じた柔軟な課金が可能に

## 影響範囲

- 対象ユーザー: 新規登録を希望していた個人ユーザー（Student / Pro / Pro+ / Max）
- 対象プラン: Student / Pro / Pro+ / Max
- API / UI / 管理者機能: GitHub アカウント設定からサインアップ

## 教材化メモ

一時的に新規サインアップが停止されていた背景（需要過多？）から再開という流れ。料金プランとアップグレードパスの整理に使える。
