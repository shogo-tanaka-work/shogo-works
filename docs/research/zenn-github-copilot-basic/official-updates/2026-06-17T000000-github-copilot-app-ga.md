# 2026-06-17 GitHub Copilot App GA

## 公式内容の日本語要約

GitHub Copilot app が Technical Preview から一般提供（GA）に移行した。macOS / Windows / Linux 対応のデスクトップアプリとして提供。Canvas（双方向コラボ UI）、クラウドスケジュール自動化、並列マルチリポジトリセッション、MCP サーバー連携が搭載される。

## できるようになったこと

- Issue / PR から直接エージェントセッションを開始できる
- 複数リポジトリへの並列セッション（parallel multi-repo sessions）が可能
- Canvas UI で Claude Code との双方向コラボレーション
- クラウドスケジュール自動化でバックグラウンド実行
- MCP サーバー連携でツール拡張
- macOS / Windows / Linux すべてに対応

## 影響範囲

- 対象ユーザー: GitHub Copilot Business / Enterprise ユーザー（管理者による有効化が必要）
- 対象プラン: Business / Enterprise
- API / UI / 管理者機能: 管理者がプレビューポリシーで有効化する必要あり

## 教材化メモ

Technical Preview から GA になったことで企業導入の検討対象になった。「IDE の補完を超えてデスクトップのエージェント環境に移行する流れ」として位置づけられる。並列マルチリポジトリは大規模リファクタリングやマイグレーション案件で価値が出る機能。
