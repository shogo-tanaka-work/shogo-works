# 2026-05-12 GitHub Copilot 4月利用レポート

## 公式内容の日本語要約

GitHub Copilot の usage-based billing に備えるため、2026年4月の Copilot activity を AI credits 換算で確認できる usage report が提供された。Copilot Business / Enterprise 管理者は enterprise 向け、Pro / Pro+ 利用者は個人利用分のレポートをダウンロードできる。

## できるようになったこと

- 4月の利用状況を AI credits 換算で確認
- 上位利用者、利用面、モデル別の消費傾向を把握
- 6月1日の AI credits 導入前に予算感を見積もる

## 影響範囲

- 対象ユーザー: Copilot Business / Enterprise 管理者、Copilot Pro / Pro+ 利用者
- 対象プラン: Copilot Business / Enterprise / Pro / Pro+
- API / UI / 管理者機能: Usage report、billing preparation

## 教材化メモ

Copilot 導入後の費用管理、利用可視化、AI credits 設計の教材で、レポートの注意点まで含めて扱うと実務的。
