# 2026-05-12 Copilot Code Review コメント体験改善

## 公式内容の日本語要約

GitHub Copilot code review のコメントが、重要度ラベル、類似コメントのグルーピング、suggested changeset UI の更新により、確認・対応しやすくなった。新しい pull requests experience に opt-in している全ユーザーが対象。

## できるようになったこと

- Copilot review comment に High / Medium / Low の severity label を表示
- 大きなPRで類似コメントをまとめ、重複感を減らす
- suggested changeset UI の更新により対応しやすくする

## 影響範囲

- 対象ユーザー: Copilot code review 利用者、PR reviewer
- 対象プラン: Copilot 利用者
- API / UI / 管理者機能: GitHub pull request UI

## 教材化メモ

AIレビュー運用教材で、severity に応じた対応基準や、重複コメントをどうレビュー会話に落とすかの演習に向く。
