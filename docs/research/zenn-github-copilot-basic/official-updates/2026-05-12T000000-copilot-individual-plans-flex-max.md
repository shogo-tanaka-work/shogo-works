# 2026-05-12 GitHub Copilot 個人プラン変更

## 公式内容の日本語要約

GitHub は、2026年6月1日の usage-based billing 移行に合わせて、Copilot 個人プランに flex allotment を導入し、Max plan を追加すると発表した。Pro / Pro+ は同じ価格のまま追加利用枠が付き、Max は高頻度利用者向けの新プランとして提供される。

## できるようになったこと

- Pro / Pro+ に subscription price と 1:1 の base credits に加え、可変の flex allotment を付与
- Max plan を月額100ドルで追加し、合計200ドル分の included usage を提供
- 有料プランの code completions と next edit suggestions は credits を消費せず無制限のまま

## 影響範囲

- 対象ユーザー: Copilot Pro / Pro+ / Max の個人利用者
- 対象プラン: Copilot Free / Pro / Pro+ / Max
- API / UI / 管理者機能: Billing dashboard、AI credits

AI coding tool のコスト管理教材で、agent run や高性能モデル利用が使用量課金に与える影響を説明する題材になる。
