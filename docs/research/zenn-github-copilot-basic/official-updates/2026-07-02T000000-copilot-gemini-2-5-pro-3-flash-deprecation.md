# 2026-07-02 Gemini 2.5 Pro / Gemini 3 FlashがCopilotから廃止予告（2026-07-31）

## 公式内容の日本語要約

GitHubは2026年7月31日にCopilot内のGemini 2.5 ProとGemini 3 Flashの提供を終了すると発表した。Gemini 2.5 ProはGemini 3.1 Proに、Gemini 3 FlashはGemini 3.5 Flashに置き換えられる。Chat・インライン編集・コード補完を含む全Copilot体験が対象で、期限後は廃止モデルが自動的に除外されるため利用者側の手動対応は不要。

## できるようになったこと

- （廃止予告のため新機能なし）Gemini 2.5 Pro / Gemini 3 Flashの提供終了日が確定
- 後継モデルとしてGemini 3.1 Pro / Gemini 3.5 Flashへの自動移行が案内された

## 影響範囲

- 対象ユーザー: 該当モデル利用中の全ユーザー・管理者
- 対象プラン: 全プラン
- API / UI / 管理者機能: モデルポリシー設定（Enterprise管理者は代替モデルの有効化が必要）

## 教材化メモ

- モデル廃止・置き換えのアナウンス例。教材では「利用中モデルが廃止予告された場合の対応手順（代替モデルの事前確認・有効化）」の題材として使える。
