# 2026-06-17 Copilot Chat Auto mode 全ユーザー開放

## 公式内容の日本語要約

GitHub Copilot Chat の Auto mode が github.com とモバイルアプリで全 Copilot プランに開放された。タスクの複雑さとシステム状態に応じてモデルを自動選択する。有料ユーザーは auto mode 利用時に 10% 割引が適用される。

## できるようになったこと

- 毎回モデルを手動選択する手間が不要になる
- タスク複雑さに応じた最適モデルが自動選択される
- 有料ユーザーは auto mode 使用時に 10% 割引
- github.com とモバイルアプリで全 Copilot プランが対象

## 影響範囲

- 対象ユーザー: 全 Copilot プランユーザー（Free 含む）
- 対象プラン: Free / Pro / Pro+ / Max / Business / Enterprise
- API / UI / 管理者機能: Chat UI（github.com・モバイル）でのみ有効

## 教材化メモ

「モデル選択の自動化」という流れの一例。コスト意識がある法人ユーザー向けには、auto mode を使うことでコストが最適化される可能性と、モデルを固定したい場面での手動選択との使い分けを説明できる。
