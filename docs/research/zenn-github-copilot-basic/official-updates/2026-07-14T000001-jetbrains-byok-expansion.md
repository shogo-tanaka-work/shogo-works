# 2026-07-14 GitHub Copilot for JetBrains expands BYOK capabilities

## 公式内容の日本語要約

JetBrains IDE向けGitHub Copilotで、OpenAI互換のカスタムエンドポイント＋APIキーによる独自モデル利用（BYOK: Bring Your Own Key）が可能になった。加えて、マーケットプレイスまたはソースリポジトリから閲覧・インストールできる統合プラグイン管理、Claudeエージェントプロバイダー対応のカスタマイズ機能（Pro+プランでプレビュー）、ローカルサンドボックス設定（プレビュー）、Copilot CLIデバッガースキル（プレビュー）が追加された。モデルピッカーや認証メッセージのUX改善、セッション永続性の強化も含まれる。

## できるようになったこと

- OpenAI互換カスタムエンドポイント＋APIキーによるBYOKでの独自モデル利用
- マーケットプレイス/ソースリポジトリからの統合プラグイン管理
- Claudeエージェントプロバイダー対応のカスタマイズ（Pro+プレビュー）
- ローカルサンドボックス設定、Copilot CLIデバッガースキル（いずれもプレビュー）

## 影響範囲

- 対象ユーザー: JetBrains IDE利用者、独自契約モデルを使いたいエンタープライズ
- 対象プラン: Pro+（一部機能はプレビュー限定）
- API / UI / 管理者機能: BYOK設定、プラグイン管理、エージェントプロバイダー設定

## 教材化メモ

- BYOKによりエンタープライズが自社契約モデル（Claude含む）をCopilot上で利用できるようになった点は、マルチベンダー運用を検討する企業向け教材で紹介価値がある。
