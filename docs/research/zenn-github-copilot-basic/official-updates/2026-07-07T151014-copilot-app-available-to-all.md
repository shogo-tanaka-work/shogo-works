# 2026-07-07 GitHub Copilotデスクトップアプリが全ユーザーに提供開始

## 公式内容の日本語要約

macOS/Windows/Linux対応のGitHub Copilotデスクトップアプリが、全ユーザーに提供開始された。GitHubアカウントでサインインするだけでCopilot Freeユーザーも利用可能になり、GitHub Educationユーザーにも対応した。Business/Enterpriseでは管理者がCopilot CLIポリシーを有効化する必要がある。サブスクリプションがなくても、自前のAPIキー（BYOK）でモデルプロバイダーを設定すればセッションを実行できる。

## できるようになったこと

- Copilot Freeユーザーがサインインのみでデスクトップアプリを利用可能に
- GitHub Educationユーザーへの対応
- Business/Enterprise管理者によるCopilot CLI有効化でチーム展開
- BYOK（自前APIキー）によるサブスクリプション不要のセッション実行

## 影響範囲

- 対象ユーザー: Copilot Free/Education/Business/Enterprise全般、非契約者（BYOK利用者）
- 対象プラン: 全プラン
- API / UI / 管理者機能: デスクトップアプリ、MDMによる企業管理、カスタムモデルプロバイダー連携

## 教材化メモ

- 「エージェント駆動開発」をデスクトップアプリ単体で完結できる点は、非エンジニア層への導入教材の入口として使いやすい。
