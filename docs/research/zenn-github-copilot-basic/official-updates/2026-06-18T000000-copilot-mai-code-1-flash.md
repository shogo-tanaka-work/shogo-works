# 2026-06-18 MAI-Code-1-Flash 追加プラットフォーム展開

## 公式内容の日本語要約

Microsoft が開発した小型高速コーディングモデル「MAI-Code-1-Flash」が、Copilot CLI・GitHub Copilot App・Visual Studio・JetBrains IDE・Eclipse・Xcode など8プラットフォームで選択可能になった。「同クラスモデルの中でベストインクラスの品質」と位置付けられており、GitHub Copilot向けに特別チューニング済み。

## できるようになったこと

- Copilot CLI、GitHub Copilot App（Web）、Visual Studio、JetBrains IDE、Eclipse、Xcodeでモデルとして選択可能
- 小型モデルとしてのレスポンス速度の優位性を活かした補完・エージェント利用

## 影響範囲

- 対象ユーザー: Copilot Free / Student / Pro / Pro+（段階的展開）。Business / Enterprise は近日対応予定
- 対象プラン: Free, Student, Pro, Pro+（Business/Enterprise は coming soon）
- API / UI / 管理者機能: UI（各IDE / Webインターフェース）

## 教材化メモ

- 「速度重視のタスクに小型モデルを選ぶ」という選択肢が増えた
- Business / Enterprise は展開待ちのため管理者は注意
