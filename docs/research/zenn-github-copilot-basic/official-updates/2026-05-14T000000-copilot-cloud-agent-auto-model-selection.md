# 2026-05-14 Copilot cloud agent auto model selection

## 公式内容の日本語要約

Copilot cloud agentがCopilot auto model selectionに対応した。model pickerでAutoを選ぶと、system healthとmodel performanceに基づき利用可能な最適モデルを選択する。通常のmodel multiplierから10% discountがあり、weekly rate limitsの影響を受けないと案内されている。

## できるようになったこと

- Cloud agentでAuto model selectionを使える。
- system health / model performanceに基づくモデル選択ができる。
- model multiplierの10% discountとweekly rate limit回避が案内された。

## 影響範囲

- 対象ユーザー: Copilot cloud agent利用者
- 対象プラン: Copilot
- API / UI / 管理者機能: cloud agent / model picker / billing

## 教材化メモ

Cloud agentのモデル選択教材では、固定モデル選択とAuto選択を分けて説明する。コストとrate limitを含めた運用判断に使える。
