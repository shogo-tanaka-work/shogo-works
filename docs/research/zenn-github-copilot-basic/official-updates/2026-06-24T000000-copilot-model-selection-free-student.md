# 2026-06-24 GitHub Copilot Free・Student プランのモデル選択変更

## 公式内容の日本語要約

GitHub Copilot の Free・Student プランで手動モデル選択が廃止された。代わりに「Copilot Auto」（タスクに応じて最適モデルを自動選択）のみが利用可能になる。各プランの制限の範囲内で複数のモデルファミリーへのアクセスは維持。Microsoft リリースのモデルから "Preview" ラベルも撤廃。

## できるようになったこと（変更点）

- **廃止**: Free・Student プランでの手動モデル選択
- **追加/変更なし**: Copilot Auto による自動最適化は引き続き（かつ唯一の選択肢として）提供
- **ラベル変更**: Microsoft リリースモデルから "(Preview)" の表記を廃止

## 影響範囲

- 対象ユーザー: GitHub Copilot Free・Student プランのユーザー（Business・Enterprise は非対象）
- 対象プラン: Free・Student のみ
- API / UI: IDE のモデル選択 UI から Free・Student ユーザーの手動選択が消える

## 教材化メモ

- 「UX の簡素化 vs ユーザーの選択肢」のトレードオフとして議論できる。GitHub の論理は「Auto の方が最適なので選ぶ必要がない」
- 有料プランとの差別化要素として「モデル選択権」を位置付けるビジネス設計の観点から教材化できる
- 個人・学生ユーザーへの影響と、Business/Enterprise との機能差分の整理に使える
