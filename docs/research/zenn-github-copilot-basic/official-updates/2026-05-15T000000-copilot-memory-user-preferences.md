# 2026-05-15 Copilot Memory remembers your preferences

## 公式内容の日本語要約

GitHub Copilot に、会話をまたいでユーザーの開発スタイルや指示を記憶する Copilot Memory が追加された。VS Code の Copilot Chat で利用でき、Pro / Pro+ ユーザーを対象に、preferred frameworks、coding patterns、communication style などを蓄積して次回以降の応答へ反映する。

## できるようになったこと

- 会話ごとに同じ好みや前提を繰り返し伝えなくてよくなる
- フレームワーク選好、コーディングパターン、応答スタイルを継続反映
- VS Code で自然言語により memory の確認・削除が可能
- `Settings > Copilot > Memory` から memory の確認・削除・有効化/無効化が可能

## 影響範囲

- 対象ユーザー: GitHub Copilot Pro / Pro+
- 対象プラン: Pro / Pro+
- API / UI / 管理者機能: VS Code の Copilot Chat / Settings

## 教材化メモ

prompt engineering から context engineering への移行を説明しやすい。単発プロンプトの工夫ではなく、長期的な好み・規約・前提をどう持たせるか、また不要な memory をどう管理するかを扱う教材に向く。
