# GitHub Copilot for Jira — GA（一般提供開始）

## 概要

2026年3月にパブリックプレビュー開始した「GitHub Copilot for Jira」が GA に移行。エージェントセッションの可視性と制御性を高める3機能を新たに追加。

## GA で追加された3機能

### 1. Streaming agent progress
Jira チケット内でコーディングエージェントの進捗をリアルタイム監視。画面を離れる必要なし。

### 2. Post-session steering
作業完了・ドラフト PR 作成後に、Jira チャットパネルから追加指示が可能。新 PR を作らず同一 PR 上で継続作業。

### 3. Simplified onboarding
GitHub 組織とリポジトリの接続設定を簡素化。必要なステップ数を削減。

## プレビュー時からの既存機能

- モデル選択
- Confluence コンテキスト（MCP 経由）
- カスタムエージェント
- カスタムフィールド
- スペースレベルのガイダンス
- レビューリクエスト通知

## セットアップ

Atlassian Marketplace から「GitHub Copilot for Jira」アプリをインストール（既存ユーザーは最新版へ更新）。
