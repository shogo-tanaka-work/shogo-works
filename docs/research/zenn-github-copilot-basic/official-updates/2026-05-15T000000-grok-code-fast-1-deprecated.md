# 2026-05-15 Grok Code Fast 1 deprecated

## 公式内容の日本語要約

GitHub Copilot で、事前告知どおり Grok Code Fast 1 が 2026-05-15 に廃止された。対象は Copilot Chat / Edit / Agent / CLI / Code Review。代替として GPT-5 mini または Claude Haiku 4.5 が案内されている。

## できるようになったこと

- 新機能ではなく、廃止予定だったモデルの実施確認
- 利用者は代替モデルへの移行が必要

## 影響範囲

- 対象ユーザー: Grok Code Fast 1 利用者
- 対象プラン: GitHub Copilot 全般
- API / UI / 管理者機能: Chat / Edit / Agent / CLI / Code Review
