# 2026-07-09 OpenAI GPT-5.6 Sol/Terra/LunaがGitHub Copilotで利用可能に

## 公式内容の日本語要約

OpenAIのGPT-5.6ファミリー3モデル（Sol / Terra / Luna）がGitHub Copilotのモデルピッカーに追加された。Solは大規模コードベースでの複雑な推論・高度なエージェンティック作業向けの最上位推論モデル、Terraは日常的な対話・エージェンティック開発向けのバランス型、Lunaは軽量・低コストでの小規模タスク向け。ロールアウトは段階的で、Enterprise/Businessでは既定で無効化されておりポリシーで有効化が必要。

前回チェック（2026-07-09サマリー）では、GPT-5.6が「一部報道はあるがGA日は未確定」として更新あり/記事化を見送っていたが、本チェックで一般提供の公式アナウンスを確認できた。

## できるようになったこと

- Sol: Copilot Pro+ / Max / Business / Enterprise で利用可能
- Terra・Luna: Pro / Pro+ / Max / Business / Enterprise で利用可能
- VS Code、Visual Studio、Copilot CLI、GitHub Copilot cloud agent、Copilotアプリ、github.com、GitHub Mobile（iOS/Android）、JetBrains、Xcode、Eclipseで選択可能
- 課金はProvider list pricingベースのUsage Based Billing

## 影響範囲

- 対象ユーザー: Copilot Pro/Pro+/Max/Business/Enterpriseユーザー
- 対象プラン: Sol は Pro+ 以上、Terra/Luna は Pro 以上
- API / UI / 管理者機能: モデルピッカー、Enterprise/Business管理者ポリシー（既定無効・要有効化）
