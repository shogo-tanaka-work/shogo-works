# 2026-06-05 GitHub Copilot（GPT-5.2 系退役 / エンタープライズ管理プラグイン）

## 公式内容の日本語要約

GitHub Copilot の changelog に 2026-06-05 付で2件掲載された。旧モデル GPT-5.2 および GPT-5.2-Codex が Copilot から退役（deprecated）したこと、および Visual Studio Code でエンタープライズが Copilot プラグインを管理できる機能が公開プレビューに入ったこと。

## できるようになったこと

- **GPT-5.2 / GPT-5.2-Codex の退役**: 旧モデルが Copilot から退役。これらを指定していたワークフローは後継モデルへの切り替えが必要。
- **エンタープライズ管理プラグイン（VS Code 公開プレビュー）**: 組織が Copilot プラグインを VS Code 内で直接管理可能に（プレビュー）。

## 影響範囲

- 対象ユーザー: Copilot 利用者全般。プラグイン管理は管理者・情シス向け。
- 対象プラン: 退役は全般。管理プラグインはエンタープライズ。
- API / UI / 管理者機能: モデル選択（退役）、組織のプラグインガバナンス（VS Code）。

## 教材化メモ

Copilot 教材の「モデル選択」章に GPT-5.2 / GPT-5.2-Codex の退役を反映し、固定指定していた場合の移行を注記（モデル退役は教材の前提が古くなる更新）。「組織管理 / ガバナンス」章にエンタープライズ管理プラグイン（VS Code・プレビュー）を追記し、許可プラグインの集中管理の文脈で扱う。
