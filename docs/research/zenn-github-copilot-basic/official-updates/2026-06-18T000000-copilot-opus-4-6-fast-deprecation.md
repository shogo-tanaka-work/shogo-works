# 2026-06-18 Opus 4.6 (fast) 廃止予告

## 公式内容の日本語要約

Copilot Chat・インライン編集・Ask/Agent モード・コード補完など全サーフェスで「Opus 4.6 (fast)」が 2026-06-29 に廃止される。後継は「Opus 4.8 (fast)」。Copilot Enterprise 管理者はモデルポリシーで Opus 4.8 (fast) を有効化する必要がある。

## できるようになったこと / 必要な対応

- 廃止日: 2026-06-29
- 移行先モデル: Opus 4.8 (fast)
- Copilot Enterprise 管理者はモデルポリシーで Opus 4.8 (fast) を有効化する必要あり

## 影響範囲

- 対象ユーザー: Copilot 全プラン（特に Enterprise 管理者は対応要）
- 対象プラン: 全プラン
- API / UI / 管理者機能: 管理者コンソール（Enterprise の場合）

## 教材化メモ

- モデルの廃止スケジュール管理の事例として紹介できる
- Enterprise 管理者が事前にポリシー更新を行わないとユーザーが影響を受ける点を強調
