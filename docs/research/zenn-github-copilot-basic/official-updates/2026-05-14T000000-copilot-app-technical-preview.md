# 2026-05-14 GitHub Copilot app technical preview

## 公式内容の日本語要約

GitHub Copilot appがtechnical previewとして公開された。GitHub-native desktop experienceとして、issue、pull request、prompt、previous sessionからagentic development sessionを開始し、branch、files、conversation、task stateを分離しながら作業できる。レビュー、検証、PR作成、Agent Mergeまで同じ流れで扱う。

## できるようになったこと

- GitHub上のissue / PR / prompt / previous sessionを起点にsessionを開始できる。
- sessionごとにbranch、files、conversation、task stateを分けて扱える。
- integrated terminal / browserで検証し、PRへつなげられる。
- Business / EnterpriseではpreviewとCopilot CLI policy設定が必要。

## 影響範囲

- 対象ユーザー: GitHub Copilot Pro / Pro+ / Business / Enterprise利用者
- 対象プラン: early access / technical preview
- API / UI / 管理者機能: desktop app / agentic development / policy

## 教材化メモ

Copilot教材では、Cloud AgentをWeb上の機能としてだけでなく、desktop app + GitHub context + session管理として説明する必要がある。Claude Code / Codex desktopとの比較軸にもなる。
