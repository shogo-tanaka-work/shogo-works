# 2026-07-01 GitHub Models 全機能廃止予告

## 公式内容の日本語要約

GitHub Models（プレイグラウンド・モデルカタログ・推論 API・BYOK を含む全機能）が 2026-07-30 に完全廃止される。2026-07-16 と 07-23 に短時間のブラウンアウト（事前告知あり）を実施。代替として Azure AI Foundry または GitHub Copilot への移行を推奨。

## できるようになったこと（変化した点）

- プレイグラウンド: 2026-07-30 に廃止
- モデルカタログ: 2026-07-30 に廃止
- 推論 API（inference API）: 2026-07-30 に廃止
- BYOK（Bring Your Own Key）: 2026-07-30 に廃止
- ブラウンアウト: 07-16・07-23 に実施（事前告知あり）

## 移行先

- **Azure AI Foundry**: より高度な推論 API と管理機能
- **GitHub Copilot**: IDE 統合、Copilot Chat、エージェント機能

## 影響範囲

- 対象ユーザー: GitHub Models を利用していたすべての開発者
- 対象期限: 2026-07-30（まで約4週間）

## 教材化メモ

- 「実験的サービス → 廃止 → 本番サービスへ統合」というGitHubのAI戦略の典型例
- GitHub Models から Azure AI Foundry への移行手順は教材候補
