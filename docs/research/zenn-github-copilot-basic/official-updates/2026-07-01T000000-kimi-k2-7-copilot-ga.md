# 2026-07-01 Kimi K2.7 Code GA in GitHub Copilot

## 公式内容の日本語要約

Moonshot AI の Kimi K2.7 Code が GitHub Copilot のモデルピッカーで選択できる初のオープンウェイトモデルとして 2026-07-01 に一般提供開始。Pro・Pro+・Max プランから順次展開し、Business・Enterprise へ拡大予定。VS Code・Visual Studio・JetBrains・Xcode・Eclipse など多数の IDE に対応。Business/Enterprise ではデフォルト無効、管理者による有効化が必要。

## できるようになったこと

- Copilot モデルピッカーで Kimi K2.7 Code を選択可能
- オープンウェイトモデルとして透明性の高い選択肢を提供
- 多数の IDE（VS Code / VS / JetBrains / Xcode / Eclipse）に対応
- コード補完・Copilot Chat で利用可能

## 影響範囲

- 対象ユーザー: Pro / Pro+ / Max（即時）、Business / Enterprise（要管理者有効化）
- Business/Enterprise: デフォルト無効、管理者ポリシーで有効化が必要

## 教材化メモ

- Copilot 初のオープンウェイトモデル採用 → クローズド/オープンの選択肢という文脈で使える
- 企業管理者がモデルポリシーでオープンウェイトを許可/制限できるガバナンス例
