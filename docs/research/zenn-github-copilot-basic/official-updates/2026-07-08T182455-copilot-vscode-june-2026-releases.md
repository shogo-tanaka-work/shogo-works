# 2026-07-08 GitHub Copilot in Visual Studio Code — 2026年6月分まとめリリース

## 公式内容の日本語要約

VS Code v1.123〜v1.127（6月〜7月上旬にかけて出荷）の変更点をまとめたリリースノート。エージェントによるページ操作・内容検査・Webアプリ検証を行う統合ブラウザツールが一般提供（GA）になり、お気に入り・履歴・Web検索・権限拡張（カメラ・位置情報・マイク）、スクリーンショット添付、リモートワークスペースからのHTTP(S)トラフィックプロキシ経由ブラウジング（パブリックプレビュー）が追加された。並行セッション・チャット表示、セッションのグルーピング・ドラッグ&ドロップ整理、チャット全体のコスト可視化・サブエージェントのクレジット使用量確認、モデルプロバイダーのMarketplace直接検索、Autopilotのタスク完了検知精度向上なども含まれる。

## できるようになったこと

- 統合ブラウザツールのGA化（ページ操作・内容検査・Webアプリ検証）
- お気に入り・履歴・Web検索、権限拡張、スクリーンショット添付
- リモートワークスペースからのHTTP(S)プロキシ経由ブラウジング（プレビュー）
- 複数エージェントタスクの並行セッション表示、セッション内の複数チャット整理
- チャット単位のコスト可視化・サブエージェントのクレジット使用量確認
- Marketplaceからのモデルプロバイダー直接検索・導入
- Autopilotのタスク完了検知精度向上（手動介入の削減）
- セッション同期、ガター表示改善、1Mコンテキストウィンドウ、MCP OAuth資格情報対応

## 影響範囲

- 対象ユーザー: VS Code + Copilot利用者全般
- 対象プラン: 全プラン（一部エンタープライズ管理機能は別途）
- API / UI / 管理者機能: VS Code拡張機能、ブラウザツール、コストダッシュボード

## 教材化メモ

- ブラウザツールGA化とAutopilot精度向上は、「エージェントに検証まで任せる」ワークフロー教材の更新ポイントとして扱いやすい。月次まとめのため、個々の機能は既出分と重複する可能性がある点に注意。
