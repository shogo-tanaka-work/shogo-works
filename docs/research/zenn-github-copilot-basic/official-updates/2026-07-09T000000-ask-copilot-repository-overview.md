# 2026-07-09 github.comでCopilotにリポジトリの概要を聞けるように

## 公式内容の日本語要約

github.comで初めて訪れるリポジトリのホームページに、Copilotが「Give me a high-level overview（概要を教えて）」という提案を表示するようになった。選択すると、Copilotがリポジトリを解析し、目的・使用技術・コントリビューションガイドラインなどを含む要約を生成する。READMEが存在しないリポジトリでは、READMEの自動生成も可能。

## できるようになったこと

- github.comナビゲーションバーのCopilotアイコンからリポジトリ概要をリクエスト
- Copilot Chatから直接リクエスト
- リポジトリホームページのショートカットボタン（「コントリビュート方法を聞く」「最新変更点を要約」等）
- READMEがない場合の自動生成

## 影響範囲

- 対象ユーザー: 全GitHub Copilotサブスクリプション層
- 対象プラン: 全プラン
- API / UI / 管理者機能: github.comのリポジトリホームページ、Copilot Chat

## 教材化メモ

- 初見リポジトリのオンボーディング短縮という切り口で、OSSコントリビューション入門教材との相性が良い。
