# 2026-05-20 GitHub Copilot Chat (web) でセマンティック Issue 検索

## 公式内容の日本語要約

GitHub Copilot Chat の Web 版で、自然言語による Issue 検索・グルーピング・分析が可能になった。新しいセマンティック Issue インデックスにより、完全一致のキーワード検索ではなく **意図ベースで関連 Issue を発見**できる。リポジトリ内のプランニング、トリアージ、Issue 発見ワークフローを支援。

## できるようになったこと

- Copilot Chat (web) で「ログイン周りで認証エラーが出ている Issue を全部見つけて」のような自然言語検索
- 表記揺れがあってもセマンティックに類似 Issue をグループ化
- トリアージ作業の効率化

## 影響範囲

- 対象: GitHub Copilot Chat (web) ユーザー
- 対象プラン: **全 Copilot プランで GA**
- 制限: web 版のみ（IDE 統合版は対象外）

## 教材化メモ

Issue トリアージ運用の教材で「セマンティック検索を起点にバックログを整理」する例として使える。Copilot Coding Agent 教材の前段に置く「Issue 整理 → エージェント割当」フローの構成素材。
