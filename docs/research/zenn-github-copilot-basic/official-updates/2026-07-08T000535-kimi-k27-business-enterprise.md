# 2026-07-07 Kimi K2.7がCopilot Business/Enterpriseにも対応拡大

## 公式内容の日本語要約

オープンウェイトモデルのKimi K2.7 Codeが、Copilot Business/Enterpriseのモデルピッカーでも選択可能になった。2026-07-01にPro/Pro+/Maxプランで先行提供されたのに続く展開。GitHubがMicrosoft Azure上でホストし、利用ベース課金の提供元標準価格で課金される。Business/Enterpriseではデフォルト無効で、管理者がポリシーを手動有効化する必要がある。

## できるようになったこと

- Business/Enterpriseプランのモデルピッカーへの追加（管理者が有効化した場合）
- 低コストなコーディング向け選択肢としての利用

## 影響範囲

- 対象ユーザー: Copilot Business/Enterpriseの組織管理者・開発者
- 対象プラン: Business, Enterprise（Pro/Pro+/Maxは07-01に対応済み）
- API / UI / 管理者機能: モデルピッカー、組織ポリシー設定
