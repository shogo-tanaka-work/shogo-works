# 2026-07-01 GitHub Copilot Vision GA

## 公式内容の日本語要約

Copilot Chat のチャットプロンプトに画像（JPEG/PNG/GIF/WebP）と PDF を直接添付し、Copilot がコードと合わせて視覚情報を解析できる「Copilot Vision」が 2026-07-01 に一般提供（GA）。VS Code / github.com / Copilot CLI で利用可能。Free・Pro・Pro+・Business・Enterprise の全プランで追加設定不要。

## できるようになったこと

- Copilot Chat に JPEG / PNG / GIF / WebP / PDF を直接添付してコードと一緒に解析
- UI スクリーンショットをもとにコード生成、ドキュメントPDFを参照しながら実装
- VS Code・github.com・Copilot CLI で利用可能
- 全プラン（Free〜Enterprise）で追加設定不要

## 影響範囲

- 対象ユーザー: 全 Copilot ユーザー
- 対象プラン: Free / Pro / Pro+ / Business / Enterprise
- 利用可能エディタ: VS Code、github.com、Copilot CLI

## 教材化メモ

- 「コードだけでなく画像・文書も入力できる」というマルチモーダル利用の典型例
- UI スクリーンショット → コード生成のユースケース説明に使える
