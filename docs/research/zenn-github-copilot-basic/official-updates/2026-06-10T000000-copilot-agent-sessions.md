# 2026-06-10 Copilot Chat がエージェントセッションを表示

## 公式内容の日本語要約

Copilot Chat が Copilot クラウドエージェントのセッションと統合された。2つの新機能「Get agent logs（エージェントログ取得）」と「Session search（セッション検索）」が追加され、Copilot エージェントの作業をチャット内でリアルタイムに把握できるようになった。

## できるようになったこと

- **Get agent logs**: Copilot クラウドエージェントが PR 作業で実行した変更内容・検証・推論をチャット内で直接取得できる。
- **Session search**: 過去のエージェントセッションをトピック・タイトル・最近性で検索・要約できる。
- チャットコンテキスト内で新しいエージェントセッションを開始することも可能。

## 影響範囲

- 対象ユーザー: Copilot クラウドエージェントセッション（PR 作成、リポジトリリサーチ等）を実行したユーザー
- 対象プラン: Pro+ / Max / Business / Enterprise（詳細は公式ページ参照）
- 追加の設定は不要
