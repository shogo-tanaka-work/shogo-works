# 2026-06-17 Copilot Agent Finder

## 公式内容の日本語要約

自然言語で MCP サーバー・スキル・エージェント・ツールを検索・接続できる Agent Finder が GitHub Copilot に追加された。ARD（Agentic Resource Discovery）仕様に準拠。エンタープライズではプライベートレジストリで社内ツールを管理可能。

## できるようになったこと

- 名前を知らなくても自然言語でAIツール・MCP サーバーを発見できる
- ARD 仕様準拠のサーバーが自動認識される
- Enterprise では社内プライベートレジストリでツールを管理可能
- 全 Copilot プランで利用可能

## 影響範囲

- 対象ユーザー: 全 Copilot プランユーザー
- 対象プラン: 全プラン（Enterprise は追加でプライベートレジストリ機能あり）
- API / UI / 管理者機能: Copilot UI + Enterprise は管理者設定でプライベートレジストリ設定

## 教材化メモ

MCP エコシステムの「発見」問題を解決するアプローチ。「MCP サーバーを探す → 設定する → 使う」という手順の前段階を自動化する試み。ARD 仕様という新しい標準が生まれているという文脈で紹介できる。
