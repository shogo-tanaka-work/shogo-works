# GitHub Copilot — MAI-Code-1-Flash が Business/Enterprise で GA

## 概要

Microsoft AI が開発したコーディング特化モデル MAI-Code-1-Flash が GitHub Copilot Business および Copilot Enterprise で一般提供（GA）開始。

## 詳細

- **対象プラン**: Copilot Business / Copilot Enterprise（Individual は対象外）
- **提供開始**: 2026-06-26（GA）
- **位置づけ**: コーディング特化・低レイテンシ。反復的なエージェント型コーディングワークフロー向け
- **有効化**: 管理者が Copilot 設定でポリシーを有効化 → チームメンバーが使用可能に
- **課金**: 使用量ベース（プロバイダー定価）
- **特徴**: 高ボリュームな繰り返しコーディングタスクに最適化、速度・効率重視
