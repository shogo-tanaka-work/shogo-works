# 2026-07-08 ByteDance Seed Seedream 5.0 Pro

## 公式内容の日本語要約

ByteDance SeedがSeedream 5.0シリーズの上位モデル「Seedream 5.0 Pro」を発表した。空間位置把握（グラウンディング）と領域単位の意味理解に強みを持ち、ポイント/投げ縄選択によるピクセル単位の精密編集、スケッチレンダリング、色・素材の置換、レイヤー分離（透過PNGでの編集可能なデザイン素材出力）、複数画像の合成に対応する。データや密なテキストを高密度なインフォグラフィック（タイムライン・チャート・写実的イメージの組み合わせ）へ変換する「複雑情報の可視化」も特徴。日本語・中国語・英語・フランス語・ドイツ語・ロシア語・韓国語・スペイン語・アラビア語など10言語以上でのテキストレンダリングにも対応し、商用のプロダクト撮影・広告クリエイティブ・ブランディング用途を想定する。コンシューマー向けにはDreaminaで提供され、アカウントごとに無料デイリークレジットが付与される。

## できるようになったこと

- ポイント/投げ縄選択によるピクセル単位の精密編集（グラウンディング・領域単位の意味理解）
- テキスト指示によるレイヤー分離（透過PNGでの編集可能なデザイン素材出力）
- データ・密なテキストを高密度インフォグラフィックへ変換する複雑情報の可視化
- 10言語以上のネイティブテキストレンダリング（日本語含む）

## 影響範囲

- 対象ユーザー: デザイナー・クリエイティブ制作者、商用素材制作担当者
- 対象プラン: Dreaminaでの提供（アカウントごとに無料デイリークレジット付与）、Seedream 5.0（無印）/5.0 Liteとの上位モデルという位置づけ
- API / UI / 管理者機能: Dreaminaアプリ内での提供。API提供有無は本文未記載
