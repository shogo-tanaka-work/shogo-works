# 2026-05-12 Dify v1.14.1

## 公式内容の日本語要約

Dify v1.14.1 は、セキュリティ強化、workflow / knowledge-base の安定化、self-hosted deployment の整理、Dify UI への移行継続を中心とした patch release。self-hosted の `SECRET_KEY` hardening、内部 metrics endpoint の保護、IDOR 修正、LiteLLM の CVE-2026-42208 対応を含む。

## できるようになったこと

- Docker deployment で public default key 依存を避け、空の `SECRET_KEY` には runtime key を生成・永続化
- `/threads` と `/db-pool-stat` の unauthenticated exposure を避ける hardening
- workflow-version loading、large app lists の online-user polling、preview resize observer loop などを修正
- Human-in-the-loop flow で selected action values を扱える
- Explore recommended apps の category 設定と order 調整を追加

## 影響範囲

- 対象ユーザー: Dify self-hosted 運用者、workflow / RAG / HITL 利用者
- 対象プラン: Dify OSS / self-hosted
- API / UI / 管理者機能: Docker deployment、API、workflow、knowledge base、web UI
